# 已有前置与本章边界

在进入大语言模型（Large Language Model, LLM）的学习之前，我们需要明确你已经掌握的知识基石，以及本章将要探索的全新领域。大模型并非凭空产生，而是建立在深厚的深度学习基础之上的。

## 1. 已有的基石：不需要重新学习的东西

已经在 `06NN系列` 和 `02模型` 中掌握了构建神经网络的核心组件和经典架构。这些知识在 LLM 中依然适用，甚至更为重要。

### 1.1 基础组件 (`06NN系列`)
LLM 的庞大参数依然是由最基础的层和算子组成的：
*   **激活函数**：Transformer 中常用的 GELU、SwiGLU 等，本质上是你学过的 `03激活函数` 的变体。
*   **归一化**：Layer Normalization (LN) 是 Transformer 的标配，RMSNorm 是其简化版，原理可见 `05归一化`。
*   **优化器**：AdamW 依然是主流，理解 `06优化器` 中的动量和自适应学习率至关重要。
*   **损失函数**：语言建模的核心依然是分类问题，`02损失函数/CrossEntropyLoss交叉熵损失.md` 是训练 LLM 的根本。

### 1.2 序列建模与注意力 (`02模型`)
*   **Transformer 架构**：这是 LLM 的心脏。你已经在 `02模型/Transformer` 中详细学习了 Self-Attention、Multi-Head Attention、Encoder-Decoder 结构。
    *   *注意*：LLM 主要使用 **Decoder-only** 架构（如 GPT 系列），而你可能更熟悉 Encoder-only（如 BERT）或 Encoder-Decoder（如 T5）。
*   **位置编码**：RoPE（旋转位置编码）是现在的标准，但其根源在于你学过的 `Positional coding.md`。
*   **Tokenization**：虽然 LLM 使用 BPE 或 SentencePiece，但概念与 `Word Embedding.md` 和 `Input Embedding层.md` 是一脉相承的。

### 1.3 预训练思想 (`02模型/自监督预训练`)
*   **自监督学习**：你已经了解了 BERT 的 MLM（掩码语言模型）。LLM 主要使用 CLM（因果语言模型，即 Next Token Prediction）。
*   **Auto-encoder**：虽然 LLM 是自回归的，但 VAE 和 Auto-encoder 的潜在空间（Latent Space）概念有助于理解 LLM 的表征能力。

## 2. 本章的增量：你需要掌握的新世界

虽然基础组件相同，但当参数量从 1亿 (BERT-Base) 增长到 1000亿+ (GPT-3, Llama) 时，量变引起了质变。本章将重点关注以下**差异化**内容：

### 2.1 从“判别”到“生成”
*   **旧范式**：输入文本 -> 模型 -> 类别标签/数值（如情感分析、NER）。
*   **新范式**：输入文本 (Prompt) -> 模型 -> 续写文本 (Completion)。
*   **核心转变**：所有的 NLP 任务都被统一为**文本生成**任务。

### 2.2 涌现能力 (Emergent Abilities)
*   当模型规模突破一定临界点（Scaling Laws），模型突然展现出小模型不具备的能力，如：
    *   **上下文学习 (In-Context Learning)**：无需梯度更新，仅通过 Prompt 中的示例就能学习新任务。
    *   **思维链 (Chain-of-Thought)**：通过一步步推理解决复杂的逻辑/数学问题。

### 2.3 训练流程的演进
传统的“预训练+微调”演变为更复杂的流水线：
1.  **Pre-training (预训练)**：学习海量知识，获得“通识”能力。（数据量级：Trillion Tokens）
2.  **SFT (Supervised Fine-Tuning, 指令微调)**：学习遵循人类指令，获得“对话”能力。
3.  **Alignment (对齐, RLHF/DPO)**：学习人类偏好，变得“有用、安全、诚实”。

### 2.4 系统工程与高效计算
*   **分布式训练**：单卡无法容纳大模型，需要 3D 并行（数据并行、张量并行、流水线并行）。
*   **显存优化**：FlashAttention、KV Cache、Quantization (量化)。
*   **应用开发**：RAG (检索增强生成)、Agents (智能体)、Tool Use (工具调用)。

## 3. 总结

| 维度 | 传统深度学习 / 小模型时代 | 大语言模型时代 (LLM) |
| :--- | :--- | :--- |
| **核心目标** | 特定任务的高性能 (Specific Task) | 通用任务求解器 (General Purpose) |
| **主要架构** | CNN, RNN, Transformer (Encoder) | Transformer (Decoder-only) |
| **学习范式** | Supervised Learning, Pre-train + Fine-tune | Pre-train + Prompt / SFT + RLHF |
| **输入输出** | 向量/张量 -> 概率/数值 | Text (Prompt) -> Text (Response) |
| **关键瓶颈** | 标注数据量, 模型设计技巧 | 算力, 显存, 高质量无监督数据 |

本章将带你从 Transformer 的细节出发，一步步构建起对庞大 LLM 系统的完整认知。

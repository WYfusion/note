# Leaky ReLU 激活函数

## 1. 背景与定义
**Leaky ReLU (Leaky Rectified Linear Unit)** 是为了解决标准 ReLU 的“神经元死亡 (Dead ReLU)”问题而提出的。它在输入为负时引入了一个很小的斜率 $\alpha$，允许少量梯度流过。

**公式**：
$$
f(x) = \begin{cases}
x, & x \geq 0 \\
\alpha x, & x < 0
\end{cases}
$$
其中 $\alpha$ 是一个小的常数超参数（通常取 $0.01$）。

## 2. 导函数

**导数公式**：
$$
f'(x) = \begin{cases}
1, & x \geq 0 \\
\alpha, & x < 0
\end{cases}
$$

## 3. 优缺点分析

### 优点
1.  **解决神经元死亡问题**：在 $x < 0$ 时导数不为 0（为 $\alpha$），保证了神经元在负区间也能更新参数。
2.  **计算高效**：不涉及指数运算，仅包含简单的线性变换和阈值判断。
3.  **收敛速度快**：保留了 ReLU 的线性特性（正区间），有助于缓解梯度消失。

### 缺点
1.  **超参数 $\alpha$ 需人工设定**：$\alpha$ 的选择是经验性的，不同任务可能需要不同的 $\alpha$。
2.  **非零中心化**：输出均值不为 0（虽然比 ReLU 稍好一点，但仍存在偏移）。
3.  **不连续性**：在 $x=0$ 处导数不连续（虽然在实际应用中通常忽略这一点）。

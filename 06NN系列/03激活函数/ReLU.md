# ReLU 激活函数 (Rectified Linear Unit)

## 1. 定义与公式
**ReLU (Rectified Linear Unit)** 是目前深度学习中最常用的激活函数。它的定义非常简单：保留正数，将负数置为 0。

**公式**：
$$ f(x) = \max(0, x) = \begin{cases} x, & x \ge 0 \\ 0, & x < 0 \end{cases} $$

## 2. 导函数

**导数公式**：
$$ f'(x) = \begin{cases} 1, & x > 0 \\ 0, & x < 0 \end{cases} $$
注意：在 $x=0$ 处导数未定义，但在实现中通常人为规定为 0 或 1（通常为 0）。

## 3. 优缺点分析

### 优点
1.  **计算简单**：只涉及阈值判断，没有复杂的指数或幂运算，计算速度极快。
2.  **解决梯度消失**：在正区间 ($x>0$) 导数恒为 1，梯度不会衰减，这使得深层网络的训练成为可能。
3.  **稀疏性 (Sparsity)**：负输入被置为 0，使得部分神经元处于非激活状态，增加了网络的稀疏性，有助于减少过拟合和提高鲁棒性。

### 缺点
1.  **Dead ReLU 问题**：如果神经元在某次更新后，其权重使得对所有输入的加权和都小于 0，那么该神经元将永远输出 0，梯度也永远为 0，参数无法再次更新。这导致神经元“死亡”。
2.  **非零中心化 (Not Zero-Centered)**：输出恒 $\ge 0$，导致后一层神经元的输入均值不为 0，可能影响收敛速度。

# 向量模型对比学习微调实战

向量模型微调的完整实战：使用 `sentence-transformers` 进行对比学习训练，含 Hard Negative Mining。

---

## 核心公式回顾

### InfoNCE / Multiple Negatives Ranking Loss

$$
\mathcal{L} = -\frac{1}{N} \sum_{i=1}^{N} \log \frac{\exp(\text{sim}(q_i, d_i^+) / \tau)}{\sum_{j=1}^{N} \exp(\text{sim}(q_i, d_j) / \tau)}
$$

---

## 实战代码

### 基础对比学习微调

```python
from sentence_transformers import (
    SentenceTransformer, SentenceTransformerTrainer,
    SentenceTransformerTrainingArguments,
    losses,
)
from datasets import Dataset

# 1. 加载模型
model = SentenceTransformer("BAAI/bge-base-en-v1.5")

# 2. 准备数据
# 格式：{"anchor": "query", "positive": "relevant doc"}
train_data = Dataset.from_dict({
    "anchor": [
        "什么是 LoRA？",
        "Python 如何读取 JSON？",
    ],
    "positive": [
        "LoRA 是一种参数高效微调方法，通过低秩矩阵分解...",
        "使用 json.load() 可以读取 JSON 文件...",
    ],
})

# 3. 损失函数
# MultipleNegativesRankingLoss: batch 内其他样本自动作为负例
loss = losses.MultipleNegativesRankingLoss(model)

# 4. 训练
args = SentenceTransformerTrainingArguments(
    output_dir="./embedding-ft",
    num_train_epochs=3,
    per_device_train_batch_size=64,   # 越大越好
    learning_rate=2e-5,
    warmup_ratio=0.1,
    bf16=True,
)

trainer = SentenceTransformerTrainer(
    model=model,
    args=args,
    train_dataset=train_data,
    loss=loss,
)
trainer.train()
model.save("./finetuned-embedding")
```

## Hard Negative Mining 代码

```python
from sentence_transformers import SentenceTransformer
import numpy as np

model = SentenceTransformer("BAAI/bge-base-en-v1.5")

def mine_hard_negatives(
    queries: list[str],
    corpus: list[str],
    positives: list[str],
    top_k: int = 10,
    num_neg: int = 3,
) -> list[list[str]]:
    """
    用当前模型召回 top-K，去掉正例后取最相似的作为 hard negatives
    """
    # 编码
    q_embs = model.encode(queries, normalize_embeddings=True)
    c_embs = model.encode(corpus, normalize_embeddings=True)

    # 计算相似度
    scores = q_embs @ c_embs.T  # (num_queries, num_corpus)

    hard_negatives = []
    for i, (query, pos) in enumerate(zip(queries, positives)):
        # 按相似度排序
        sorted_indices = np.argsort(-scores[i])

        negatives = []
        for idx in sorted_indices:
            if corpus[idx] != pos and len(negatives) < num_neg:
                negatives.append(corpus[idx])
        hard_negatives.append(negatives)

    return hard_negatives

# 使用
hard_negs = mine_hard_negatives(queries, corpus, positives)
```

## Matryoshka Embedding 代码

```python
from sentence_transformers import losses

# Matryoshka Loss: 同时优化多个维度截断
matryoshka_dims = [64, 128, 256, 512, 768]
inner_loss = losses.MultipleNegativesRankingLoss(model)
loss = losses.MatryoshkaLoss(model, inner_loss, matryoshka_dims)

# 训练同上
trainer = SentenceTransformerTrainer(
    model=model,
    args=args,
    train_dataset=train_data,
    loss=loss,
)
trainer.train()

# 推理时可选择维度
emb_768 = model.encode("query", normalize_embeddings=True)  # 完整 768 维
emb_256 = emb_768[:256]  # 截取前 256 维，仍然有效！
```

## 训练技巧

- **Batch size 越大越好**：更多 in-batch negatives = 更好的对比学习效果
- **Hard negatives 至关重要**：简单负例学不到东西
- **多阶段训练**：先 domain pretrain（无标注），再 retrieval finetune（有标注）
- **温度参数** $\tau$：通常 0.05-0.1，太小容易坍塌，太大区分度不够

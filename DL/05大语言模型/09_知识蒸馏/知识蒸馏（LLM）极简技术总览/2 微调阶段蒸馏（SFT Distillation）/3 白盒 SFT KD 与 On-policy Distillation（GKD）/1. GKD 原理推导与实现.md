GKD（Generalized Knowledge Distillation, ICLR 2024）让 student 在自己生成的轨迹上接受 teacher 反馈，解决 exposure bias。

---

## 1. 核心公式

$$L_{GKD} = \mathbb{E}_{x \sim D}\left[\mathbb{E}_{y \sim p_S(\cdot|x)}\left[\sum_{t} D_f(p_T(\cdot|x, y_{<t}) \| p_S(\cdot|x, y_{<t}))\right]\right]$$

> [!important] 关键区别

> 标准 KD：Student 在 Teacher 的 token 序列上学习。GKD：Student 在**自己的** token 序列上学习。

---

## 2. Python 实现

```Python
import torch
import torch.nn.functional as F

class GKDTrainer:
    def __init__(self, student, teacher, divergence="forward_kl", temperature=1.0):
        self.student = student
        self.teacher = teacher
        self.divergence = divergence
        self.T = temperature
        for p in self.teacher.parameters():
            p.requires_grad = False

    def train_step(self, batch):
        input_ids = batch['input_ids']
        # Step 1: On-policy generation
        with torch.no_grad():
            generated = self.student.generate(
                input_ids=input_ids, max_new_tokens=512,
                do_sample=True, temperature=0.8, top_p=0.95
            )
        # Step 2: Teacher logits on student's sequences
        with torch.no_grad():
            t_logits = self.teacher(input_ids=generated).logits[:, :-1, :]
        # Step 3: Student logits (with grad)
        s_logits = self.student(input_ids=generated).logits[:, :-1, :]
        # Step 4: Compute divergence on generated tokens
        prompt_len = input_ids.size(1)
        s_gen = s_logits[:, prompt_len-1:, :]
        t_gen = t_logits[:, prompt_len-1:, :]

        s_log_p = F.log_softmax(s_gen / self.T, dim=-1)
        t_p = F.softmax(t_gen / self.T, dim=-1)
        loss = F.kl_div(s_log_p, t_p, reduction="none").sum(-1).mean()
        return loss * (self.T ** 2)
```

## 3. 超参数

|参数|推荐|
|---|---|
|Divergence|Forward KL 或 JSD|
|On-policy ratio|0.3~0.5|
|Generation temp|0.8|

## 1. CoT 数据生成

```Python
def generate_structured_cot(teacher_client, instruction: str) -> dict:
    prompt = f"""Solve the following. First reasoning steps, then final answer.

Problem: {instruction}

Reasoning:
[step-by-step reasoning]

Final Answer:
[concise answer]"""
    response = teacher_client.generate(prompt, temperature=0.3)
    parts = response.split('Final Answer:')
    reasoning = parts[0].replace('Reasoning:', '').strip() if len(parts) > 1 else response
    answer = parts[1].strip() if len(parts) > 1 else ''
    return {'instruction': instruction, 'reasoning': reasoning, 'answer': answer, 'full_cot': response}
```

## 2. 质量控制：Self-Consistency

> [!important] 错误的推理链比没有推理链更有害

```Python
def self_consistency_cot(teacher_client, instruction, n_samples=5):
    samples = [generate_structured_cot(teacher_client, instruction) for _ in range(n_samples)]
    from collections import Counter
    answer_counts = Counter(s['answer'].strip().lower() for s in samples)
    majority = answer_counts.most_common(1)[0]
    best = next(s for s in samples if s['answer'].strip().lower() == majority[0])
    best['consistency'] = majority[1] / n_samples
    return best
```

## 3. Step-by-Step 双任务训练

$$L = L_{label}(y, \hat{y}) + \lambda \cdot L_{rationale}(r, \hat{r})$$

```Python
class StepByStepDistillation:
    def __init__(self, model, tokenizer, lambda_rationale=0.5):
        self.model = model
        self.lambda_r = lambda_rationale

    def compute_loss(self, batch):
        loss_answer = self.model(
            input_ids=batch['answer_input_ids'],
            labels=batch['answer_labels']
        ).loss
        loss_rationale = self.model(
            input_ids=batch['rationale_input_ids'],
            labels=batch['rationale_labels']
        ).loss
        return loss_answer + self.lambda_r * loss_rationale
```

> [!tip] 770M 模型通过此方法可超越使用 few-shot CoT 的 540B PaLM（Hsieh et al., 2023）

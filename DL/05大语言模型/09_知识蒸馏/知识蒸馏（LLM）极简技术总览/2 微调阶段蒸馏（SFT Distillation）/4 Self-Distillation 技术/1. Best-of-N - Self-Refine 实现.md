Best-of-N 和 Self-Refine 是自蒸馏的两大核心技术。

---

## 1. Best-of-N 实现

```Python
class BestOfNDistiller:
    def __init__(self, model_client, reward_model=None, n_samples=16):
        self.client = model_client
        self.rm = reward_model
        self.n = n_samples

    def select_best(self, prompt: str) -> dict:
        candidates = [
            self.client.generate(prompt, temperature=0.8)
            for _ in range(self.n)
        ]
        scores = [self.rm.score(prompt, c) for c in candidates]
        best_idx = max(range(len(scores)), key=lambda i: scores[i])
        return {'prompt': prompt, 'response': candidates[best_idx], 'score': scores[best_idx]}
```

> [!tip] Best-of-N 隐式做 Jeffreys divergence 优化，比 rejection sampling 更稳定。

## 2. Self-Refine 实现

```Python
class SelfRefineDistiller:
    def __init__(self, model_client, max_rounds=3):
        self.client = model_client
        self.max_rounds = max_rounds

    def refine(self, prompt: str) -> dict:
        response = self.client.generate(prompt, temperature=0.7)
        for r in range(self.max_rounds):
            critique = self.client.generate(
                f"Review and find issues:\nQ: {prompt}\nA: {response}\nIssues:",
                temperature=0.3
            )
            if 'no significant issues' in critique.lower():
                break
            response = self.client.generate(
                f"Improve based on critique:\nQ: {prompt}\nA: {response}\nCritique: {critique}\nImproved:",
                temperature=0.3
            )
        return {'prompt': prompt, 'response': response, 'rounds': r + 1}
```

## 3. 选择指南

|维度|Best-of-N|Self-Refine|
|---|---|---|
|需外部 RM|✅|❌|
|适用模型|任意|需足够强的自批判能力|
|稳定性|高|中（弱模型可能越改越差）|

本页面从数学角度严谨推导 Hinton KD 中 soft targets 与 temperature scaling 的原理。

---

## 1. 标准 Softmax 与温度缩放

给定 logits 向量 $z in mathbb{R}^V$（$V$ 为词表大小），标准 softmax：

$$p_i = \frac{\exp(z_i)}{\sum_{j=1}^{V} \exp(z_j)}$$

引入温度参数 $T > 0$ 后的 **tempered softmax**：

$$q_i^{(T)} = \frac{\exp(z_i / T)}{\sum_{j=1}^{V} \exp(z_j / T)}$$

**温度的效果**：

- $T = 1$：标准 softmax，分布保持原始尖锐度

- $T to 0^+$：分布趋向 one-hot（argmax）

- $T to infty$：分布趋向均匀分布 $1/V$

> [!important] 直觉

> 高温使分布"变软"，暴露出 teacher 对各类别相对关系的细微判断（"暗知识"）。例如 teacher 认为 "cat" 和 "kitten" 比 "car" 更相似——这种信息在 $T=1$ 的尖锐分布中几乎不可见。

---

## 2. KL 散度蒸馏损失

给定 teacher 的 tempered 分布 $q_T^{(T)}$ 和 student 的 tempered 分布 $q_S^{(T)}$：

$$L_{KD} = KL(q_T^{(T)} \| q_S^{(T)}) = \sum_{i=1}^{V} q_{T,i}^{(T)} \log \frac{q_{T,i}^{(T)}}{q_{S,i}^{(T)}}$$

展开后：

$$L_{KD} = -\sum_{i} q_{T,i}^{(T)} \log q_{S,i}^{(T)} + \sum_{i} q_{T,i}^{(T)} \log q_{T,i}^{(T)}$$

第二项为 teacher 分布的熵（常数），因此优化等价于最小化**交叉熵**：

$$L_{KD} \propto -\sum_{i} q_{T,i}^{(T)} \log q_{S,i}^{(T)} = H(q_T^{(T)}, q_S^{(T)})$$

---

## 3. $T^2$ 缩放因子的推导

当 $T$ 较大时，对 $L_{KD}$ 关于 student logits $z_S$ 求梯度：

$$\frac{\partial L_{KD}}{\partial z_{S,i}} = \frac{1}{T}(q_{S,i}^{(T)} - q_{T,i}^{(T)})$$

注意梯度中含 $1/T$ 因子。Hinton 提出乘以 $T^2$ 补偿，使梯度量级与硬标签损失可比：

$$L_{KD}^{scaled} = T^2 \cdot KL(q_T^{(T)} \| q_S^{(T)})$$

> [!tip] 为什么是 $T^2$

> 因为 KL 散度的梯度中有 $1/T$ 因子，而 KL 散度本身在高温下也近似缩小 $1/T$，两者相乘得 $1/T^2$。乘以 $T^2$ 恢复梯度的原始量级。

---

## 4. 完整蒸馏损失

$$L_{total} = \alpha \cdot L_{CE}(y, p_S) + (1 - \alpha) \cdot T^2 \cdot KL(q_T^{(T)} \| q_S^{(T)})$$

其中：

- $L_{CE}(y, p_S)$：student 在 $T=1$ 时与真实标签的交叉熵（硬标签损失）

- $KL(q_T^{(T)} | q_S^{(T)})$：teacher 和 student 在温度 $T$ 下的 KL 散度（软标签损失）

- $alpha$：平衡系数，通常 $\alpha \in [0.1, 0.5]$

- $T$：温度，通常 $T \in [2, 20]$

---

## 5. 高温近似分析

当 $T \gg \max(|z_i|)$ 时：

$$q_i^{(T)} \approx \frac{1 + z_i/T}{V + \sum_j z_j/T} \approx \frac{1}{V} + \frac{z_i}{VT}$$

此时 KL 散度近似为：

$$KL(q_T^{(T)} \| q_S^{(T)}) \approx \frac{1}{2VT^2} \|z_T - z_S\|_2^2$$

> [!important] 关键结论

> 在高温极限下，KL 蒸馏退化为 logits 的 **MSE 损失**。这解释了为什么有些工作直接用 MSE(logits) 也能取得不错的效果。

## 6. Python 实现

```Python
import torch
import torch.nn.functional as F


def hinton_distillation_loss(
    student_logits: torch.Tensor,  # (B, V)
    teacher_logits: torch.Tensor,  # (B, V)
    labels: torch.Tensor,          # (B,)
    temperature: float = 4.0,
    alpha: float = 0.3,
) -> torch.Tensor:
    """Hinton KD loss: hard label CE + soft label KL."""
    # Hard label loss (T=1)
    loss_ce = F.cross_entropy(student_logits, labels)

    # Soft label loss (tempered)
    student_soft = F.log_softmax(student_logits / temperature, dim=-1)
    teacher_soft = F.softmax(teacher_logits / temperature, dim=-1)
    loss_kd = F.kl_div(
        student_soft,
        teacher_soft,
        reduction="batchmean",
    )

    # Combined loss with T^2 scaling
    loss = alpha * loss_ce + (1 - alpha) * (temperature ** 2) * loss_kd
    return loss


# --- Usage example ---
B, V = 8, 32000  # batch size, vocab size
student_logits = torch.randn(B, V, requires_grad=True)
teacher_logits = torch.randn(B, V)
labels = torch.randint(0, V, (B,))

loss = hinton_distillation_loss(
    student_logits, teacher_logits, labels,
    temperature=4.0, alpha=0.3,
)
loss.backward()
print(f"Distillation loss: {loss.item():.4f}")
```

## 7. 超参数选择指南

|超参数|推荐范围|选择逻辑|
|---|---|---|
|$T$|2~10|teacher 越强、student 越弱 → T 越大；分类任务常用 4~6|
|$\alpha$|0.1~0.5|数据充足 → 偏低（更多依赖 teacher）；数据稀缺 → 偏高|
|teacher size|2x~10x student|gap 过大效果反降|

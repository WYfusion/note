## 1. DistilBERT 实现

### 1.1 架构

- Teacher：BERT-base（12层, 768d, 110M）

- Student：6层, 768d, 66M（隔层初始化）

- 初始化策略：student 第 $i$ 层从 teacher 第 $2i$ 层复制权重

### 1.2 三重损失

$$L = L_{MLM} + L_{KD} + L_{cos}$$

```Python
import torch
import torch.nn.functional as F


def distilbert_loss(
    student_logits: torch.Tensor,   # (B, L, V)
    teacher_logits: torch.Tensor,   # (B, L, V)
    student_hidden: torch.Tensor,   # (B, L, 768)
    teacher_hidden: torch.Tensor,   # (B, L, 768)
    labels: torch.Tensor,           # (B, L) MLM labels
    temperature: float = 2.0,
) -> torch.Tensor:
    """DistilBERT triple loss."""
    # 1. MLM loss
    loss_mlm = F.cross_entropy(
        student_logits.view(-1, student_logits.size(-1)),
        labels.view(-1),
        ignore_index=-100,
    )

    # 2. KD loss (soft labels)
    loss_kd = F.kl_div(
        F.log_softmax(student_logits / temperature, dim=-1),
        F.softmax(teacher_logits / temperature, dim=-1),
        reduction="batchmean",
    ) * (temperature ** 2)

    # 3. Cosine embedding loss
    cos_target = torch.ones(student_hidden.size(0) * student_hidden.size(1))
    cos_target = cos_target.to(student_hidden.device)
    loss_cos = F.cosine_embedding_loss(
        student_hidden.reshape(-1, student_hidden.size(-1)),
        teacher_hidden.reshape(-1, teacher_hidden.size(-1)),
        cos_target,
    )

    return loss_mlm + loss_kd + loss_cos
```

## 2. TinyBERT 实现

### 2.1 架构

- Teacher：BERT-base（12层）

- Student：4层（映射 teacher 的 {3, 6, 9, 12} 层）

- 需要线性变换矩阵 $W_h, W_e$ 对齐维度

### 2.2 四重蒸馏损失

```Python
import torch
import torch.nn as nn
import torch.nn.functional as F


class TinyBERTLoss(nn.Module):
    def __init__(self, student_dim=312, teacher_dim=768):
        super().__init__()
        # 线性映射：student dim -> teacher dim
        self.hidden_proj = nn.Linear(student_dim, teacher_dim, bias=False)
        self.embed_proj = nn.Linear(student_dim, teacher_dim, bias=False)

    def attention_loss(self, student_attn, teacher_attn):
        """MSE between attention matrices.
        student_attn, teacher_attn: (B, num_heads, L, L)
        """
        return F.mse_loss(student_attn, teacher_attn)

    def hidden_loss(self, student_hidden, teacher_hidden):
        """MSE between projected hidden states."""
        projected = self.hidden_proj(student_hidden)
        return F.mse_loss(projected, teacher_hidden)

    def embedding_loss(self, student_embed, teacher_embed):
        """MSE between projected embeddings."""
        projected = self.embed_proj(student_embed)
        return F.mse_loss(projected, teacher_embed)

    def prediction_loss(self, student_logits, teacher_logits, T=1.0):
        """Soft cross-entropy at temperature T."""
        return F.kl_div(
            F.log_softmax(student_logits / T, dim=-1),
            F.softmax(teacher_logits / T, dim=-1),
            reduction="batchmean",
        ) * (T ** 2)

    def forward(
        self,
        s_embed, t_embed,
        s_hiddens, t_hiddens,  # lists of (B, L, D)
        s_attns, t_attns,      # lists of (B, H, L, L)
        s_logits, t_logits,
        layer_map=None,        # e.g. {0:2, 1:5, 2:8, 3:11}
    ):
        if layer_map is None:
            layer_map = {0: 2, 1: 5, 2: 8, 3: 11}

        loss = self.embedding_loss(s_embed, t_embed)

        for s_idx, t_idx in layer_map.items():
            loss += self.hidden_loss(s_hiddens[s_idx], t_hiddens[t_idx])
            loss += self.attention_loss(s_attns[s_idx], t_attns[t_idx])

        loss += self.prediction_loss(s_logits, t_logits)
        return loss
```

## 3. MiniLM 实现

### 3.1 架构

- 只蒸馏 teacher 最后一层的 self-attention value relation

- 不需要层映射，不需要维度对齐

### 3.2 Value-Relation Transfer

$$A^{(V)} = \text{softmax}\left(\frac{V \cdot V^T}{\sqrt{d_v}}\right)$$

```Python
import torch
import torch.nn.functional as F


def minilm_loss(
    student_values: torch.Tensor,  # (B, H, L, d_v)
    teacher_values: torch.Tensor,  # (B, H', L, d_v')
):
    """MiniLM value-relation distillation.
    Distills self-attention value relations from teacher's
    last layer to student's last layer.
    """
    # Compute value-relation matrices
    # student: (B, H, L, L)
    s_rel = torch.matmul(student_values, student_values.transpose(-2, -1))
    s_rel = s_rel / (student_values.size(-1) ** 0.5)
    s_rel = F.log_softmax(s_rel, dim=-1)

    # teacher: (B, H', L, L)
    t_rel = torch.matmul(teacher_values, teacher_values.transpose(-2, -1))
    t_rel = t_rel / (teacher_values.size(-1) ** 0.5)
    t_rel = F.softmax(t_rel, dim=-1)

    # If different number of heads, average teacher heads
    if s_rel.size(1) != t_rel.size(1):
        # Reshape and average
        ratio = t_rel.size(1) // s_rel.size(1)
        t_rel = t_rel.view(
            t_rel.size(0), s_rel.size(1), ratio,
            t_rel.size(2), t_rel.size(3)
        ).mean(dim=2)

    # KL divergence
    loss = F.kl_div(s_rel, t_rel, reduction="batchmean")
    return loss
```

## 4. 三者工程对比

|维度|DistilBERT|TinyBERT|MiniLM|
|---|---|---|---|
|实现复杂度|低（直接训练）|高（需层映射+线性投影）|中（只需 value 提取）|
|额外参数|无|$W_h, W_e$ 投影矩阵|无|
|训练阶段|预训练|预训练 + 微调|预训练|
|维度兼容|需相同|投影对齐|自动兼容|

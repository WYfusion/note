这些方法统一的目标：不训练显式的 reward model，直接用偏好数据优化 policy。它们是当前对齐蒸馏最流行的落地方案。

---

## 1. DPO（Direct Preference Optimization）

> [!info] 核心创新

> 证明 RLHF 的 reward model + PPO 可以等价转化为一个直接在偏好数据上的分类问题。

**损失函数**：

$$L_{DPO} = -\mathbb{E}_{(x,y_w,y_l)}\left[\log \sigma\left(\beta \log \frac{\pi_\theta(y_w|x)}{\pi_{ref}(y_w|x)} - \beta \log \frac{\pi_\theta(y_l|x)}{\pi_{ref}(y_l|x)}\right)\right]$$

**直觉**：增大 chosen response 的概率、降低 rejected response 的概率，相对于 reference model。

**特点**：

- 需要 reference model（$pi_{ref}$，通常为 SFT 后的模型）

- 不需要 reward model

- 训练稳定性远优于 PPO

- 现已成为最常用的离线偏好优化方法

---

## 2. ORPO（Odds Ratio Preference Optimization）

> [!info] 核心创新

> 在 odds-space（而非 log probability space）中做偏好优化，将 SFT 和 preference optimization 合并为单一训练阶段。

**损失函数**：

$$L_{ORPO} = L_{SFT} + \lambda \cdot L_{OR}$$

其中 $L_{OR} = -log sigmaleft(log frac{odds(y_w)}{odds(y_l)}right)$，$odds(y) = frac{p(y)}{1-p(y)}$

**特点**：

- 无需 reference model

- 无需单独的 SFT 阶段

- 一个阶段完成 SFT + 偏好对齐

---

## 3. SimPO（NeurIPS 2024）

> [!info] 核心创新

> Reference-free，用序列平均 log probability 作为隐式奖励函数。

**隐式奖励**：

$$r(x, y) = \frac{\beta}{|y|} \log \pi_\theta(y|x)$$

**损失函数**：

$$L_{SimPO} = -\mathbb{E}\left[\log \sigma\left(\frac{\beta}{|y_w|} \log \pi_\theta(y_w|x) - \frac{\beta}{|y_l|} \log \pi_\theta(y_l|x) - \gamma\right)\right]$$

其中 $\gamma$ 为 margin 项，鼓励更大的偏好差距。

**特点**：

- 无需 reference model（训练更高效）

- 长度归一化避免长度偏差

- SimPO 在多个 benchmark 上超越 DPO

---

## 4. KTO（Kahneman-Tversky Optimization）

> [!info] 核心创新

> 基于 prospect theory，只需要二元信号（好/坏），不需要成对偏好数据。

**动机**：现实中收集"好"或"坏"的单条标注比收集成对偏好更容易。

**损失函数**（简化）：

$$L_{KTO} = \mathbb{E}_{y \sim D^+}[\lambda_+ \cdot \sigma(-r(x,y))] + \mathbb{E}_{y \sim D^-}[\lambda_- \cdot \sigma(r(x,y))]$$

**特点**：

- 不需要 pairwise preference 数据

- 只需要 binary desirability labels

- 更容易扩展数据收集

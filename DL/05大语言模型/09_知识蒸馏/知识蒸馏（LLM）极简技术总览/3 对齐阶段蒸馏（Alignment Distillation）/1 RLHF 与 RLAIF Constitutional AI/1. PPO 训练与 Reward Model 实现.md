## 1. Reward Model 训练

基于 Bradley-Terry 模型：

$$P(y_w \succ y_l | x) = \sigma(r_\theta(x, y_w) - r_\theta(x, y_l))$$

$$L_{RM} = -\mathbb{E}[\log \sigma(r_\theta(x, y_w) - r_\theta(x, y_l))]$$

```Python
import torch
import torch.nn as nn

class RewardModel(nn.Module):
    def __init__(self, base_model):
        super().__init__()
        self.backbone = base_model
        self.reward_head = nn.Linear(base_model.config.hidden_size, 1)

    def forward(self, input_ids, attention_mask):
        outputs = self.backbone(input_ids=input_ids, attention_mask=attention_mask)
        last_hidden = outputs.last_hidden_state
        # Pool: use last token
        seq_lengths = attention_mask.sum(dim=1) - 1
        pooled = last_hidden[range(len(seq_lengths)), seq_lengths]
        return self.reward_head(pooled).squeeze(-1)

def reward_model_loss(rm, chosen_ids, chosen_mask, rejected_ids, rejected_mask):
    r_chosen = rm(chosen_ids, chosen_mask)
    r_rejected = rm(rejected_ids, rejected_mask)
    return -torch.log(torch.sigmoid(r_chosen - r_rejected)).mean()
```

## 2. PPO 优化

**目标**：

$$\max_{\pi} \mathbb{E}_{x,y \sim \pi}[r(x,y)] - \beta \cdot KL(\pi \| \pi_{ref})$$

**PPO Clipped Objective**：

$$L_{PPO} = \min\left(\frac{\pi_\theta}{\pi_{old}} A_t, \text{clip}\left(\frac{\pi_\theta}{\pi_{old}}, 1-\epsilon, 1+\epsilon\right) A_t\right)$$

```Python
def ppo_step(policy, ref_policy, value_model, reward_model,
             prompts, old_log_probs, beta=0.1, clip_eps=0.2):
    # Generate responses
    responses = policy.generate(prompts, do_sample=True)

    # Compute rewards
    rewards = reward_model(prompts, responses)

    # KL penalty
    log_probs = policy.log_prob(prompts, responses)
    ref_log_probs = ref_policy.log_prob(prompts, responses)
    kl_penalty = log_probs - ref_log_probs
    adjusted_rewards = rewards - beta * kl_penalty

    # Advantages (simplified)
    values = value_model(prompts, responses)
    advantages = adjusted_rewards - values.detach()

    # PPO clipped loss
    ratio = (log_probs - old_log_probs).exp()
    surr1 = ratio * advantages
    surr2 = torch.clamp(ratio, 1 - clip_eps, 1 + clip_eps) * advantages
    policy_loss = -torch.min(surr1, surr2).mean()

    # Value loss
    value_loss = (values - adjusted_rewards).pow(2).mean()

    return policy_loss + 0.5 * value_loss
```

> [!faq] PPO 训练为什么困难？

> - reward hacking：模型利用 RM 漏洞

> - 需同时维护 4 个模型（policy, ref, reward, value）

> - KL 系数、clip ratio、lr 需精细调参

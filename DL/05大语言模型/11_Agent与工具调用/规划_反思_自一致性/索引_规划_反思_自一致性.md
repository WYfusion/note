# 规划、反思与自一致性

本章节探讨 Agent 如何处理复杂任务，通过规划分解步骤，通过反思修正错误，通过自一致性提高鲁棒性。

## 目录

### [01_TreeOfThought_SelfConsistency.md](01_TreeOfThought_SelfConsistency.md)
- **Chain of Thought (CoT)**：语音情感分析的推理链。
- **Tree of Thoughts (ToT)**：探索多种配乐方案的决策树。
- **Self-Consistency**：
  - ASR 多次采样投票纠错。
  - TTS/MusicGen 多次生成优选。
- **任务规划**：播客制作的全流程分解 (Script -> TTS -> Mix)。

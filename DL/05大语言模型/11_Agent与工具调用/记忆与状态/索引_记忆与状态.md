# 记忆与状态管理

本章节介绍 Agent 如何维护长期和短期记忆，特别是 Audio Agent 特有的声纹与偏好记忆。

## 目录

### [01_短期记忆_对话摘要_Context管理.md](01_短期记忆_对话摘要_Context管理.md)
- **记忆分类**：Short-term (Context) vs Long-term (Vector DB)。
- **Audio Memory**：
  - **声纹记忆**：识别用户身份 (Speaker Embedding)。
  - **偏好记忆**：记录 TTS 音色、语速、音乐风格偏好。
  - **资产记忆**：检索历史生成的音频文件 (CLAP)。
- **Context 管理**：ASR 文本摘要与状态机 (FSM) 管理。

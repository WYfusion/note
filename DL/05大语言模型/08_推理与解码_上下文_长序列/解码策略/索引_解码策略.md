# 解码策略

本章节涵盖了 LLM 推理时的解码方法，从确定性搜索（Greedy/Beam）到随机采样（Top-K/Top-P），并针对语音生成的特殊问题（如重复循环）提供了解决方案。

## 目录

### [01_Greedy_BeamSearch.md](01_Greedy_BeamSearch.md)
- **Greedy Search**：每步选最大概率 Token，简单但易陷入局部最优。
- **Beam Search**：保留 Top-K 候选路径，长度惩罚公式。
- **语音应用**：
  - ASR 中的 Beam Search + 语言模型。
  - 投机采样（Speculative Decoding）加速 TTS。
  - 强制对齐（Forced Alignment）生成时间戳。

### [02_TopK_TopP_温度_重复惩罚.md](02_TopK_TopP_温度_重复惩罚.md)
- **采样策略**：Top-K, Top-P (Nucleus), Temperature 调节。
- **重复惩罚**：Repetition Penalty 公式。
- **语音特殊性**：
  - Audio Token 的"结巴"问题与强惩罚策略。
  - 温度对语音风格（机器音 vs 情感丰富）的影响。
  - 动态温度调节（句首高温、句中低温）。

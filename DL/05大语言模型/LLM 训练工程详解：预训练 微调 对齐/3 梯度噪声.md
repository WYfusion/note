# 3. 梯度噪声

梯度噪声 = mini-batch gradient 相对于 full-batch gradient 的随机波动（方差）。它是连接 batch size、LR、收敛速度的桥梁概念。

---

## 本质

每个 mini-batch 采样的梯度 $g_B$ 是 full-batch 梯度 $g$ 的无偏估计：

$E[g_B] = g, \quad \text{Var}(g_B) \propto \frac{\sigma^2}{B}$

其中 $B$ 为 batch size，$sigma^2$ 为数据固有方差。batch 越大，噪声越小，但收益递减。

---

## 影响

- **决定可用 batch 上限**：噪声降到一定水平后再加大 batch 不再有收益（critical batch size）
- **影响 LR**：高噪声 → 需要更小 LR 保稳定
- **影响收敛速度与稳定性**：噪声过大时 update 方向不一致，loss 抖动大

---

## 观察代理

梯度噪声无法直接测量，但可通过代理指标间接观察：

1. **step-to-step grad norm 波动**：方差大 → 噪声大
2. **loss 方差**：连续 step 的 loss 变化幅度
3. **batch 增大后收益是否递减**：同配置下翻倍 batch，loss 曲线无改善 → 已过 critical batch size

---

## 常见结论

<aside>
📊

- **噪声大** → 需要更小 LR / 更长 warmup / 更强 grad clipping
- **噪声小** → 可提高 batch size 或提升吞吐（不浪费计算）
</aside>

> McCandlish et al. (2018) 的 *critical batch size* 理论提供了定量框架，可参考论文 "An Empirical Model of Large-Batch Training"。

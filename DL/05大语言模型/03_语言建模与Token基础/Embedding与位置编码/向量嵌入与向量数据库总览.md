## 一句话理解
- **向量嵌入**：把任意对象（文本、图片、音频……）变成一组数字（向量），让"语义相近 = 数值距离近"
- **向量数据库**：把这些向量存起来，并能高效地找到最相似的那些
- **当前最主流方案**：Dense + Sparse（Hybrid）+ Rerank + Metadata Filter

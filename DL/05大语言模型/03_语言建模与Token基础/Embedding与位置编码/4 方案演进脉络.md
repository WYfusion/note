从经典到最新，梳理各领域向量嵌入与检索系统的技术演进路线。

<aside>
💡

**通俗理解**：技术不是一步到位的，了解"从哪来、到哪去"能帮你判断当前技术的成熟度和未来趋势。

</aside>

---

## 4.1 文本嵌入脉络

- **经典基础**：Word2Vec / GloVe / BM25
- **神经句向量经典**：SBERT（2019）
- **检索经典**：DPR（2020）、E5（2022）
- **当前流行开源**：E5/mE5、BGE、BGE-M3、Jina Embeddings、Nomic
- **当前流行闭源**：OpenAI `text-embedding-3-*`、Voyage
- **最新方向**：长上下文 embedding、Late Chunking、多向量检索、任务蒸馏、multilingual >100 语言

---

## 4.2 图像 / 多模态脉络

- **经典**：CLIP（2021）
- **流行升级**：SigLIP / SigLIP2
- **当前热门**：PDF/图表/视觉文档 embedding；统一 text+image embedding
- **最新方向**：多模态统一检索、视觉丰富文档 RAG、dense + late interaction 同时支持

---

## 4.3 音频 / 语音脉络

- **经典语音表征**：wav2vec 2.0
- **经典说话人**：x-vector → ECAPA-TDNN
- **音频-文本共享经典**：CLAP
- **最新方向**：更长时长、更细粒度、更强时序感知（T-CLAP / SLAP）

---

## 4.4 检索系统脉络

- **经典**：BM25 单独打天下
- **第一代 RAG**：chunk → dense embedding → vector DB → top-k
- **当前主流**：hybrid → rerank → answer
- **最新增强**：
    - Contextual Retrieval / Late Chunking
    - Multi-vector retrieval
    - Long-context embeddings
    - 文档级/页级/块级联合索引
    - 多路召回融合（dense + sparse + metadata + graph）

---

🔑

**关键洞察**：检索系统从"单一信号"（纯 BM25 或纯 dense）发展为"多信号融合+多阶段排序"，这是当前工程实践的核心范式。

</aside>

# 表示层与距离度量：向量空间的数学基础

> [!abstract] 摘要
> 表示层与距离度量是嵌入技术的数学基石。本章节深入探讨向量空间的理论基础，包括向量的数学定义、各种距离度量方法（余弦相似度、欧氏距离、内积等），以及近似最近邻算法的核心原理。这些数学工具为嵌入模型的训练、评估和应用提供了坚实的理论支撑。

## 0. 统一概念：什么是表示层与距离度量？

> [!important] 表示层与距离度的核心作用
> **表示层**是将原始数据映射到向量空间的过程，通过数学变换将离散的、高维的数据转换为连续的、低维的向量表示。**距离度量**是定义向量间相似度的数学方法，它决定了嵌入模型如何理解和比较不同样本之间的关系。

### 表示层的核心作用

| 作用 | 数学原理 | 实现方式 | 应用价值 |
|------|----------|----------|----------|
| **特征提取** | 线性/非线性变换 | 神经网络、矩阵运算 | 将复杂数据转换为数值向量 |
| **维度降维** | 流形学习、PCA | 降维算法、特征选择 | 解决维度灾难，提高计算效率 |
| **语义对齐** | 语义空间映射 | 对比学习、对比损失 | 将不同模态映射到统一空间 |
| **相似度计算** | 距离度量函数 | 余弦相似度、欧氏距离 | 量化样本间关系 |

### 距离度量的数学意义

| 距离类型 | 数学定义 | 几何意义 | 适用场景 |
|----------|----------|----------|----------|
| **欧氏距离** | $\sqrt{\sum_{i=1}^{n}(x_i-y_i)^2}$ | 直线距离 | 空间数据、连续变量 |
| **余弦相似度** | $\frac{x \cdot y}{\|x\| \|y\|}$ | 角度相似度 | 文本语义、方向性数据 |
| **曼哈顿距离** | $\sum_{i=1}^{n}|x_i-y_i|$ | 网格距离 | 离散数据、城市距离 |
| **KL散度** | $\sum_{i=1}^{n}x_i \log \frac{x_i}{y_i}$ | 信息差异 | 概率分布、信息论 |

## 1. 向量空间基础

### 1.1 向量与向量空间

> [!note] 向量空间的数学定义

**向量**：在 n 维空间中的有向线段，可以表示为 $v = (v_1, v_2, ..., v_n)$。

**向量空间**：满足以下条件的集合 V：
1. **加法封闭**：若 $u, v \in V$，则 $u + v \in V$
2. **标量乘法封闭**：若 $v \in V$，$\alpha \in \mathbb{R}$，则 $\alpha v \in V$
3. **加法结合律**：$(u + v) + w = u + (v + w)$
4. **加法单位元**：存在零向量 $0 \in V$，使得 $v + 0 = v$

```python
import numpy as np
from typing import List

class VectorSpace:
    def __init__(self, dimension: int):
        """初始化向量空间"""
        self.dimension = dimension
        self.zero_vector = np.zeros(dimension)

    def is_valid_vector(self, vector: np.ndarray) -> bool:
        """验证向量是否属于该空间"""
        return vector.shape == (self.dimension,)

    def vector_addition(self, v1: np.ndarray, v2: np.ndarray) -> np.ndarray:
        """向量加法"""
        if not (self.is_valid_vector(v1) and self.is_valid_vector(v2)):
            raise ValueError("向量维度不匹配")
        return v1 + v2

    def scalar_multiplication(self, vector: np.ndarray, scalar: float) -> np.ndarray:
        """标量乘法"""
        return scalar * vector

# 示例：2D 向量空间操作
space_2d = VectorSpace(2)
v1 = np.array([1, 2])
v2 = np.array([3, 4])

# 向量加法
sum_vector = space_2d.vector_addition(v1, v2)  # [4, 6]

# 标量乘法
scaled_vector = space_2d.scalar_multiplication(v1, 2)  # [2, 4]
```

### 1.2 向量范数与单位化

> [!important] 向量的度量属性

**范数（Norm）**：衡量向量长度的函数，满足以下性质：
1. **正定性**：$\|v\| \geq 0$，且 $\|v\| = 0$ 当且仅当 $v = 0$
2. **齐次性**：$\|\alpha v\| = |\alpha| \cdot \|v\|$
3. **三角不等式**：$\|u + v\| \leq \|u\| + \|v\|$

**常见范数类型**：
- L1 范数：$\|v\|_1 = \sum_{i=1}^{n}|v_i|$
- L2 范数：$\|v\|_2 = \sqrt{\sum_{i=1}^{n}v_i^2}$
- L∞ 范数：$\|v\|_\infty = \max_{i}|v_i|$
- p 范数：$\|v\|_p = \left(\sum_{i=1}^{n}|v_i|^p\right)^{1/p}$

```python
class VectorNorms:
    @staticmethod
    def l1_norm(vector: np.ndarray) -> float:
        """L1 范数"""
        return np.sum(np.abs(vector))

    @staticmethod
    def l2_norm(vector: np.ndarray) -> float:
        """L2 范数（欧几里得范数）"""
        return np.sqrt(np.sum(vector ** 2))

    @staticmethod
    def infinity_norm(vector: np.ndarray) -> float:
        """L∞ 范数"""
        return np.max(np.abs(vector))

    @staticmethod
    def p_norm(vector: np.ndarray, p: float) -> float:
        """p 范数"""
        return np.power(np.sum(np.abs(vector) ** p), 1/p)

    @staticmethod
    def normalize(vector: np.ndarray, norm_type: str = 'l2') -> np.ndarray:
        """向量单位化"""
        if norm_type == 'l1':
            norm = VectorNorms.l1_norm(vector)
        elif norm_type == 'l2':
            norm = VectorNorms.l2_norm(vector)
        elif norm_type == 'infinity':
            norm = VectorNorms.infinity_norm(vector)
        else:
            raise ValueError(f"不支持的范数类型: {norm_type}")

        if norm == 0:
            return vector
        return vector / norm

# 示例
vector = np.array([3, 4])

# 不同范数
l1_norm = VectorNorms.l1_norm(vector)  # 7
l2_norm = VectorNorms.l2_norm(vector)  # 5
inf_norm = VectorNorms.infinity_norm(vector)  # 4

# L2 单位化
normalized = VectorNorms.normalize(vector)  # [0.6, 0.8]
```

### 1.3 内积与角度计算

> [!tip] 向量间的几何关系

**内积（点积）**：$u \cdot v = \sum_{i=1}^{n}u_i v_i = \|u\| \|v\| \cos \theta$

其中 $\theta$ 是向量 u 和 v 的夹角。

```python
class VectorOperations:
    @staticmethod
    def dot_product(u: np.ndarray, v: np.ndarray) -> float:
        """内积计算"""
        return np.dot(u, v)

    @staticmethod
    def cosine_similarity(u: np.ndarray, v: np.ndarray) -> float:
        """余弦相似度"""
        dot_product = VectorOperations.dot_product(u, v)
        norm_u = VectorNorms.l2_norm(u)
        norm_v = VectorNorms.l2_norm(v)

        if norm_u == 0 or norm_v == 0:
            return 0
        return dot_product / (norm_u * norm_v)

    @staticmethod
    def angle_between_vectors(u: np.ndarray, v: np.ndarray) -> float:
        """计算向量夹角（弧度）"""
        cos_sim = VectorOperations.cosine_similarity(u, v)
        # 处理数值误差
        cos_sim = np.clip(cos_sim, -1, 1)
        return np.arccos(cos_sim)

    @staticmethod
    def angle_between_vectors_degrees(u: np.ndarray, v: np.ndarray) -> float:
        """计算向量夹角（度数）"""
        angle_rad = VectorOperations.angle_between_vectors(u, v)
        return np.degrees(angle_rad)

# 示例
u = np.array([1, 0])  # x轴
v = np.array([0, 1])  # y轴

# 内积
dot = VectorOperations.dot_product(u, v)  # 0

# 余弦相似度
cos_sim = VectorOperations.cosine_similarity(u, v)  # 0

# 夹角
angle = VectorOperations.angle_between_vectors(u, v)  # π/2 弧度
angle_deg = VectorOperations.angle_between_vectors_degrees(u, v)  # 90 度
```

## 2. 距离度量详解

### 2.1 欧氏距离与曼哈顿距离

> [!important] 几何距离与网格距离

**欧氏距离（Euclidean Distance）**：
$$d_{\text{euclid}}(x, y) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)^2}$$

**曼哈顿距离（Manhattan Distance）**：
$$d_{\text{manhattan}}(x, y) = \sum_{i=1}^{n}|x_i - y_i|$$

```python
class DistanceMetrics:
    @staticmethod
    def euclidean_distance(x: np.ndarray, y: np.ndarray) -> float:
        """欧氏距离"""
        return np.sqrt(np.sum((x - y) ** 2))

    @staticmethod
    def manhattan_distance(x: np.ndarray, y: np.ndarray) -> float:
        """曼哈顿距离"""
        return np.sum(np.abs(x - y))

    @staticmethod
    def minkowski_distance(x: np.ndarray, y: np.ndarray, p: float) -> float:
        """闵可夫斯基距离"""
        return np.power(np.sum(np.abs(x - y) ** p), 1/p)

# 示例
point1 = np.array([1, 2])
point2 = np.array([4, 6])

# 欧氏距离
eu_dist = DistanceMetrics.euclidean_distance(point1, point2)  # 5.0

# 曼哈顿距离
man_dist = DistanceMetrics.manhattan_distance(point1, point2)  # 7

# 闵可夫斯基距离（p=1 是曼哈顿，p=2 是欧氏）
minkowski_dist = DistanceMetrics.minkowski_distance(point1, point2, p=2)  # 5.0
```

### 2.2 余弦距离与夹角距离

> [!note] 方向相似性的度量

**余弦距离**：$d_{\text{cosine}}(x, y) = 1 - \cos(x, y) = 1 - \frac{x \cdot y}{\|x\| \|y\|}$

**夹角距离**：$d_{\text{angle}}(x, y) = \frac{\theta}{\pi}$，其中 $\theta$ 是向量夹角

```python
class CosineDistance:
    @staticmethod
    def cosine_distance(x: np.ndarray, y: np.ndarray) -> float:
        """余弦距离"""
        cos_sim = VectorOperations.cosine_similarity(x, y)
        return 1 - cos_sim

    @staticmethod
    def angle_distance(x: np.ndarray, y: np.ndarray) -> float:
        """夹角距离（0-1）"""
        angle = VectorOperations.angle_between_vectors(x, y)
        return angle / np.pi  # 归一化到 [0, 1]

# 示例
text1 = np.array([0.9, 0.1, 0.5])
text2 = np.array([0.8, 0.2, 0.6])
text3 = np.array([-0.9, -0.1, -0.5])  # 相反方向

# 余弦距离（方向相反，距离大）
cos_dist_12 = CosineDistance.cosine_distance(text1, text2)  # 小
cos_dist_13 = CosineDistance.cosine_distance(text1, text3)  # 2.0（最大）
```

### 2.3 信息论距离度量

> [!important] 概率分布的度量方法

**KL散度（Kullback-Leibler Divergence）**：
$$D_{\text{KL}}(P \| Q) = \sum_{i=1}^{n} P(i) \log \frac{P(i)}{Q(i)}$$

**JS散度（Jensen-Shannon Divergence）**：
$$D_{\text{JS}}(P \| Q) = \frac{1}{2}D_{\text{KL}}(P \| M) + \frac{1}{2}D_{\text{KL}}(Q \| M)$$
其中 $M = \frac{P + Q}{2}$

```python
import scipy.stats as stats

class InformationTheoryDistance:
    @staticmethod
    def kl_divergence(p: np.ndarray, q: np.ndarray) -> float:
        """KL散度"""
        # 确保概率和为1
        p = p / np.sum(p)
        q = q / np.sum(q)

        # 避免 log(0)
        p = np.clip(p, 1e-10, 1)
        q = np.clip(q, 1e-10, 1)

        return np.sum(p * np.log(p / q))

    @staticmethod
    def jensen_shannon_divergence(p: np.ndarray, q: np.ndarray) -> float:
        """JS散度"""
        m = 0.5 * (p + q)
        return 0.5 * InformationTheoryDistance.kl_divergence(p, m) + 0.5 * InformationTheoryDistance.kl_divergence(q, m)

    @staticmethod
    def hellinger_distance(p: np.ndarray, q: np.ndarray) -> float:
        """Hellinger距离"""
        return np.sqrt(1 - np.sqrt(np.sum(np.sqrt(p * q))))

# 示例：两个概率分布
dist1 = np.array([0.1, 0.4, 0.5])
dist2 = np.array([0.3, 0.3, 0.4])

# KL散度（不对称）
kl = InformationTheoryDistance.kl_divergence(dist1, dist2)  # ~0.105

# JS散度（对称）
js = InformationTheoryDistance.jensen_shannon_divergence(dist1, dist2)  # ~0.034
```

### 2.4 马氏距离

> [!tip] 考虑数据分布的距离

**马氏距离**：$d_{\text{mahalanobis}}(x, y) = \sqrt{(x - y)^T \Sigma^{-1} (x - y)}$

其中 $\Sigma$ 是协方差矩阵。

```python
class MahalanobisDistance:
    def __init__(self, data: np.ndarray):
        """初始化马氏距离计算器"""
        self.mean = np.mean(data, axis=0)
        self.cov = np.cov(data, rowvar=False)
        self.inv_cov = np.linalg.inv(self.cov)

    def compute(self, x: np.ndarray, y: np.ndarray) -> float:
        """计算两个点的马氏距离"""
        diff = x - y
        return np.sqrt(diff @ self.inv_cov @ diff)

    def compute_to_center(self, x: np.ndarray) -> float:
        """计算点到中心的马氏距离"""
        return self.compute(x, self.mean)

# 示例：二维数据
data = np.random.randn(100, 2)  # 100个二维点

# 创建马氏距离计算器
mahalanobis = MahalanobisDistance(data)

# 计算距离
point1 = np.array([1, 2])
point2 = np.array([2, 1])
distance = mahalanobis.compute(point1, point2)
```

## 3. 近似最近邻搜索

### 3.1 空间划分与索引结构

> [!important] ANN 搜索的基本原理

**近似最近邻（Approximate Nearest Neighbor, ANN）** 是在保证一定精度的前提下，大幅加速最近邻搜索的算法。

**主要 ANN 算法**：
- **KD-Tree**：空间划分树
- **Ball-Tree**：球体划分树
- **LSH（局部敏感哈希）**：哈希近似
- **HNSW（Hierarchical Navigable Small World）**：层次化小世界
- **IVF（Inverted File）**：倒排索引

```python
class ANNBase:
    """ANN 算法基类"""
    def __init__(self, data: np.ndarray):
        self.data = data

    def build_index(self):
        """构建索引"""
        raise NotImplementedError

    def query(self, query_vector: np.ndarray, k: int = 5) -> List[tuple]:
        """查询最近邻"""
        raise NotImplementedError

class KDTreeANN(ANNBase):
    """KD-Tree 实现"""
    def __init__(self, data: np.ndarray):
        super().__init__(data)
        self.tree = None

    def build_index(self):
        """构建 KD-Tree 索引"""
        from scipy.spatial import cKDTree
        self.tree = cKDTree(self.data)

    def query(self, query_vector: np.ndarray, k: int = 5) -> List[tuple]:
        """查询最近邻"""
        if self.tree is None:
            self.build_index()

        distances, indices = self.tree.query(query_vector, k=k)

        # 转换为 (index, distance) 格式
        results = []
        if isinstance(distances, np.ndarray):
            for i, (dist, idx) in enumerate(zip(distances, indices)):
                results.append((idx, dist))
        else:
            # 单个查询结果
            results.append((indices, distances))

        return results

# 示例
data = np.random.rand(1000, 32)  # 1000个32维向量

# 创建并使用 KD-Tree
ann = KDTreeANN(data)
ann.build_index()

# 查询
query = np.random.rand(32)
neighbors = ann.query(query, k=5)  # 返回5个最近邻
```

### 3.2 LSH 局部敏感哈希

> [!note] 基于哈希的近似搜索

**局部敏感哈希（LSH）**的核心思想：
- 相似的向量更有可能被哈希到相同的桶
- 不同哈希函数产生独立的结果

```python
import hashlib

class LSH:
    def __init__(self, dimension: int, num_planes: int = 10):
        """初始化 LSH"""
        self.dimension = dimension
        self.num_planes = num_planes
        # 随机生成超平面
        self.planes = np.random.randn(num_planes, dimension)

    def hash_vector(self, vector: np.ndarray, num_table: int = 1) -> List[int]:
        """向量的 LSH 哈希"""
        hashes = []

        for i in range(num_table):
            # 选择 num_planes 个超平面
            selected_planes = self.planes[i * self.num_planes // num_table : (i + 1) * self.num_planes // num_table]

            # 计算投影符号
            projections = np.dot(vector, selected_planes.T)
            binary_hash = ''.join(['1' if x > 0 else '0' for x in projections])

            # 转换为整数
            hash_value = int(binary_hash, 2)
            hashes.append(hash_value)

        return hashes

    def similar_vectors(self, vector: np.ndarray, dataset: np.ndarray, threshold: float = 0.8) -> List[tuple]:
        """查找相似向量"""
        similar = []

        for i, data_vector in enumerate(dataset):
            sim = VectorOperations.cosine_similarity(vector, data_vector)
            if sim >= threshold:
                similar.append((i, sim))

        # 按相似度排序
        similar.sort(key=lambda x: x[1], reverse=True)
        return similar

# 示例
dimension = 32
lsh = LSH(dimension, num_planes=20)

dataset = np.random.rand(1000, dimension)

# 哈希查询向量
query = np.random.rand(32)
hashes = lsh.hash_vector(query)

# 查找相似向量
similar = lsh.similar_vectors(query, dataset, threshold=0.7)
```

### 3.3 HNSW 层次化小世界

> [!important] 高性能 ANN 算法

**HNSW（Hierarchical Navigable Small World）**：
- 多层图结构，高层连接稀疏，底层连接密集
- 通过贪婪搜索快速找到最近邻

```python
class HNSWNode:
    """HNSW 图节点"""
    def __init__(self, vector: np.ndarray, id: int):
        self.vector = vector
        self.id = id
        self.connections = {}  # level -> [neighbor_ids]

    def add_connection(self, level: int, neighbor_id: int):
        if level not in self.connections:
            self.connections[level] = []
        self.connections[level].append(neighbor_id)

class HNSW:
    """HNSW 实现（简化版）"""
    def __init__(self, max_connections: int = 16, levels: int = 4):
        self.max_connections = max_connections
        self.levels = levels
        self.nodes = {}
        self.entry_point = None

    def add_node(self, vector: np.ndarray):
        """添加节点"""
        node_id = len(self.nodes)
        node = HNSWNode(vector, node_id)

        # 从顶层开始
        self._insert_node(node)
        self.nodes[node_id] = node

        return node_id

    def _insert_node(self, new_node: HNSWNode):
        """插入节点"""
        if not self.nodes:
            # 第一个节点
            new_node.add_connection(0, new_node.id)
            self.entry_point = new_node.id
            return

        # 从高层开始搜索
        current_level = min(len(self.nodes) // 10, self.levels - 1)
        current_id = self.entry_point

        # 下降搜索
        while current_level >= 0:
            # 贪心搜索
            best_id = current_id
            best_dist = float('inf')

            for neighbor_id in self.nodes[current_id].connections.get(current_level, []):
                dist = DistanceMetrics.euclidean_distance(
                    new_node.vector, self.nodes[neighbor_id].vector
                )
                if dist < best_dist:
                    best_dist = dist
                    best_id = neighbor_id

            current_id = best_id
            current_level -= 1

        # 连接到最近邻
        for level in range(len(self.nodes) // 10 + 1):
            if level < self.levels:
                # 限制连接数
                if level not in self.nodes[current_id].connections:
                    self.nodes[current_id].connections[level] = []

                if len(self.nodes[current_id].connections[level]) < self.max_connections:
                    self.nodes[current_id].connections[level].append(new_node.id)
                    new_node.add_connection(level, current_id)

    def search(self, query: np.ndarray, k: int = 5) -> List[tuple]:
        """搜索最近邻"""
        if not self.nodes:
            return []

        # 简化版：线性搜索
        distances = []
        for node_id, node in self.nodes.items():
            dist = DistanceMetrics.euclidean_distance(query, node.vector)
            distances.append((node_id, dist))

        # 排序
        distances.sort(key=lambda x: x[1])
        return distances[:k]

# 示例
hnsw = HNSW(max_connections=16, levels=4)

# 添加节点
for i in range(1000):
    vector = np.random.rand(32)
    hnsw.add_node(vector)

# 搜索
query = np.random.rand(32)
neighbors = hnsw.search(query, k=5)
```

## 4. 高级距离优化技术

### 4.1 批量距离计算优化

> [!tip] 向量化计算提升性能

```python
class BatchDistanceCalculator:
    @staticmethod
    def batch_euclidean_distance(X: np.ndarray, Y: np.ndarray) -> np.ndarray:
        """批量计算欧氏距离矩阵"""
        # X: (n, d), Y: (m, d)
        # 输出: (n, m)

        # 使用广播
        diff = X[:, np.newaxis, :] - Y[np.newaxis, :, :]  # (n, m, d)
        distances = np.sqrt(np.sum(diff ** 2, axis=2))  # (n, m)

        return distances

    @staticmethod
    def batch_cosine_similarity(X: np.ndarray, Y: np.ndarray) -> np.ndarray:
        """批量计算余弦相似度矩阵"""
        # X: (n, d), Y: (m, d)
        # 输出: (n, m)

        # 归一化
        X_norm = X / np.linalg.norm(X, axis=1, keepdims=True)
        Y_norm = Y / np.linalg.norm(Y, axis=1, keepdims=True)

        # 矩阵乘法
        similarities = X_norm @ Y_norm.T

        return similarities

    @staticmethod
    def optimized_cosine(X: np.ndarray, Y: np.ndarray) -> np.ndarray:
        """优化的余弦相似度计算"""
        # 使用 scipy 的 cdist
        from scipy.spatial.distance import cdist
        similarities = 1 - cdist(X, Y, 'cosine')
        return similarities

# 示例
n_queries = 100
n_database = 10000
dimension = 128

queries = np.random.randn(n_queries, dimension)
database = np.random.randn(n_database, dimension)

# 批量计算
eu_distances = BatchDistanceCalculator.batch_euclidean_distance(queries, database)
cos_similarities = BatchDistanceCalculator.batch_cosine_similarity(queries, database)
```

### 4.2 近似距离计算

> [!important] 平衡精度与速度

```python
class ApproximateDistance:
    def __init__(self, data: np.ndarray, sample_ratio: float = 0.1):
        """初始化近似距离计算器"""
        self.data = data
        self.sample_size = int(len(data) * sample_ratio)
        self.sample_indices = np.random.choice(len(data), self.sample_size, replace=False)
        self.sample_data = data[self.sample_indices]

    def approximate_euclidean(self, query: np.ndarray, k: int = 5) -> List[tuple]:
        """近似欧氏距离搜索"""
        # 只计算采样点的距离
        distances = []
        for i, sample_idx in enumerate(self.sample_indices):
            dist = DistanceMetrics.euclidean_distance(query, self.sample_data[i])
            distances.append((sample_idx, dist))

        # 排序并返回前 k 个
        distances.sort(key=lambda x: x[1])
        return distances[:k]

    def approximate_cosine(self, query: np.ndarray, k: int = 5) -> List[tuple]:
        """近似余弦相似度搜索"""
        # 计算与采样点的相似度
        similarities = []
        for i, sample_idx in enumerate(self.sample_indices):
            sim = VectorOperations.cosine_similarity(query, self.sample_data[i])
            similarities.append((sample_idx, sim))

        # 按相似度排序
        similarities.sort(key=lambda x: x[1], reverse=True)
        return similarities[:k]

# 示例
data = np.random.rand(10000, 128)  # 10000个向量
approx_dist = ApproximateDistance(data, sample_ratio=0.1)  # 采样1000个

query = np.random.rand(128)

# 近似搜索
approx_neighbors = approx_dist.approximate_cosine(query, k=5)
```

```python
class DocumentSimilaritySystem:
    def __init__(self, embeddings: np.ndarray):
        """初始化文档相似度系统"""
        self.embeddings = embeddings
        self.ann_index = None

    def build_ann_index(self, method='hnsw'):
        """构建 ANN 索引"""
        if method == 'hnsw':
            self.ann_index = HNSW()
            for embedding in self.embeddings:
                self.ann_index.add_node(embedding)

    def find_similar_documents(self, query_embedding: np.ndarray, top_k: int = 5) -> List[tuple]:
        """查找相似文档"""
        if self.ann_index is None:
            self.build_ann_index()

        # 使用 ANN 搜索
        neighbors = self.ann_index.search(query_embedding, k=top_k * 10)  # 获取更多候选

        # 精确排序
        candidates = [self.embeddings[idx] for idx, _ in neighbors[:top_k * 3]]

        # 计算精确相似度
        similarities = []
        for i, candidate in enumerate(candidates):
            sim = VectorOperations.cosine_similarity(query_embedding, candidate)
            similarities.append((i, sim))

        # 排序返回
        similarities.sort(key=lambda x: x[1], reverse=True)
        return similarities[:top_k]

# 示例
# 假设有 1000 个文档嵌入
document_embeddings = np.random.rand(1000, 768)

# 创建相似度系统
similarity_system = DocumentSimilaritySystem(document_embeddings)

# 查询相似文档
query_embedding = np.random.rand(768)
similar_docs = similarity_system.find_similar_documents(query_embedding, top_k=5)
```

```python
class RecommendationSystem:
    def __init__(self, user_embeddings: np.ndarray, item_embeddings: np.ndarray):

        self.item_embeddings = item_embeddings

        self.item_ann = None

    def build_indexes(self):
        """构建用户和物品的索引"""

        self.user_ann = HNSW()
        for user_emb in self.user_embeddings:

        # 物品索引
        self.item_ann = HNSW()
        for item_emb in self.item_embeddings:
            self.item_ann.add_node(item_emb)

    def recommend_items(self, user_idx: int, top_k: int = 10) -> List[tuple]:
        """为用户推荐物品"""
        if self.user_ann is None:
            self.build_indexes()

        user_embedding = self.user_embeddings[user_idx]

        # 搜索相似用户
        similar_users = self.user_ann.search(user_embedding, k=50)

        # 获取这些用户喜欢的物品
        item_scores = {}
        for user_id, _ in similar_users:
            # 这里简化处理，实际应该有用户-交互矩阵

        for item_id, item_embedding in enumerate(self.item_embeddings):

            item_scores[item_id] = sim
```

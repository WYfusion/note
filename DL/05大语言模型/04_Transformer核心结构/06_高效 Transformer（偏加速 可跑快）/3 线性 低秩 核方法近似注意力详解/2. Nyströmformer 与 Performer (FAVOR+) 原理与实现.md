# Part 1：Nyströmformer

## 1.1 Nyström 方法原理

### 经典 Nyström 近似

对于半正定矩阵 $A \in \mathbb{R}^{n \times n}$，选取 $m$ 个 landmark 点，Nyström 近似为：

$$A \approx A_{n \times m} \cdot A_{m \times m}^{-1} \cdot A_{m \times n}$$

其中：

- $A_{n times m}$：原矩阵中 $n$ 行 $m$ 列的子矩阵

- $A_{m times m}$：$m times m$ 的核心子矩阵

- 核心思想：用少量 landmark 重建完整矩阵

### 应用到 Self-Attention

设 softmax attention 矩阵为 $\hat{A} = \text{softmax}(QK^T/\sqrt{d})$。

选取 $m$ 个 landmark query $\tilde{Q}$ 和 key $\tilde{K}$（通过 average pooling）：

$$\tilde{Q} = \text{AvgPool}(Q, m), \quad \tilde{K} = \text{AvgPool}(K, m)$$

$$\hat{A} \approx \underbrace{\text{softmax}(Q\tilde{K}^T)}_{n \times m} \cdot \underbrace{[\text{softmax}(\tilde{Q}\tilde{K}^T)]^{-1}}_{m \times m} \cdot \underbrace{\text{softmax}(\tilde{Q}K^T)}_{m \times n}$$

### 复杂度

$$O(nm \cdot d + m^2 \cdot d + m \cdot n \cdot d) = O(nmd)$$

当 $m \ll n$ 时为线性。

## 1.2 Python 实现

```Python
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class NystromAttention(nn.Module):
    def __init__(self, d_model: int, n_heads: int, num_landmarks: int = 64):
        super().__init__()
        self.n_heads = n_heads
        self.d_head = d_model // n_heads
        self.num_landmarks = num_landmarks
        
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        self.W_o = nn.Linear(d_model, d_model)
    
    def forward(self, x: torch.Tensor) -> torch.Tensor:
        B, n, _ = x.shape
        m = self.num_landmarks
        scale = 1.0 / math.sqrt(self.d_head)
        
        Q = self.W_q(x).view(B, n, self.n_heads, self.d_head).transpose(1, 2)
        K = self.W_k(x).view(B, n, self.n_heads, self.d_head).transpose(1, 2)
        V = self.W_v(x).view(B, n, self.n_heads, self.d_head).transpose(1, 2)
        
        # 通过 segment-means 生成 landmark
        seg_len = n // m
        Q_land = Q.reshape(B, self.n_heads, m, seg_len, self.d_head).mean(dim=3)
        K_land = K.reshape(B, self.n_heads, m, seg_len, self.d_head).mean(dim=3)
        
        # 三个 softmax 矩阵
        # (1) Q @ K_land^T: (B, h, n, m)
        attn_1 = F.softmax(Q @ K_land.transpose(-1, -2) * scale, dim=-1)
        # (2) Q_land @ K_land^T: (B, h, m, m)
        attn_2 = F.softmax(Q_land @ K_land.transpose(-1, -2) * scale, dim=-1)
        # (3) Q_land @ K^T: (B, h, m, n)
        attn_3 = F.softmax(Q_land @ K.transpose(-1, -2) * scale, dim=-1)
        
        # 伪逆（用迭代法避免不稳定的直接逆）
        attn_2_inv = self._iterative_inv(attn_2)
        
        # Nyström 近似: attn_1 @ attn_2_inv @ attn_3 @ V
        out = attn_1 @ (attn_2_inv @ (attn_3 @ V))
        
        out = out.transpose(1, 2).contiguous().view(B, n, -1)
        return self.W_o(out)
    
    def _iterative_inv(self, mat: torch.Tensor, n_iter: int = 6):
        """Newton-Schulz 迭代求伪逆，避免直接逆的数值问题"""
        I = torch.eye(mat.size(-1), device=mat.device)
        K = mat
        V = 1 / (torch.max(torch.sum(K, dim=-2)) * 
                  torch.max(torch.sum(K, dim=-1))) * K.transpose(-1, -2)
        for _ in range(n_iter):
            KV = torch.matmul(K, V)
            V = torch.matmul(0.25 * V, 13 * I - KV @ (15 * I - KV @ (7 * I - KV)))
        return V

# 使用
model = NystromAttention(d_model=512, n_heads=8, num_landmarks=64)
x = torch.randn(2, 2048, 512)
out = model(x)
print(f"Output: {out.shape}")  # (2, 2048, 512)
```

# Part 2：Performer（FAVOR+）

## 2.1 核方法重写 Softmax

### 标准 Softmax Attention

$$\text{Attn}(Q, K, V)_i = \frac{\sum_j \exp(Q_i^T K_j / \sqrt{d}) \cdot V_j}{\sum_j \exp(Q_i^T K_j / \sqrt{d})}$$

### 核方法替换

如果存在映射 $\phi: \mathbb{R}^d \to \mathbb{R}^m$ 使得：

$$\exp(Q_i^T K_j / \sqrt{d}) \approx \phi(Q_i)^T \phi(K_j)$$

则可以利用**矩阵乘法结合律**：

$$\text{Attn} \approx \frac{\phi(Q) \cdot (\phi(K)^T V)}{\phi(Q) \cdot \phi(K)^T \mathbf{1}}$$

关键：先算 $phi(K)^T V in mathbb{R}^{m times d}$，再左乘 $phi(Q)$。总复杂度 $O(nmd)$。

### 2.2 FAVOR+（Fast Attention Via positive Orthogonal Random features）

映射函数：

$$\phi(x) = \frac{\exp(-\|x\|^2 / 2)}{\sqrt{m}} \cdot [\exp(\omega_1^T x), \dots, \exp(\omega_m^T x)]$$

其中 $\omega_1, \dots, \omega_m$ 是**正交随机向量**。

- **正值**：保证近似出来的 "attention weight" 非负

- **正交**：降低方差，比 i.i.d. 随机特征更精确

- **无偏**：$mathbb{E}[phi(Q_i)^T phi(K_j)] = exp(Q_i^T K_j / sqrt{d})$

## 2.3 Python 实现

```Python
import torch
import torch.nn as nn
import math

def favor_plus_features(
    x: torch.Tensor,      # (B, n, d)
    random_features: torch.Tensor,  # (m, d)
    is_query: bool = True
) -> torch.Tensor:
    """FAVOR+ 随机特征映射"""
    # x @ omega^T: (B, n, m)
    projection = x @ random_features.T
    
    # 数值稳定：减去 ||x||^2 / 2
    norm_sq = (x ** 2).sum(dim=-1, keepdim=True) / 2.0
    
    # phi(x) = exp(projection - norm_sq) / sqrt(m)
    features = torch.exp(projection - norm_sq) / math.sqrt(random_features.shape[0])
    
    return features

class PerformerAttention(nn.Module):
    def __init__(self, d_model: int, n_heads: int, n_features: int = 256):
        super().__init__()
        self.n_heads = n_heads
        self.d_head = d_model // n_heads
        self.n_features = n_features
        
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        self.W_o = nn.Linear(d_model, d_model)
        
        # 正交随机特征矩阵
        omega = torch.randn(n_features, self.d_head)
        # 用 QR 分解正交化
        omega, _ = torch.linalg.qr(omega.T)
        self.register_buffer('omega', omega.T)  # (m, d)
    
    def forward(self, x: torch.Tensor) -> torch.Tensor:
        B, n, _ = x.shape
        
        Q = self.W_q(x).view(B, n, self.n_heads, self.d_head)
        K = self.W_k(x).view(B, n, self.n_heads, self.d_head)
        V = self.W_v(x).view(B, n, self.n_heads, self.d_head)
        
        # 对每个 head 做 FAVOR+ 映射
        Q_prime = favor_plus_features(Q, self.omega)  # (B, n, h, m)
        K_prime = favor_plus_features(K, self.omega)
        
        # 利用结合律: (B, h, m, d) = K'^T @ V
        Q_prime = Q_prime.permute(0, 2, 1, 3)  # (B, h, n, m)
        K_prime = K_prime.permute(0, 2, 1, 3)
        V = V.permute(0, 2, 1, 3)              # (B, h, n, d)
        
        KV = K_prime.transpose(-1, -2) @ V     # (B, h, m, d)
        Z = K_prime.transpose(-1, -2).sum(dim=-1, keepdim=True)  # (B, h, m, 1)
        
        # Q' @ (K'^T V) / (Q' @ K'^T 1)
        numerator = Q_prime @ KV               # (B, h, n, d)
        denominator = Q_prime @ Z              # (B, h, n, 1)
        
        out = numerator / (denominator + 1e-6)
        out = out.permute(0, 2, 1, 3).contiguous().view(B, n, -1)
        return self.W_o(out)

# 使用
model = PerformerAttention(d_model=512, n_heads=8, n_features=256)
x = torch.randn(2, 4096, 512)
out = model(x)
print(f"Output: {out.shape}")  # (2, 4096, 512)
```

## 3. 对比总结

|维度|Nyströmformer|Performer|
|---|---|---|
|**近似方法**|Nyström 矩阵近似|核随机特征映射|
|**超参数**|landmark 数 $m$|随机特征数 $m$|
|**LRA 表现**|**58.95%**（最佳）|53.63%|
|**数值稳定性**|需要迭代伪逆|指数映射可能溢出|
|**Causal 支持**|需要额外处理|支持前缀和递推|
|**成熟度**|中等|中等|

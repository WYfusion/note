## 概述

Linformer 的核心发现：self-attention 矩阵近似低秩。通过对 K, V 做线性投影将序列维度从 $n$ 压到 $k$，实现 $O(nk)$ 的线性注意力。

---

## 1. 动机：Attention 矩阵的低秩性

### 实验发现

Linformer 论文中的关键实验：

- 对 BERT 各层的 attention 矩阵做 SVD

- **大部分信息集中在少数特征值**：前 128 个奇异值即可捕获 >90% 信息

- 层越深，低秩性越明显

- 累积特征值比例曲线急剧上升 → **低秩假设成立**

---

## 2. 数学推导

### 2.1 标准 Attention

$$\text{Attn}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d}}\right) V$$

其中 $Q, K, V \in \mathbb{R}^{n \times d}$，$QK^T \in \mathbb{R}^{n \times n}$。

### 2.2 Linformer 投影

引入投影矩阵 $E_K, E_V \in \mathbb{R}^{k \times n}$（$k \ll n$）：

$$\tilde{K} = E_K K \in \mathbb{R}^{k \times d}$$

$$\tilde{V} = E_V V \in \mathbb{R}^{k \times d}$$

$$\text{LinAttn}(Q, K, V) = \text{softmax}\left(\frac{Q \tilde{K}^T}{\sqrt{d}}\right) \tilde{V}$$

### 2.3 复杂度分析

- $Q \tilde{K}^T$：$(n \times d) \cdot (d \times k) = O(nkd)$

- softmax：$O(nk)$

- $\times \tilde{V}$：$(n \times k) \cdot (k \times d) = O(nkd)$

- **总计**：$O(nkd)$，当 $k$ 为常数时线性于 $n$

### 2.4 投影矩阵的选择

论文中测试了几种方案：

- **随机初始化（不训练）**：简单有效

- **可训练投影**：略好但增加参数

- **层间共享**：进一步节省参数

- **头间共享**：跨 head 共享 $E_K, E_V$

## 3. Python 实现

```Python
import torch
import torch.nn as nn
import math

class LinformerAttention(nn.Module):
    def __init__(
        self, 
        d_model: int, 
        n_heads: int, 
        max_seq_len: int, 
        proj_k: int = 128,
        share_kv_proj: bool = True
    ):
        super().__init__()
        self.n_heads = n_heads
        self.d_head = d_model // n_heads
        self.proj_k = proj_k
        
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        self.W_o = nn.Linear(d_model, d_model)
        
        # 低秩投影矩阵：将序列长度 n -> k
        self.E_k = nn.Parameter(
            torch.randn(n_heads, proj_k, max_seq_len) / math.sqrt(max_seq_len)
        )
        if share_kv_proj:
            self.E_v = self.E_k
        else:
            self.E_v = nn.Parameter(
                torch.randn(n_heads, proj_k, max_seq_len) / math.sqrt(max_seq_len)
            )
    
    def forward(self, x: torch.Tensor) -> torch.Tensor:
        B, n, _ = x.shape
        
        Q = self.W_q(x).view(B, n, self.n_heads, self.d_head).transpose(1, 2)
        K = self.W_k(x).view(B, n, self.n_heads, self.d_head).transpose(1, 2)
        V = self.W_v(x).view(B, n, self.n_heads, self.d_head).transpose(1, 2)
        # Q, K, V: (B, heads, n, d_head)
        
        # 投影 K, V: (B, heads, n, d) -> (B, heads, k, d)
        E_k = self.E_k[:, :, :n]  # 截断到实际序列长度
        E_v = self.E_v[:, :, :n]
        
        K_proj = torch.einsum('hkn,bhnd->bhkd', E_k, K)  # (B, heads, k, d)
        V_proj = torch.einsum('hkn,bhnd->bhkd', E_v, V)
        
        # 计算 attention: Q @ K_proj^T -> (B, heads, n, k)
        scale = 1.0 / math.sqrt(self.d_head)
        attn = torch.softmax(
            torch.matmul(Q, K_proj.transpose(-1, -2)) * scale, 
            dim=-1
        )
        
        # 输出: (B, heads, n, k) @ (B, heads, k, d) -> (B, heads, n, d)
        out = torch.matmul(attn, V_proj)
        
        out = out.transpose(1, 2).contiguous().view(B, n, -1)
        return self.W_o(out)

# 使用
model = LinformerAttention(
    d_model=512, n_heads=8, 
    max_seq_len=4096, proj_k=128
)
x = torch.randn(2, 4096, 512)
out = model(x)
print(f"Output: {out.shape}")  # (2, 4096, 512)
print(f"投影维度 k=128, 序列长 n=4096, 压缩比={4096/128:.1f}x")
```

## 4. 局限性分析

### 4.1 固定投影长度

- $E \in \mathbb{R}^{k \times n}$ 依赖固定的 max_seq_len

- 变长序列需要截断或 padding

- 不适合动态长度的 decoder

### 4.2 近似误差

- softmax 使得 attention 矩阵不完全低秩

- 任务敏感：在某些任务上近似误差可控，另一些上不可

### 4.3 Causal 场景受限

- 投影矩阵 $E$ 混合了未来信息

- 对自回归生成不直接适用

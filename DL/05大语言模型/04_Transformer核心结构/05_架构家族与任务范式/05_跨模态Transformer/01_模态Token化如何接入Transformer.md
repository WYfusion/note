# 模态 Token 化如何接入 Transformer

## 问题定义

图像、音频、视频不是天然离散 token，如何把它们接入 Transformer？

## 直觉解释

核心思路都是先把连续信号切成离散片段，再映射成向量。

## 形式化推导

图像常切 patch，音频常切帧或 codec token，视频常切时空 patch；再经线性投影或卷积前端转成隐藏向量。

## 工程意义

Token 化方式直接决定序列长度、局部结构保留程度和后续 attention 代价。

## 常见误解

> [!warning] 常见误解
> - “只要能变成 token 就够了。” 不对，怎么切、切多粗都会影响效果和成本。

## 例子或反例

视频若切得过细，序列长度会爆炸；切得过粗，又会丢掉局部运动细节。

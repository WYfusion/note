# 函数调用与结构化输出

本章节详细介绍 Agent 如何通过 Function Calling 与外部世界交互，特别是如何定义和使用语音处理工具。

## 目录

### [01_FunctionCalling_JSONSchema.md](./01_FunctionCalling_JSONSchema.md)
- **原理**：LLM -> JSON -> Runtime -> Result。
- **JSON Schema**：定义工具的 Name, Description, Parameters。
- **语音工具示例**：
  - `transcribe_audio`: ASR 参数设计。
  - `generate_speech`: TTS 情感与语速控制。
- **结构化输出**：直接生成 JSON 控制音乐合成器或剪辑列表 (EDL)。

# 多模态与跨模态

本章节探讨大语言模型如何扩展感知边界，实现对图像、音频、视频的统一理解与生成。

## 目录

### [01_多模态对齐_融合_解码.md](./01_多模态对齐_融合_解码.md)
- **对齐 (Alignment)**：CLAP (Audio-Text), ImageBind (Joint Embedding)。
- **融合 (Fusion)**：
  - **Early Fusion**: Audio Tokenization (VQ-VAE)。
  - **Adapter**: Projection Layer (Qwen-Audio)。
- **解码 (Decoding)**：
  - **Cascaded**: LLM -> Text -> TTS。
  - **Native**: LLM -> Audio Tokens -> Codec Decoder。
- **Any-to-Any**: Speech-to-Speech, Image-to-Audio。

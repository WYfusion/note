# 向量嵌入技术核心

无论哪种模态的嵌入，底层都围绕几个关键技术问题：怎么表示、怎么训练、怎么检索、怎么评测。

<aside>
💡

**通俗理解**：嵌入模型就像一台"语义翻译机"——输入任意对象，输出一串数字。训练过程就是教这台机器"相似的东西要翻译成相近的数字"。

</aside>

---

## 四大核心主题

### 表示层与距离度量

把对象映射到高维空间，用 cosine / dot-product / L2 衡量相似度。

### 训练范式

对比学习、Bi-Encoder、Cross-Encoder、Late Interaction、蒸馏与指令微调。

### 检索视角的五类表示

Dense 单向量 → Sparse 稀疏 → Hybrid 混合 → Multi-vector → Contextual chunk embeddings。

### 评测体系

离线指标（Recall@k、MRR、nDCG、MTEB）与在线指标（点击率、延迟、成本）。

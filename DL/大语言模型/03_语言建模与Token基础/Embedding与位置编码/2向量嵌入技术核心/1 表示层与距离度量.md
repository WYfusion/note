向量嵌入的核心目标：把对象映射到高维空间，使得**语义相近的对象在向量空间中距离也近**。

---

## 核心思想

将任意对象（文本、图片、音频……）编码为一个固定长度的浮点数数组（向量），通常维度为 128~4096。这个向量就是对象的"语义指纹"。

---

## 三种主流距离度量

### Cosine Similarity（余弦相似度）

- **定义**：两个向量夹角的余弦值
- **范围**：[-1, 1]，1 表示完全相同方向，0 表示正交，-1 表示完全相反
- **特点**：只关注方向，不关注长度；**最常用**
- **适用**：归一化后的嵌入向量（绝大多数场景）

$\text{cosine}(a, b) = \frac{a \cdot b}{\|a\| \cdot \|b\|}$

### Dot Product（点积）

- **定义**：两个向量对应元素相乘后求和
- **特点**：同时考虑方向和长度；向量归一化后等价于 cosine
- **适用**：当向量长度有意义时（如表示"重要性"）

$\text{dot}(a, b) = \sum_{i} a_i \cdot b_i$

### L2 Distance（欧氏距离）

- **定义**：两点之间的直线距离
- **特点**：同时考虑方向和长度；值越小越相似
- **适用**：聚类、某些特定索引类型

$L2(a, b) = \sqrt{\sum_{i} (a_i - b_i)^2}$

---

## 三者关系

<aside>
🔑

当向量**归一化**（长度为 1）后，cosine similarity、dot product 和 L2 distance **等价排序**。这就是为什么大多数嵌入模型输出归一化向量——此时选哪种距离都一样。

</aside>

## Python 实战：距离度量对比

```python
import numpy as np
from numpy.linalg import norm

# 两个示例向量
a = np.array([0.3, 0.5, 0.8, 0.1])
b = np.array([0.2, 0.6, 0.7, 0.3])

# --- Cosine Similarity ---
cosine_sim = np.dot(a, b) / (norm(a) * norm(b))
print(f"Cosine Similarity: {cosine_sim:.4f}")

# --- Dot Product ---
dot_prod = np.dot(a, b)
print(f"Dot Product: {dot_prod:.4f}")

# --- L2 Distance ---
l2_dist = norm(a - b)
print(f"L2 Distance: {l2_dist:.4f}")

# --- 归一化后对比 ---
a_norm = a / norm(a)
b_norm = b / norm(b)

print("\n--- 归一化后 ---")
print(f"Cosine Similarity: {np.dot(a_norm, b_norm):.4f}")
print(f"Dot Product: {np.dot(a_norm, b_norm):.4f}")
print(f"L2 Distance: {norm(a_norm - b_norm):.4f}")
# 归一化后: cosine = dot product, L2² = 2(1 - cosine)
```

## 高维空间的特殊性质

```python
import numpy as np

# 高维空间中随机向量几乎正交（"维度诅咒"）
np.random.seed(42)
dim = 768  # 典型嵌入维度
n_vectors = 1000
vectors = np.random.randn(n_vectors, dim)

# 归一化
vectors = vectors / np.linalg.norm(vectors, axis=1, keepdims=True)

# 计算所有向量对的 cosine similarity
from itertools import combinations
sims = []
for i, j in list(combinations(range(100), 2)):  # 取前100个
    sim = np.dot(vectors[i], vectors[j])
    sims.append(sim)

print(f"随机向量对的 cosine similarity 统计:")
print(f"  均值: {np.mean(sims):.4f} (接近 0)")
print(f"  标准差: {np.std(sims):.4f} (非常小)")
print(f"  最大值: {np.max(sims):.4f}")
print(f"  最小值: {np.min(sims):.4f}")
print(f"\n→ 高维空间中，有意义的相似度通常 > 0.3")
```

## 选择建议

| 场景 | 推荐距离 | 原因 |
| --- | --- | --- |
| 大多数嵌入检索 | Cosine | 归一化向量标配 |
| 向量长度有意义 | Dot Product | 保留幅度信息 |
| 聚类/KNN | L2 | 与几何距离直觉一致 |
| 不确定 | Cosine | 最安全的默认选择 |

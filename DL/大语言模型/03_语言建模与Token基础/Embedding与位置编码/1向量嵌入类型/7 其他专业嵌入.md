除通用模态外，许多专业领域有自己的嵌入方法，针对特定数据结构和业务需求优化。

💡

**通俗理解**：通用嵌入是"万能钥匙"，专业嵌入是"定制钥匙"——在特定领域效果更好。

</aside>

## 五大领域

### 表格/结构化嵌入

- **对象**：表格列语义、schema linking、cell/row/table
- **用途**：表格问答（Table QA）、schema 匹配、结构化数据检索
- **代表**：TaPas、TURL、表格预训练模型

### 图/网络嵌入

- **对象**：图中的节点、边、子图
- **代表**：Node2Vec、GraphSAGE、GNN embeddings
- **用途**：社交网络分析、知识图谱补全、推荐系统、欺诈检测

### 推荐系统嵌入

- **对象**：用户（user）和物品（item）
- **代表**：Two-Tower Retrieval、Matrix Factorization
- **用途**：个性化推荐、候选召回、用户画像

### 地理/时序/传感器嵌入

- **对象**：GPS 轨迹、IoT 传感器数据、金融时序
- **用途**：轨迹相似匹配、异常检测、预测

### 3D/点云/机器人嵌入

- **对象**：3D 点云、场景、机器人感知
- **代表**：Point-Bind、3D multimodal embedding
- **用途**：3D 物体检索、场景理解、机器人导航

## Python 实战：Node2Vec 图嵌入

```python
import networkx as nx
from node2vec import Node2Vec
import numpy as np

# 创建示例图
G = nx.karate_club_graph()  # 空手道俱乐部经典图

# 训练 Node2Vec
node2vec = Node2Vec(
    G,
    dimensions=64,     # 嵌入维度
    walk_length=30,     # 随机游走长度
    num_walks=200,      # 每个节点的游走次数
    p=1,                # 返回参数（控制 BFS vs DFS）
    q=1,                # 进出参数
    workers=4,
)

# 训练 Word2Vec 模型
model = node2vec.fit(window=10, min_count=1, batch_words=4)

# 获取节点嵌入
node_0_embed = model.wv["0"]  # 节点 0 的嵌入向量
print(f"节点 0 嵌入维度: {node_0_embed.shape}")

# 找最相似节点
similar_nodes = model.wv.most_similar("0", topn=5)
print("与节点 0 最相似的节点:")
for node, score in similar_nodes:
    print(f"  节点 {node}: {score:.4f}")
```

## Two-Tower 推荐系统嵌入

```python
import torch
import torch.nn as nn

class TwoTowerModel(nn.Module):
    """双塔推荐模型：user tower + item tower"""
    def __init__(self, n_users, n_items, embed_dim=64):
        super().__init__()
        self.user_tower = nn.Sequential(
            nn.Embedding(n_users, 128),
            nn.ReLU(),
            nn.Linear(128, embed_dim),
        )
        self.item_tower = nn.Sequential(
            nn.Embedding(n_items, 128),
            nn.ReLU(),
            nn.Linear(128, embed_dim),
        )
    
    def forward(self, user_ids, item_ids):
        user_embeds = self.user_tower(user_ids)  # [batch, embed_dim]
        item_embeds = self.item_tower(item_ids)  # [batch, embed_dim]
        # 归一化
        user_embeds = nn.functional.normalize(user_embeds, dim=-1)
        item_embeds = nn.functional.normalize(item_embeds, dim=-1)
        # 点积相似度
        scores = (user_embeds * item_embeds).sum(dim=-1)
        return scores

# 使用示例
model = TwoTowerModel(n_users=10000, n_items=50000, embed_dim=64)
user_ids = torch.tensor([0, 1, 2])
item_ids = torch.tensor([100, 200, 300])
scores = model(user_ids, item_ids)
print(f"推荐分数: {scores}")
```

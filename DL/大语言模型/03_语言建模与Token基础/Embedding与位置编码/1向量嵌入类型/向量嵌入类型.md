# 向量嵌入类型

向量嵌入按**输入模态**分为多种类型。核心思想都一样：把某种对象映射到一组数字（向量），让语义相近的对象在向量空间中距离更近。

<aside>
💡

**通俗理解**：就像给每个对象发一张"语义坐标"，坐标越近，含义越像。不同模态（文字、图片、声音……）各有专属的"坐标生成器"。

</aside>

---

## 各模态嵌入一览

| 模态 | 典型模型 | 核心用途 |
| --- | --- | --- |
| 文本 | SBERT、E5、BGE、OpenAI | 检索、RAG、聚类、分类 |
| 代码 | CodeBERT、Jina Code | NL↔Code 检索、代码相似 |
| 图片 | ViT、CLIP、SigLIP | 图像检索、图文匹配 |
| 音频 | wav2vec、CLAP | 语音检索、说话人识别 |
| 视频 | VideoCLIP | 视频片段检索 |
| 多模态 | CLIP、ImageBind、Jina v4 | 跨模态统一检索 |
| 专业领域 | Node2Vec、GraphSAGE | 图网络、推荐、时序 |

已有搜索引擎（Elasticsearch、OpenSearch、Vespa）可以扩展向量检索能力，实现传统搜索与语义搜索的一体化。

---

## 三大产品

| 产品 | 特点 | 适合 |
| --- | --- | --- |
| **Elasticsearch** | dense vector + semantic/hybrid search | 已有 ES 技术栈 |
| **OpenSearch** | k-NN/vector search，AWS 生态 | 搜索系统一体化 |
| **Vespa** | 强排名、多阶段检索、tensor、ColBERT 支持强 | 高级排序需求 |

## Elasticsearch 向量搜索

```python
from elasticsearch import Elasticsearch
import numpy as np

es = Elasticsearch("http://localhost:9200")

# 创建索引（含 dense_vector 字段）
es.indices.create(
    index="my_docs",
    body={
        "mappings": {
            "properties": {
                "title": {"type": "text"},
                "content": {"type": "text"},
                "embedding": {
                    "type": "dense_vector",
                    "dims": 768,
                    "index": True,
                    "similarity": "cosine"
                },
                "category": {"type": "keyword"}
            }
        }
    },
    ignore=400
)

# 插入文档
for i in range(100):
    es.index(
        index="my_docs",
        id=i,
        body={
            "title": f"Document {i}",
            "content": f"This is the content of document {i}",
            "embedding": np.random.randn(768).tolist(),
            "category": "tech"
        }
    )

# 纯向量搜索 (kNN)
results = es.search(
    index="my_docs",
    body={
        "knn": {
            "field": "embedding",
            "query_vector": np.random.randn(768).tolist(),
            "k": 10,
            "num_candidates": 100
        }
    }
)

# Hybrid 搜索（向量 + BM25 + filter）
results = es.search(
    index="my_docs",
    body={
        "query": {
            "bool": {
                "must": [
                    {"match": {"content": "machine learning"}}
                ],
                "filter": [
                    {"term": {"category": "tech"}}
                ]
            }
        },
        "knn": {
            "field": "embedding",
            "query_vector": np.random.randn(768).tolist(),
            "k": 10,
            "num_candidates": 100,
            "boost": 0.5  # 调节向量和 BM25 的权重
        }
    }
)

for hit in results["hits"]["hits"][:5]:
    print(f"Score: {hit['_score']:.4f} | {hit['_source']['title']}")
```

## Vespa 多阶段检索

```python
# Vespa 配置示例（schema 定义）
vespa_schema = """
schema my_doc {
    document my_doc {
        field title type string {
            indexing: summary | index
        }
        field content type string {
            indexing: summary | index
        }
        field embedding type tensor<float>(d[768]) {
            indexing: attribute | index
            attribute {
                distance-metric: angular
            }
            index {
                hnsw {
                    max-links-per-node: 16
                    neighbors-to-explore-at-insert: 200
                }
            }
        }
    }
    
    # 多阶段排序
    rank-profile hybrid {
        inputs {
            query(q_embedding) tensor<float>(d[768])
        }
        first-phase {
            expression: closeness(field, embedding) + bm25(content)
        }
        # 可选：second-phase 用更复杂的模型精排
    }
}
"""

# Vespa Python 客户端
from vespa.application import Vespa

app = Vespa(url="http://localhost:8080")

# Hybrid 查询
result = app.query(
    body={
        "yql": "select * from my_doc where userQuery() or ({targetHits: 100}nearestNeighbor(embedding, q_embedding))",
        "query": "machine learning",
        "input.query(q_embedding)": np.random.randn(768).tolist(),
        "ranking.profile": "hybrid",
        "hits": 10
    }
)
```

## 与专用向量数据库对比

| 维度 | 搜索引擎扩展 | 专用向量数据库 |
| --- | --- | --- |
| 向量检索性能 | 好 | 更好 |
| BM25/全文搜索 | 原生支持 | 通常需外部 |
| Hybrid 搜索 | 原生一体化 | 需组合方案 |
| 运维复杂度 | 已有栈则低 | 新增组件 |
| 适合 | 已有 ES/Vespa 生态 | 纯向量检索场景 |

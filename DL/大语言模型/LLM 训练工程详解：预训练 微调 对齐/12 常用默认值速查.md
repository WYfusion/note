# 12. 常用默认值速查

训练配置千变万化，但以下默认值覆盖了 80% 的场景，可作为起点快速启动。

---

## 速查表

| 配置项 | 默认值 | 备注 |
| --- | --- | --- |
| **精度** | `bf16` | 优先 bf16；硬件不支持时退回 fp16 + GradScaler |
| **优化器** | `AdamW` | β₁=0.9, β₂=0.95（pretrain）或 0.999（finetune）, wd=0.1 |
| **梯度裁剪** | `global grad clip = 1.0` | 所有阶段通用 |
| **Warmup** | `~1% total steps` | 新训练取 1%~3%；resume 可更短 |
| **Scheduler** | `cosine` 或 `linear decay` | cosine 更常见；min_lr = peak_lr × 0.1 |
| **Checkpoint** | 早期密、后期疏 | last-3 + best-2 + milestone |
| **验证** | 固定小验证集高频 + 完整验证低频 | 小集每 100~500 step；全集每 1%~5% steps |
| **监控** | W&B / TensorBoard + Prometheus/Grafana + profiler | 三层：训练指标 + 系统指标 + profiling |

---

## 典型 Pretrain 配置示例（7B）

```python
# AdamW
lr = 3e-4
betas = (0.9, 0.95)
weight_decay = 0.1
eps = 1e-8

# Scheduler
warmup_steps = 2000          # ~1% of 200K total steps
scheduler = "cosine"
min_lr = 3e-5                # peak_lr * 0.1

# Grad clipping
max_grad_norm = 1.0

# Precision
dtype = "bf16"

# Batch
global_batch_size = 4M tokens  # ~2048 seqs × 2048 seq_len
gradient_accumulation_steps = 8

# Checkpoint
save_every_steps = 500
keep_last = 3
keep_best = 2
```

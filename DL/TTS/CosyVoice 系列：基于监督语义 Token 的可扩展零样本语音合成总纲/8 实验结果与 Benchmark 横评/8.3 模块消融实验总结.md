## 消融实验总览

CosyVoice 系列在论文中提供了详细的消融实验，量化每个模块的贡献：

### v2 关键消融

| **消融项**                      | **test-zh CER** | **SIM** | **结论**               |
| ---------------------------- | --------------- | ------- | -------------------- |
| Full v2 model                | 1.45%           | 0.748   | —                    |
| • LLM init (随机初始化)           | 1.89%           | 0.735   | LLM 预训练权重贡献 CER ↓23% |
| • Drop Spk Emb → 保留 x-vector | 1.52%           | 0.741   | 移除 spk emb 提升韵律自然度   |
| • FSQ → 换回 VQ                | 1.78%           | 0.740   | FSQ 贡献 CER ↓19%      |
| • DPO 后训练                    | 1.62%           | 0.745   | DPO 贡献 CER ↓10%      |

### v3 关键消融

|**消融项**|**CER 变化**|**结论**|
|---|---|---|

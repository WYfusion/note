### 各阶段目标

| **阶段**       | **数据**       | **目标**                  |
| ------------ | ------------ | ----------------------- |
| ① 大规模预训练     | 1M h 多语言无标注  | 学习通用的文本→token 映射能力      |
| ② DiffRO 后训练 | 高质量子集 + 奖励模型 | 突破数据质量上限，优化 CER/SIM/MOS |
| ③ 持续预训练      | 特定领域/语言数据    | 适配新语言、新领域               |
| ④ mSFT       | 多说话人精标数据     | 说话人级别能力定制               |

## 0.5B → 1.5B 规模化

### 规模化收益

- **内容一致性** ↑：更大模型更好理解文本语义

- **韵律自然度** ↑：更强的上下文建模能力

- **多语言泛化** ↑：跨语言 transfer 更有效

- **指令遵循** ↑：更好理解自然语言指令

### Scaling Law 观察

$$\text{Quality} \propto \log(\text{Model Size}) + \log(\text{Data Size})$$

v3 同时扩展了模型参数（0.5B → 1.5B）和训练数据（~200K h → 1M h），双重 scaling 带来了显著的性能提升。

> [!important]
> 
> **工程判断**：1.5B 是训练阶段的完整模型。开源的 Fun-CosyVoice3-0.5B 是经过蒸馏/压缩后的推理版本，在保持质量的同时降低了部署成本。

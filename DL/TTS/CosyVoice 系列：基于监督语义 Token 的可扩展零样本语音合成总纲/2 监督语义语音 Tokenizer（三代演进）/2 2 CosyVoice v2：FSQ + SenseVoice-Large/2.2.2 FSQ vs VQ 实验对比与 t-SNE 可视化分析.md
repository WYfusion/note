## t-SNE 可视化分析

### VQ token 分布

VQ token 在 t-SNE 空间中呈现**不均匀聚类**：

- 大部分数据集中在少数几个簇

- 大量码本对应的区域几乎没有数据点

- 说话人信息部分残留（同说话人 token 聚在一起）

### FSQ token 分布

FSQ token 在 t-SNE 空间中分布更**均匀**：

- 数据点覆盖整个码本空间

- 不同说话人的 token 混合良好（说话人无关性更强）

- 语义相近的内容在空间上相邻

>

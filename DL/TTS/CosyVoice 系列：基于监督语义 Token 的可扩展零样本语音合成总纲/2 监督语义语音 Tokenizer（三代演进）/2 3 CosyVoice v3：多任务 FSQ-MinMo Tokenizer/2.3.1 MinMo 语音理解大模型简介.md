## MinMo 概述

MinMo（Mini-Omni / Multi-modal）是阿里通义实验室开发的**大规模多模态语音理解模型**，作为 CosyVoice v3 Tokenizer 的基座编码器。

### 核心特性

|**维度**|**说明**|
|---|---|
|**架构**|Audio Encoder + LLM backbone|
|**训练数据**|1.4M 小时多语言语音|
|**支持任务**|ASR · LID · SER · AED · SA · 语音翻译 · 口语理解|
|**语言覆盖**|多语言 (中英日韩法德西意等)|

## 为什么用 MinMo 替代 SenseVoice？

|**维度**|**SenseVoice (v1/v2)**|**MinMo (v3)**|
|---|---|---|
|**训练数据**|~数十万小时|**1.4M 小时**|
|**任务覆盖**|ASR 为主|**7+ 任务**|
|**副语言能力**|有限|**强（情感/事件/语种）**|
|**表征质量**|优秀|**SOTA**|

### 对 Tokenizer 的收益

MinMo 的多任务表征使 FSQ 量化后的 token 不仅编码语义内容，还编码副语言信息（情感、声学事件等），为下游 TTS 的可控生成提供更丰富的条件信号。

## 0. 定位

> 一句话：`swift eval` 是 ms-swift 在 EvalScope 上层的统一评测 CLI，支持多 benchmark、多后端、自定义任务。

---

## 1. 基本用法

```bash
swift eval \
    --model out/qwen-sft \
    --eval_dataset mmlu ceval gsm8k \
    --infer_backend vllm \
    --eval_batch_size 8 \
    --output_dir eval/qwen-sft \
    --eval_limit 1000
```

## 2. 关键参数

|参数|说明|
|---|---|
|`--eval_limit`|每个 benchmark 采样数|
|`--infer_backend`|vllm / lmdeploy / pt|
|`--max_new_tokens`|生成上限|
|`--custom_eval_config`|自定义任务配置|

## 3. 输出结构

```jsx
eval/qwen-sft/
  summary.json         # 总分汇总
  mmlu/
    raw.jsonl          # 每条样本预测
    metrics.json       # 指标
  ceval/
    raw.jsonl
    metrics.json
  gsm8k/
    raw.jsonl
    metrics.json
```

```json
// summary.json
{
  "model": "out/qwen-sft",
  "results": {
    "mmlu":  {"score": 0.6745, "n_samples": 1000},
    "ceval": {"score": 0.7012, "n_samples": 1000},
    "gsm8k": {"score": 0.5821, "n_samples": 1000}
  }
}
```

## 4. 多模型对比

```bash
# 写一个简单脚本批量评测
for M in base sft dpo grpo; do
    swift eval \
        --model out/qwen-${M} \
        --eval_dataset mmlu ceval gsm8k \
        --output_dir eval/qwen-${M}
done

# 汇总
python - <<'PY'
import json, glob, pandas as pd
rows = []
for f in glob.glob('eval/*/summary.json'):
    s = json.load(open(f))
    name = f.split('/')[1]
    row = {'model': name}
    for k, v in s['results'].items():
        row[k] = v['score']
    rows.append(row)
print(pd.DataFrame(rows).to_string(index=False))
PY
```

## 5. 与 vLLM 的整合

```bash
# 复用已起的 vLLM 服务做评测（不重启加载）
swift deploy --model out/qwen-sft --infer_backend vllm --port 8000 &

swift eval \
    --eval_url http://localhost:8000/v1 \
    --model_id_or_path my-qwen \
    --eval_dataset mmlu ceval
```

## 6. 多模态评测

```bash
swift eval \
    --model out/qwen-vl-sft \
    --eval_dataset mmbench mmmu \
    --infer_backend vllm \
    --eval_batch_size 4
```

## 7. 踩坑提示

> [!important]
> 
> - **基座模型评测要用 base benchmark**：base 模型不应在指令对话集上评，应该用 MMLU/PPL 这类基础题。
> 
> - **vLLM batch 不一致**：vLLM 内部已批，`eval_batch_size` 调过大反而慢。
> 
> - **小样本评测方差大**：`eval_limit < 200` 时分数波动大，比较时务必同 limit。
> 
> - **few-shot 数固定**：MMLU 标准 5-shot，不要随便改。

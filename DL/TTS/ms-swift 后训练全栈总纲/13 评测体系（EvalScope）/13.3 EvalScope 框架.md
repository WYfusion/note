## 0. 定位

> 一句话：EvalScope 是 ms-swift 背后的评测引擎，独立可用，支持插件化任务、批量模型对比、可视化报告。

## 2. 独立使用

```bash
pip install evalscope

# 命令行
evalscope eval \
    --model Qwen/Qwen2.5-7B-Instruct \
    --datasets mmlu ceval gsm8k \
    --eval-backend Native \
    --limit 1000
```

## 3. Python API

```python
from evalscope import TaskConfig, run_task

cfg = TaskConfig(
    model='Qwen/Qwen2.5-7B-Instruct',
    datasets=['mmlu', 'ceval', 'gsm8k'],
    eval_backend='Native',
    limit=1000,
    eval_batch_size=8,
    generation_config={'temperature': 0.0, 'max_new_tokens': 512},
)
results = run_task(cfg)
for name, r in results.items():
    print(f'{name}: {r["score"]:.4f}')
```

## 4. 自定义 benchmark

```python
# my_bench.py
from evalscope.benchmarks import DataAdapter
from evalscope.metrics import exact_match
from evalscope.utils import register_benchmark

@register_benchmark('my-domain')
class MyDomainAdapter(DataAdapter):
    def load(self, **kw):
        # 返回 [{'question':..., 'answer':...}, ...]
        import json
        return [json.loads(l) for l in open('my_test.jsonl')]

    def gen_prompt(self, item, **kw):
        return f"问题：{item['question']}\n答案："

    def parse_pred(self, pred: str, item: dict):
        return pred.strip().split('\n')[0]

    def match(self, pred, gold):
        return exact_match(pred, gold)
```

```bash
evalscope eval \
    --model out/qwen-sft \
    --datasets my-domain \
    --custom-config my_bench.py \
    --limit 500
```

## 5. Arena 模式（pairwise）

```bash
evalscope arena \
    --models out/qwen-sft out/qwen-dpo \
    --datasets alpaca-eval \
    --judge gpt-4o
```

输出：A vs B win-rate、置信区间、显著性。

## 6. 性能压测一并做

```bash
evalscope perf \
    --url http://localhost:8000/v1 \
    --model my-qwen \
    --num-prompts 1000 \
    --request-rate 10
```

输出：TTFT、TPOT、Throughput、P50/P95/P99 延迟、失败率。

## 7. 与 ms-swift 的关系

|层|角色|ms-swift `swift eval`|统一封装，封装训练后产物路径|
|---|---|---|---|
|EvalScope|底层引擎，benchmark / metric / backend|独立使用|不依赖 ms-swift 也可|

> [!important]
> 
> - **多版本警告**：EvalScope 与 ms-swift 版本不匹配会引起 benchmark 不存在的报错。
> 
> - **benchmark 污染检查**：自定义 benchmark 务必与训练数据做 hash 去重。

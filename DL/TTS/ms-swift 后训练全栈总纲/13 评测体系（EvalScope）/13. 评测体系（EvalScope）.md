## 2. 评测命令

```bash
# 评测本地模型在多个标准 benchmark
swift eval --model output/sft/checkpoint-xxx \
    --eval_backend Native \
    --eval_dataset gsm8k mmlu humaneval \
    --eval_limit 500 \
    --infer_backend vllm

# 评测业务集
swift eval --model output/sft/checkpoint-xxx \
    --eval_backend Native \
    --custom_eval_config eval_config.json
```

其中 `eval_config.json` 示例：

```json
{
  "datasets": [
    {
      "name": "my_business_eval",
      "path": "./eval/business.jsonl",
      "metrics": ["acc", "f1", "rouge"]
    }
  ],
  "infer_cfg": {
    "max_new_tokens": 512,
    "temperature": 0
  }
}
```

## 5. 评测集冻结与污染检查

```python
# 伪代码：N-gram 污染检测
from collections import Counter

def ngrams(text, n=13):
    tokens = text.split()
    return set(' '.join(tokens[i:i+n]) for i in range(len(tokens)-n+1))

def contamination_rate(train_texts, eval_texts, n=13):
    train_grams = set()
    for t in train_texts:
        train_grams.update(ngrams(t, n))
    contaminated = 0
    for e in eval_texts:
        e_grams = ngrams(e, n)
        if e_grams & train_grams:
            contaminated += 1
    return contaminated / len(eval_texts)

rate = contamination_rate(train_set, eval_set, n=13)
print(f'Contamination rate: {rate:.2%}')  # 期望 < 0.5%
```

> [!important]
> 
> **生产上线前**：训练集 vs 评测集的 13-gram 重合率**必须 < 0.5%**。超过需重新拆分或手动清除。

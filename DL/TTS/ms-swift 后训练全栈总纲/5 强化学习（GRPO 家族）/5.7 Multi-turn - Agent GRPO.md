## 0. 定位

> 一句话：Multi-turn GRPO = rollout 不再是单轮生成，而是**多轮交互**（调工具 / 接环境反馈），适用于 Agent 训练。

## 2. 适用场景

| 场景                | 是否适用 multi-turn GRPO | 数学推理（纯推理）      | ❌ 单轮 GRPO 即可 |
| ----------------- | -------------------- | -------------- | ------------ |
| 调用计算器 / Python 环境 | ✅ multi-turn         | 调用搜索 / RAG     | ✅ multi-turn |
| 多步决策（留下一步 ↔ 考虑环境） | ✅ multi-turn         | WebShop 、OS 代理 | ✅ multi-turn |

## 3. 数据与采样过程

```jsx
# 训练数据（system + 工具定义）
{
  "messages": [
    {"role":"system","content":"You can call tools: calculator(expr)."},
    {"role":"user",  "content":"求 (12*45-37)/8"}
  ],
  "solution": "66.625"
}
```

rollout 过程：

```jsx
[user]      求 (12*45-37)/8
[assistant] <think>...</think>
            <tool_call>{"name":"calculator","arguments":{"expr":"12*45-37"}}</tool_call>
[tool]      503
[assistant] <think>...</think>
            <tool_call>{"name":"calculator","arguments":{"expr":"503/8"}}</tool_call>
[tool]      62.875
[assistant] <answer>62.875</answer>          → reward=1 if matches solution
```

## 4. 命令

```bash
swift rlhf \
    --rlhf_type grpo \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset my-tool-math-data \
    --multi_turn true \
    --multi_turn_func toolbench \          # 内置多轮环境函数
    --max_turns 6 \
    --max_completion_length 4096 \
    --reward_funcs accuracy format \
    --num_generations 4 \
    --use_vllm true \
    --output_dir out/qwen-tool-grpo
```

## 5. 自定义多轮环境函数

```python
from swift.plugin import register_multi_turn

@register_multi_turn('my_calc')
def my_calc_env(messages, **kwargs):
    """接收 messages（包含最新 assistant 轮），返回下一轮 tool/user 消息.
    返回 None 表示对话结束."""
    last = messages[-1]['content']
    import re
    m = re.search(r'<tool_call>(.+?)</tool_call>', last, re.S)
    if not m:
        return None                     # 没有调用，结束
    
    import json
    call = json.loads(m.group(1))
    if call['name'] == 'calculator':
        try:
            result = str(eval(call['arguments']['expr']))
        except Exception as e:
            result = f'Error: {e}'
        return {'role': 'tool', 'content': result}
    return None
```

```bash
swift rlhf --rlhf_type grpo \
    --external_plugins my_pkg/envs.py \
    --multi_turn_func my_calc \
    --max_turns 6 ...
```

## 6. Loss 与 Loss Mask

> [!important]
> 
> - 多轮采样中，**只对 assistant 轮计算 GRPO loss**，tool/user 轮 label 置 -100。
> 
> - KL 与 ratio 计算**按轮拼接**，但只给 assistant token 贡献梯度。
> 
> - 使用 `loss_scale` 可进一步调节 think/answer/tool_call 不同部分权重。

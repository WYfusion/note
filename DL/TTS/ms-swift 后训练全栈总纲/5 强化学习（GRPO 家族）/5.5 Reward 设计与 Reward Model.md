## 0. 定位

> 一句话：GRPO 的 reward 可以是**规则函数**、**可验证 reward**、或 **Reward Model** 打分，多个 reward 可加权叠加。

---

## 1. ms-swift 内置 reward

|名称|含义|适用|`accuracy`|与 solution 完全匹配 (1) 或不匹配 (0)|数学、事实问答|
|---|---|---|---|---|---|
|`format`|是否包含 `<think>...</think><answer>...</answer>`|R1 风格训练|`cosine`|与参考 logp 的 cosine 相似|风格保持|
|`repetition`|重复 token 惩罚|防重复|`soft_overlong`|超长软惩罚|控制输出长度|

## 2. 多 reward 叠加

```bash
swift rlhf --rlhf_type grpo \
    --reward_funcs accuracy format soft_overlong \
    --reward_weights 2.0 0.5 0.2 \
    --max_completion_length 2048
```

总 reward = 2.0×accuracy + 0.5×format + 0.2×soft_overlong。

## 3. 自定义 reward二例子

```python
# my_pkg/rewards.py
import re
from swift.plugin import register_orm  # ms-swift orm = outcome reward model

@register_orm('math_format')
def math_format_reward(completion, solution=None, **kwargs):
    """要求输出包含 \\boxed{...} 并与参考答案一致."""
    m = re.search(r'\\boxed\{([^}]*)\}', completion)
    if not m:
        return 0.0
    pred = m.group(1).strip()
    return 1.0 if pred == solution.strip() else 0.0

@register_orm('code_pass')
def code_pass_reward(completion, test_cases=None, **kwargs):
    """要求代码能运行通过所有测试。这里仅示意."""
    code = extract_python_code(completion)        # 你自己实现
    passed = run_in_sandbox(code, test_cases)     # 你自己实现
    return passed / len(test_cases) if test_cases else 0.0
```

```bash
# 训练时引用
swift rlhf --rlhf_type grpo \
    --external_plugins my_pkg/rewards.py \
    --reward_funcs math_format \
    ...
```

## 5. Reward Hacking 排查

|现象|可能原因|应对|
|---|---|---|
|输出越来越长|隐含长度 bias|加 soft_overlong 软惩罚|
|输出乱码但包含答案|accuracy 正则太宽|收窄匹配正则|

> [!important]
> 
> **训练中要同时监控**：reward / 平均长度 / KL / pass@1。只看 reward 处于状态错误。

## 6. 设计原则

> [!important]
> 
> - **主 reward 一定要严格可验证**（accuracy / pass rate），权重 ≥ 1.0。
> 
> - **辅助 reward 只作为 tie-breaker**（format / length / repetition），权重 0.1-0.5。
> 
> - **避免多个辅助 reward 总和接近主 reward**，防止黯淡主信号。

## 7. 完整示例：数学推理混合 reward

```bash
swift rlhf --rlhf_type grpo \
    --model Qwen/Qwen2.5-Math-7B \
    --dataset AI-MO/NuminaMath-CoT \
    --reward_funcs accuracy format soft_overlong \
    --reward_weights 2.0 0.3 0.2 \
    --max_completion_length 4096 \
    --soft_overlong_min_len 2048 \
    --num_generations 8 \
    --use_vllm true \
    --output_dir out/qwen-math-mixed
```

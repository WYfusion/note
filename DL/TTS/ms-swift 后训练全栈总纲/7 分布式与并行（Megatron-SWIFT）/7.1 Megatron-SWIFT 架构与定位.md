## 0. 定位

> 一句话：Megatron-SWIFT = ms-swift 接入 Megatron-LM 与 Megatron-Core，提供全维度并行，适合**70B+ 到 600B+ MoE** 训练。

---

## 3. 安装

```bash
# Megatron-SWIFT 依赖额外安装
pip install -U ms-swift[megatron]

# Megatron-Core（从源码安装推荐）
git clone https://github.com/NVIDIA/Megatron-LM.git
cd Megatron-LM && pip install -e .
```

## 4. 转换权重

```bash
# HuggingFace ⚡ Megatron-Core 格式
swift export \
    --model Qwen/Qwen2.5-72B-Instruct \
    --to_mcore true \
    --output_dir mcore/qwen2.5-72b

# Megatron ⚡ HF（训练后）
swift export \
    --mcore_model out/qwen-mg-sft \
    --to_hf true \
    --output_dir hf/qwen-sft-merged
```

## 5. 最小 Megatron 训练样本

```bash
# 8×H100 单机上训 32B
NPROC_PER_NODE=8 \
megatron sft \
    --load mcore/qwen2.5-32b \
    --dataset alpaca-en \
    --tensor_model_parallel_size 4 \
    --pipeline_model_parallel_size 2 \
    --sequence_parallel true \
    --train_type full \
    --micro_batch_size 1 \
    --global_batch_size 16 \
    --bf16 true \
    --learning_rate 1e-5 \
    --save out/qwen-32b-mg-sft
```

注：命令入口为 `megatron sft / megatron rlhf`，参数名与 Megatron-LM 保持一致。

## 6. 与 transformers 后端的差异

|项|transformers|Megatron-SWIFT|并行|FSDP/DeepSpeed|TP/PP/EP/SP/CP/VPP|
|---|---|---|---|---|---|
|MoE|仅 EP=1|全 EP 支持|上下文长度|≤ 32k 游刃|≥ 128k 可行|
|检点格式|HF|MCore (互转)|调试便利|高|中等|

## 7. 遵循原则

> [!important]
> 
> - **能不上 Megatron 就不上**：调试复杂度高，不是 "免费加速"。
> 
> - **先从 TP/PP 开始**：不要一上来同时开 SP/CP/EP/VPP。
> 
> - **检点与 HF 互转必须验证**：转后跑一次 inference 确认 loss 与原模型一致。

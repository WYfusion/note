## 0. 定位

> 一句话：本节以中文领域 QA 为例，给出可复现的 Embedding + Reranker 训练与评测脚本。

## 1. 资源与选型

|场景|Embedding|Reranker|
|---|---|---|
|中文 + 可面事则|Qwen3-Embedding-8B|Qwen3-Reranker-4B|
|资源受限|BGE-Small (109M)|BGE-Reranker-Base (110M)|

## 2. Embedding 训练脚本

```bash
#!/bin/bash
# 8×A100, BGE-M3 微调中文 QA
swift sft \
    --model BAAI/bge-m3 \
    --task_type embedding \
    --train_type lora \
    --lora_rank 8 \
    --target_modules all-linear \
    --dataset my-zh-qa-pair \
    --loss_type infonce \
    --temperature 0.05 \
    --num_negatives 4 \
    --hard_negatives_mining true \
    --max_length 512 \
    --per_device_train_batch_size 32 \
    --gradient_accumulation_steps 1 \        # 总 batch=32*8=256
    --learning_rate 2e-5 \
    --num_train_epochs 3 \
    --output_dir out/bge-m3-zh
```

## 3. Reranker 训练脚本

```bash
swift sft \
    --model BAAI/bge-reranker-v2-m3 \
    --task_type reranker \
    --train_type lora \
    --lora_rank 8 \
    --target_modules all-linear \
    --dataset my-zh-qa-listwise \
    --loss_type listwise \
    --max_length 512 \
    --per_device_train_batch_size 16 \
    --learning_rate 1e-5 \
    --num_train_epochs 3 \
    --output_dir out/reranker-zh
```

## 5. RAG 闭环集成

```python
import numpy as np
import faiss
from swift.llm import infer_main, InferArguments

# 1. Embedding 入库
emb_args = InferArguments(model='out/bge-m3-zh', task_type='embedding', infer_backend='pt')
docs = open('docs.txt').read().splitlines()
doc_embs = np.array(infer_main(emb_args, docs))     # [N, h]

index = faiss.IndexFlatIP(doc_embs.shape[1])
index.add(doc_embs.astype('float32'))

# 2. 检索 + 精排
rr_args = InferArguments(model='out/reranker-zh', task_type='reranker', infer_backend='pt')

def retrieve(query: str, top_emb=100, top_final=10):
    q_emb = np.array(infer_main(emb_args, [query]))[0:1]
    _, idx = index.search(q_emb.astype('float32'), top_emb)
    candidates = [docs[i] for i in idx[0]]
    
    pairs = [{'query': query, 'doc': c} for c in candidates]
    scores = infer_main(rr_args, pairs)
    ordered = sorted(zip(candidates, scores), key=lambda x: -x[1])
    return ordered[:top_final]

results = retrieve('Notion 怎么使用 AI?')
for doc, score in results:
    print(f'{score:.3f}  {doc[:80]}')
```

## 7. 踩坑提示

> [!important]
> 
> - **评测集与训练集不能重叠**：N-gram 过滤必要。
> 
> - **batch_size 需足够大**：InfoNCE 重要依赖。勿设 < 32。
> 
> - **Reranker 推理贵**：主流 cross-encoder 三 100 个 doc 需 ~3s（CPU）或 ~300ms (GPU)。考虑上限 50。
> 
> - **合并 LoRA 后部署**：训练后 `swift export --merge_lora true` 出 HF 格式供 vLLM 加载。

## 0. 定位

> 一句话：GaLore 不是 PEFT，而是**全参数训练**的省显存技术——把**梯度**投影到低秩子空间再做优化器更新，省下大半 optimizer 显存。

---

## 1. GaLore vs LoRA

|维度|LoRA|GaLore|
|---|---|---|
|是否改前向|是（加旁路）|否（与原始 forward 完全一致）|
|训练上限|低秩约束|同全参|
|训练参数|仅 LoRA 矩阵 BA|**全参数**|
|显存节省点|梯度 + optimizer|仅 optimizer state|
|典型组合|—|+8bit Adam，效果接近 LoRA|

## 2. 数学原理

对每个参数矩阵 $W$，标准优化器存 `m, v`（一阶/二阶动量）各一份，显存 = $2 \times |W|$。

GaLore 把梯度 $G$ 投影到秩-$r$ 子空间：

$G_{\text{low}} = P^{\top} G \quad (r \times n)$

其中 $P \in \mathbb{R}^{m \times r}$ 是从 $G$ 的 SVD 中取的左奇异向量（每隔 $T$ 步刷新一次）。

优化器只在低秩空间里维护 $m, v$，回投到原空间更新：

$\Delta W = - \eta \cdot P \cdot \text{Adam}(G_{\text{low}})$

Optimizer 显存从 $2|W|$ 降到 $\frac{2 r}{m}|W|$，对 $m=4096, r=128$ 仅 6%。

## 3. Q-GaLore：再叠加 8bit

Q-GaLore 在 GaLore 基础上：

1. **8bit 量化**优化器状态（来自 8bit Adam）。

2. **自适应 P 更新频率**：当 P 不再变化时降低更新，节省 SVD 开销。

3. 让 7B 全参在 24GB 显存可训。

## 4. 命令

```bash
# 标准 GaLore
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset my-data \
    --train_type full \
    --use_galore true \
    --galore_rank 128 \
    --galore_update_proj_gap 200 \
    --galore_scale 0.25 \
    --gradient_checkpointing true \
    --learning_rate 2e-5 \
    --output_dir out/qwen-galore

# Q-GaLore（叠加 8bit Adam）
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset my-data \
    --train_type full \
    --use_galore true \
    --galore_rank 128 \
    --galore_quantization true \
    --galore_proj_quant true \
    --optim galore_adamw_8bit \
    --gradient_checkpointing true \
    --learning_rate 2e-5 \
    --output_dir out/qwen-qgalore
```

## 5. 关键超参

|参数|常用值|含义|
|---|---|---|
|`galore_update_proj_gap`|200|每隔多少步重做 SVD 刷新 P|
|`optim`|`galore_adamw_8bit`|建议直接用 8bit 版本|

## 6. 何时用 GaLore vs LoRA vs 全参

![[3.6 GaLore - Q-GaLore（梯度低秩投影） - 6. 何时用 GaLore vs LoRA vs 全参.excalidraw|800]]

> [!important]
> 
> **何时优先 GaLore**：你需要全参数训练的效果（如 CPT、复杂下游任务），但又没有足够显存做标准全参。GaLore 提供「全参表达力 + LoRA 级显存」的折中。

## 7. 局限

> [!important]
> 
> - SVD 刷新 P 有计算开销，长训练任务建议增大 `update_proj_gap`。  
>     - 与 DeepSpeed Z3 / FSDP 配合时需要谨慎，部分版本不兼容。  
>     - 经验上 GaLore 收敛速度略慢于全参，需要稍长 warmup。

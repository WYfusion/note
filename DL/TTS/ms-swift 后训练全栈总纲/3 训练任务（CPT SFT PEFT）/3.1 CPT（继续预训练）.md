## 0. 定位

> 一句话：CPT（Continued Pre-Training）= 在通用 base 模型上继续做**纯文本 next-token prediction**，注入领域知识。

---

## 1. 何时用 CPT

|需求|是否需要 CPT|
|---|---|
|注入大量领域语料（金融/医学/法律）|✅ CPT → SFT|
|对齐风格 / 偏好|❌ 用 DPO/SimPO|
|对齐到指令格式（IM/Chat）|❌ 用 SFT 即可|
|扩词表 / 跨语言适配|✅ 扩词表后必须 CPT|
|提升数学 / 代码推理|❌ 用 GRPO（除非缺少基础知识）|

## 2. 数据 schema

```jsx
{"response": "医保基金的征缴范围包括基本医疗保险、生育保险...（一段较长的领域文本）"}
{"response": "截至 2024 年 6 月,某省累计参保人数达 8632 万,基金支出 412 亿元..."}
```

仅一个字段 `response`，被当作连续 token 流；通常配合 `packing=true` 把多条短文本拼到 `max_length`。

## 3. Loss

标准的语言模型 next-token 交叉熵：

$\mathcal{L}_{\mathrm{CPT}} = -\dfrac{1}{T}\sum_{t=1}^{T} \log P_\theta(x_t \mid x_{<t})$

所有 token 都参与监督（与 SFT 仅监督 response 不同）。

## 4. 命令骨架

```bash
NPROC_PER_NODE=8 swift pt \
    --model Qwen/Qwen2.5-7B \
    --dataset my-finance-corpus#5000000 \
    --max_length 4096 \
    --packing true \
    --attn_impl flash_attn \
    --train_type full \
    --torch_dtype bfloat16 \
    --deepspeed zero3 \
    --learning_rate 5e-6 \
    --warmup_ratio 0.02 \
    --max_steps 20000 \
    --save_strategy steps --save_steps 1000 \
    --gradient_checkpointing true \
    --output_dir out/qwen2.5-7b-cpt
```

关键参数说明：

|参数|建议|
|---|---|
|`--warmup_ratio`|0.01-0.05|
|`--max_steps`|用 steps 而非 epoch（流式数据更友好）|
|`--learning_rate`|5e-6 ~ 2e-5（远小于 SFT 的 1e-4）|
|`--packing true`|必开，否则 GPU 利用率低|
|`--train_type full`|知识注入推荐全参；显存不足可用 LoRA-Plus 但效果折扣|

## 5. 完整 Python 调用

```python
from swift.llm import TrainArguments, sft_main

args = TrainArguments(
    model='Qwen/Qwen2.5-7B',
    template='qwen',
    dataset=['my-finance-corpus'],
    max_length=4096,
    packing=True,
    attn_impl='flash_attn',
    train_type='full',
    torch_dtype='bfloat16',
    deepspeed='zero3',
    learning_rate=5e-6,
    warmup_ratio=0.02,
    max_steps=20000,
    save_strategy='steps', save_steps=1000,
    gradient_checkpointing=True,
    output_dir='out/qwen2.5-7b-cpt',
    # 关键：pt 模式下 ms-swift 会切换到纯 LM loss
    task_type='pt',
)
sft_main(args)
```

## 6. CPT 与扩词表配合

```bash
# 步骤 1: 扩词表（脚本略，关键是 resize_token_embeddings 并初始化新行）
# 步骤 2: CPT 阶段
swift pt \
    --model my-extended-vocab-qwen \
    --dataset my-zh-corpus \
    --train_type full \
    --learning_rate 1e-5     # 新嵌入需要稍大 LR；可单独提高 embed LR
```

> [!important]
> 
> **经验**：扩词表后建议先 freeze 主体只训 embed，再放开全参 CPT，避免新 embed 漂移过大引发崩溃。

## 0. 定位

> 一句话：LoRA = 在 base 权重 $W$ 旁加旁路 $\Delta W = BA$，**只训练** $B, A$ **两个低秩矩阵**，base 权重冻结。

---

## 1. 数学原理

对线性层 $y = Wx$，LoRA 改写为：

$y = W x + \Delta W \cdot x = W x + \frac{\alpha}{r}B A x$

- $W \in \mathbb{R}^{d_{out} \times d_{in}}$：冻结的预训练权重

- $A \in \mathbb{R}^{r \times d_{in}}$：高斯初始化

- $B \in \mathbb{R}^{d_{out} \times r}$：零初始化（保证训练初 $\Delta W = 0$）

- $r \ll \min(d_{in}, d_{out})$：秩

- $\alpha$：缩放系数，常取 $\alpha = 2r$

参数量从 $d_{in} \times d_{out}$ 降到 $r(d_{in}+d_{out})$，对 7B 模型 ≈ 0.1%-1% 可训参。

## 3. 命令模板

```bash
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset AI-ModelScope/alpaca-gpt4-data-zh#10000 \
    --train_type lora \
    --lora_rank 8 \
    --lora_alpha 32 \
    --target_modules all-linear \
    --lora_dropout 0.0 \
    --num_train_epochs 2 \
    --learning_rate 1e-4 \
    --output_dir out/qwen-lora
```

## 4. 关键超参速查

|超参|常用范围|含义|
|---|---|---|
|`lora_alpha`|= 2 r 或 = r|缩放，与 r 联动|
|`lora_dropout`|0.0 - 0.1|大数据集可设 0|
|`lora_lr_ratio`|16.0 (LoRA+)|给 B 更大 LR，提升表达力|

## 5. 一段最小 Python 代码：手动构造 LoRA

```python
import torch, torch.nn as nn

class LoRALinear(nn.Module):
    def __init__(self, base: nn.Linear, r: int = 8, alpha: int = 16):
        super().__init__()
        self.base = base
        for p in self.base.parameters():
            p.requires_grad = False                 # 冻结 base
        self.A = nn.Parameter(
            torch.empty(r, base.in_features).normal_(std=1/r))
        self.B = nn.Parameter(
            torch.zeros(base.out_features, r))      # 零初始化
        self.scale = alpha / r

    def forward(self, x):
        return self.base(x) + (x @ self.A.T @ self.B.T) * self.scale

# 替换某个线性层
from transformers import AutoModelForCausalLM
model = AutoModelForCausalLM.from_pretrained('Qwen/Qwen2.5-7B-Instruct', torch_dtype=torch.bfloat16)
for name, mod in model.named_modules():
    if name.endswith('q_proj') or name.endswith('v_proj'):
        parent = model.get_submodule(name.rsplit('.', 1)[0])
        setattr(parent, name.rsplit('.', 1)[1], LoRALinear(mod, r=8, alpha=16))

# 只训 LoRA 参数
trainable = [p for p in model.parameters() if p.requires_grad]
print(f'trainable params: {sum(p.numel() for p in trainable)/1e6:.2f} M')
```

这段示意代码帮助理解 LoRA 内部机制；生产场景直接用 `swift sft --train_type lora` 即可（底层是 PEFT）。

## 7. LoRA vs 全参对比经验

> [!important]
> 
> - **能力激发** 任务（指令对齐、风格调整）：LoRA 与全参差距很小，r=8 即够。  
>     - **知识注入** 任务（CPT-SFT 大数据）：LoRA 容量瓶颈明显，建议 r=64+ 或全参。  
>     - **多任务 / 多领域** 切换：LoRA 优势巨大，每个域一个 adapter，按需热切。

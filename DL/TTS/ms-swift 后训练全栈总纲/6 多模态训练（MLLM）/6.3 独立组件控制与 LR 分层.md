## 0. 定位

> 一句话：ms-swift 可以独立控制 vision/aligner/LLM 三部件的 **冻结状态** 与 **学习率**，能够复现几乎所有经典训练配方。

---

## 1. 四种常见训练配置

|配置|vision|aligner|LLM|适用场景|
|---|---|---|---|---|
|**Connector-Train**|冻|训|冻|从头对齐 / aligner 重设计|
|**Full**|训|训|训|从头预训练（少见）|

## 2. 冻结控制参数

```bash
# 默认 LoRA-Light
swift sft \
    --model Qwen/Qwen2.5-VL-7B-Instruct \
    --dataset my-vqa \
    --train_type lora \
    --target_modules all-linear \
    --freeze_vit true \
    --freeze_aligner true \
    --output_dir out/light

# Connector-Train
swift sft --model Qwen/Qwen2.5-VL-7B-Instruct \
    --dataset my-align-data \
    --train_type full \
    --freeze_vit true --freeze_llm true \
    --aligner_lr 2e-4   # 又高于默认 lr
```

## 3. 三部件独立 LR

论文经验：aligner 需要较大 LR，vision 需要较小 LR。

```bash
swift sft --model Qwen/Qwen2.5-VL-7B-Instruct \
    --dataset my-vqa \
    --train_type full \
    --freeze_vit false \
    --freeze_aligner false \
    --freeze_llm false \
    --vit_lr 1e-6 \
    --aligner_lr 2e-5 \
    --llm_lr 1e-5 \
    --learning_rate 1e-5    # 默认 lr（其他组件覆盖）
```

## 4. LR 调参参考表

|场景|vit_lr|aligner_lr|llm_lr|
|---|---|---|---|
|Stage 2（aligner+LLM）|冻|2e-5|2e-5|
|LoRA-Light（领域任务）|冻|冻|1e-4 (LoRA)|

## 5. Python 等价调用

```python
from swift.llm import TrainArguments, sft_main

sft_main(TrainArguments(
    model='Qwen/Qwen2.5-VL-7B-Instruct',
    template='qwen2_5_vl',
    dataset=['my-vqa'],
    train_type='lora',
    target_modules='all-linear',
    freeze_vit=True,
    freeze_aligner=True,
    freeze_llm=False,
    learning_rate=1e-4,
    output_dir='out/light',
))
```

## 6. 验证 trainable params

```python
import torch

def summarize_trainable(model):
    parts = {'visual': 0, 'aligner': 0, 'llm': 0, 'other': 0}
    for name, p in model.named_parameters():
        if not p.requires_grad: continue
        if name.startswith('visual') and 'merger' not in name:
            parts['visual'] += p.numel()
        elif 'merger' in name or 'aligner' in name or 'connector' in name:
            parts['aligner'] += p.numel()
        elif name.startswith('model') or name.startswith('language_model'):
            parts['llm'] += p.numel()
        else:
            parts['other'] += p.numel()
    for k, v in parts.items():
        print(f'{k:8s}: {v/1e6:7.2f} M trainable')

summarize_trainable(model)
```

训练启动后严重建议先调用一次，确认各部分可训参量与预期一致。

## 7. 常见问题

> [!important]
> 
> - **vit_lr 太大** → 视觉能力衰减（原图识别错）。
> 
> - **aligner_lr 太小** → 领域适配不动。
> 
> - **未冻 LLM 但不冻 vit** → 避免的：训练不稳定性高，资源浪费。

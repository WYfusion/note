## 0. 定位

> 一句话：主流 MLLM = **vision encoder + aligner + LLM** 三部件，训练时可独立控制冻结 / 调优。

---

## 1. 三部件架构

![[6.1 多模态架构与三部件 - 1. 三部件架构 - 图 01.excalidraw|800]]

|部件|作用|参数量参考|**Vision Encoder**|图像 → 视觉 token|0.3B - 4B|
|---|---|---|---|---|---|
|**Aligner**|视觉 token → LLM 输入空间|仅几 M - 100M|**LLM**|生成文本|1B - 70B+|

## 2. 常见 MLLM 系列

|系列|视觉编码器|对齐器|LLM|
|---|---|---|---|
|**LLaVA**|CLIP-ViT-L/14|MLP|LLaMA / Vicuna|
|**MiniCPM-V**|SigLIP|Resampler|MiniCPM|

## 4. 独立控制三部件

```bash
# 只训 LLM，冻结 vision + aligner（最常见）
swift sft \
    --model Qwen/Qwen2.5-VL-7B-Instruct \
    --dataset my-vqa-data \
    --train_type lora \
    --target_modules all-linear \
    --freeze_vit true \
    --freeze_aligner true \
    --output_dir out/qwen-vl-llm-only

# 只训 aligner，冻结 vision 与 LLM
swift sft \
    --model Qwen/Qwen2.5-VL-7B-Instruct \
    --dataset my-vqa-data \
    --train_type full \
    --freeze_vit true \
    --freeze_llm true \
    --learning_rate 2e-4
```

## 5. 重要参数

|参数|作用|
|---|---|
|`--freeze_aligner true\|false`|冻结对齐层|
|`--vit_lr` / `--aligner_lr` / `--llm_lr`|三部件独立 LR|
|`--video_max_frames`|视频最大帧数|

## 6. 加载与检查三部件

```python
from swift.llm import get_model_tokenizer

model, tokenizer = get_model_tokenizer(
    'Qwen/Qwen2.5-VL-7B-Instruct',
    torch_dtype='bfloat16',
)

# 看三个子模块的参数量
for name, sub in model.named_modules():
    if name in ('visual', 'visual.merger', 'model'):  # 会随架构变
import torch
for part_name in ['visual', 'visual.merger', 'model.embed_tokens']:
    sub = dict(model.named_modules()).get(part_name)
    if sub is None:
        continue
    n = sum(p.numel() for p in sub.parameters())
    print(f'{part_name:24s}: {n/1e6:7.1f} M')
```

## 7. 训练推荐组合

|场景|推荐组合|
|---|---|
|领域适配（医学影像）|vit-LoRA + aligner 训 + LLM-LoRA|
|仅优化文本生成风格|vit/aligner 都冻，只 LoRA LLM|

> [!important]
> 
> 冻结 vision 是默认选项：视觉编码器与预训练量极不匹配，随意训会损坏一般能力。

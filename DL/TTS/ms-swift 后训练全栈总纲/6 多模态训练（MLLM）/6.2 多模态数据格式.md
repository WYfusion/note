## 0. 定位

> 一句话：多模态样本在 messages 中用占位符（`<image>` / `<video>` / `<audio>`）引用，外部资源放 `images` / `videos` / `audios` 字段。

---

## 1. 图像 SFT

```jsx
# 单图
{"messages": [
    {"role":"user",     "content":"<image>这是什么品种的猫?"},
    {"role":"assistant","content":"这是一只布偶猫，颜色为重点色..."}
], "images": ["/data/img/cat_001.jpg"]}

# 多图
{"messages": [
    {"role":"user",     "content":"比较两张图 ：<image><image>"},
    {"role":"assistant","content":"第一张是...第二张是..."}
], "images": ["/data/a.jpg", "/data/b.jpg"]}

# 从 URL 加载
{"messages": [
    {"role":"user",     "content":"<image>描述这张图"},
    {"role":"assistant","content":"..."}
], "images": ["https://example.com/img.jpg"]}
```

## 2. 视频 SFT

```jsx
# 视频文件
{"messages": [
    {"role":"user",     "content":"<video>视频里发生了什么?"},
    {"role":"assistant","content":"一只黑色猞猜从床上跳下。"}
], "videos": ["/data/clip_42.mp4"]}

# 多个映像文件以 base64
{"messages": [...],
 "videos": ["data:video/mp4;base64,..."]}
```

## 3. 音频 SFT

```jsx
{"messages": [
    {"role":"user",     "content":"<audio>转写这段录音"},
    {"role":"assistant","content":"“今天天气不错，适合出门。”"}
], "audios": ["/data/audio_000.wav"]}
```

## 4. 多模态偏好数据（DPO/GRPO）

```jsx
# DPO
{"messages":[{"role":"user","content":"<image>这个 UI 是什么?"}],
 "chosen":   [{"role":"assistant","content":"这是 Notion 的侧边栏..."}],
 "rejected": [{"role":"assistant","content":"这是一个文本编辑器。"}],
 "images": ["/data/notion_screenshot.png"]}
```

## 5. 多模态预处理示例

```python
from swift.llm import get_template, get_model_tokenizer

model, tokenizer = get_model_tokenizer('Qwen/Qwen2.5-VL-7B-Instruct',
                                       load_model=False)
template = get_template('qwen2_5_vl', tokenizer)

sample = {
    'messages': [
        {'role':'user','content':'<image>What animal is this?'},
        {'role':'assistant','content':'A Ragdoll cat.'},
    ],
    'images': ['/data/cat.jpg'],
}
enc = template.encode(sample)
print(enc.keys())  # input_ids, labels, pixel_values, image_grid_thw, ...
print('vision tokens:', (enc['input_ids']==template.image_token_id).sum().item())
```

## 6. 资源控制参数

## 7. 踩坑提示

> [!important]
> 
> - **占位符不匹配**：messages 中 `<image>` 个数必须等于 `images` 列表长度。
> 
> - **图像存取**：远程 URL 不建议用于大规模训练（需预下载缓存）。
> 
> - **视频帧数与显存**：单个视频 token 数 ≈ 帧数 × 每帧 token，多轮加上下文易 OOM。
> 
> - **单机多卡 packing**：单样本包含多图时，packing 逻辑变复杂，建议先关闭调试。

## 0. 定位

> 一句话：`swift` CLI 是 ms-swift 的**唯一推荐入口**，把 9 个核心对象的组合用一行命令固化下来。

## 1. 子命令一览

|子命令|用途|对应 Trainer|
|---|---|---|
|`swift sft`|SFT / PEFT|Seq2SeqTrainer|
|`swift rollout`|启动 rollout server（GRPO Async）|—|
|`swift infer`|本地推理（CLI/REPL）|—|
|`swift app`|启动 Web UI|—|
|`swift export`|合并 LoRA / 量化导出|—|
|`swift pt`|继续预训练 (CPT)|Seq2SeqTrainer|
|`swift rlhf`|偏好 / 强化（DPO/GRPO/KTO/...）|DPOTrainer / GRPOTrainer / ...|
|`swift sample`|拒绝采样 / 蒸馏数据生产|—|
|`swift deploy`|启动 OpenAI-兼容 API|—|
|`swift eval`|评测（EvalScope）|—|
|`megatron sft`/`pt`|Megatron-SWIFT 大模型训练|MegatronTrainer|

## 2. 命令骨架

```bash
# SFT-LoRA 单卡（最小可跑）
CUDA_VISIBLE_DEVICES=0 \
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset AI-ModelScope/alpaca-gpt4-data-zh \
    --train_type lora --lora_rank 8 --lora_alpha 32 \
    --num_train_epochs 1 --learning_rate 1e-4 \
    --output_dir out/qwen2.5-lora

# DPO 偏好对齐
swift rlhf \
    --rlhf_type dpo \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset hjh0119/shareAI-Llama3-DPO-zh-en-emoji \
    --train_type lora --beta 0.1 \
    --output_dir out/qwen2.5-dpo

# GRPO（数学推理）
swift rlhf \
    --rlhf_type grpo \
    --model Qwen/Qwen2.5-Math-7B \
    --dataset AI-MO/NuminaMath-CoT \
    --reward_funcs accuracy format \
    --num_generations 8 --use_vllm true \
    --output_dir out/qwen-grpo

# 合并 LoRA + AWQ 4bit 量化导出
swift export \
    --adapters out/qwen2.5-lora/checkpoint-final \
    --merge_lora true \
    --quant_method awq --quant_bits 4 \
    --output_dir out/qwen2.5-awq

# 部署 OpenAI 兼容 API（vLLM 后端）
swift deploy \
    --model out/qwen2.5-awq \
    --infer_backend vllm \
    --port 8000
```

## 3. 参数来源优先级

![[1.3 命令行入口（swift CLI） - 3. 参数来源优先级.excalidraw|800]]

> [!important]
> 
> 优先级（高→低）：**命令行 > yaml/json > 环境变量 > 默认值**。生产环境推荐用 yaml 固化基线，命令行只覆盖少量差异参数。

## 4. yaml 配置示例

```yaml
# configs/qwen_sft_lora.yaml
model: Qwen/Qwen2.5-7B-Instruct
template: qwen
dataset:
  - AI-ModelScope/alpaca-gpt4-data-zh
  - swift/self-cognition  # 自我认知数据
train_type: lora
lora_rank: 8
lora_alpha: 32
target_modules: all-linear
num_train_epochs: 2
per_device_train_batch_size: 4
gradient_accumulation_steps: 8
learning_rate: 1.0e-4
warmup_ratio: 0.05
save_strategy: epoch
output_dir: out/qwen2.5-lora
```

```bash
swift sft --config configs/qwen_sft_lora.yaml --learning_rate 5e-5  # 覆盖 lr
```

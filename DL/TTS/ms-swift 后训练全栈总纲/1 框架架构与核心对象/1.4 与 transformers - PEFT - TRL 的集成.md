## 0. 定位

> 一句话：ms-swift **不重造轮子**，而是把 transformers/PEFT/TRL/DeepSpeed/Megatron 等组件按「后训练流水线」拼装并补齐工程缺口。

---

## 1. 集成关系一览

|组件|来源|ms-swift 在哪里增强|
|---|---|---|
|LoRA / DoRA / PiSSA|HF PEFT|统一 Args、自动 target_modules、保存格式|
|PPO / GRPO|TRL + 自研|vLLM/SGLang rollout、Async colocate、custom reward|
|Megatron-LM|NVIDIA|HF↔MCore 权重转换、TP/PP/EP 解锁 MoE 10x|
|Flash-Attention 2/3|Tri Dao 等|`attn_impl=flash_attn`|
|Model / Tokenizer|HF transformers|统一注册表 + patch（template/loss mask）|
|DPO / KTO / ORPO / SimPO|HF TRL|统一 Trainer 入口、template 兼容、多轮支持|
|DeepSpeed / FSDP|HF accelerate|预置 Z2/Z3 配置、FSDP-QLoRA 支持|
|Liger Kernel|LinkedIn|一键开关 `use_liger_kernel`|
|vLLM / SGLang / LMDeploy|外部|统一 `infer_backend` 抽象|

## 3. 与原生 PEFT 等价对照

```python
# ===== 原生 transformers + PEFT 写法 =====
from transformers import AutoModelForCausalLM, AutoTokenizer, Trainer, TrainingArguments
from peft import LoraConfig, get_peft_model
from datasets import load_dataset

model = AutoModelForCausalLM.from_pretrained('Qwen/Qwen2.5-7B-Instruct', torch_dtype='bfloat16')
tok = AutoTokenizer.from_pretrained('Qwen/Qwen2.5-7B-Instruct')
model = get_peft_model(model, LoraConfig(
    r=8, lora_alpha=32, target_modules='all-linear',
    lora_dropout=0.0, bias='none', task_type='CAUSAL_LM',
))
ds = load_dataset('json', data_files='data.jsonl')

# ⚠️ 你还需要：手写 chat template、loss mask、collator、save_adapter...
trainer = Trainer(
    model=model, args=TrainingArguments(...),
    train_dataset=ds, tokenizer=tok,
)
trainer.train()

# ===== ms-swift 等价命令 =====
# swift sft \
#     --model Qwen/Qwen2.5-7B-Instruct \
#     --train_type lora --lora_rank 8 --lora_alpha 32 \
#     --target_modules all-linear \
#     --dataset data.jsonl --template qwen \
#     --output_dir out
```

> [!important]
> 
> ms-swift 把 **template 拼接、loss mask、collator、adapter 保存、断点续训** 等工程细节固化进 CLI，避免每次重写。

## 4. 何时直接用底层组件

|场景|建议|
|---|---|
|研究自定义 RL 目标|fork swift/trainers 或继承 GRPOTrainer|
|规模化生产|用 swift CLI + yaml 固化|

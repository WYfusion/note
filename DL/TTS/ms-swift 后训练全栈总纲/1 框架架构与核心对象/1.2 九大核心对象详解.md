## 0. 定位

> 一句话：把 ms-swift 一切操作收敛到 **九个核心对象**，知道每个对象的职责与生命周期，就能定位任意 bug。

---

## 1. 九大对象速查

|对象|类型|核心职责|生命周期|
|---|---|---|---|
|**Model**|nn.Module|权重 + forward|load → patch → train → save|
|**Template**|Template|chat 拼接 + loss mask|encode/decode 阶段|
|**Trainer**|Seq2SeqTrainer 派生|训练循环|train()/predict()|
|**Backend**|vLLM/SGLang/...|推理后端|infer/rollout 阶段|
|**Args**|dataclass|所有超参与开关|程序启动时一次性解析|
|**Tokenizer**|PreTrainedTokenizer|文本 ↔ token id|与 Model 同寿|
|**Dataset**|HF Dataset|样本流|train loop 内迭代|
|**Strategy**|DDP/DeepSpeed/FSDP|分布式编排|由 Args 决定|
|**Plugin**|函数注册表|用户自定义扩展点|启动时注册|

## 3. 一段代码看清五个对象

```python
from swift.llm import (
    TrainArguments, get_model_tokenizer,
    get_template, load_dataset, Seq2SeqTrainer,
)

# ① Args
args = TrainArguments(
    model='Qwen/Qwen2.5-7B-Instruct',
    train_type='lora',
    dataset=['AI-ModelScope/alpaca-gpt4-data-zh'],
    template='qwen',
    output_dir='out/qwen2.5-lora',
    num_train_epochs=1,
    lora_rank=8, lora_alpha=32,
)

# ② Model + ③ Tokenizer（patch 在内部完成）
model, tokenizer = get_model_tokenizer(args.model, args)

# ④ Template（决定 chat 拼接 & loss mask）
template = get_template(args.template, tokenizer)

# ⑤ Dataset（已被 template encode 成 input_ids/labels）
train_ds, val_ds = load_dataset(
    args.dataset, template=template, split_dataset_ratio=0.05,
)

# ⑥ Trainer（内部根据 Args 选择 Strategy）
trainer = Seq2SeqTrainer(
    model=model, args=args.training_args,
    train_dataset=train_ds, eval_dataset=val_ds,
    data_collator=template.data_collator,
    tokenizer=tokenizer,
)
trainer.train()
trainer.save_model()
```

## 4. 易错点对照

> [!important]
> 
> - **Template 选错** → loss 计算到 prompt 上，模型学错。
> 
> - **Tokenizer 不匹配** → special token 漂移，infer 输出乱码。
> 
> - **Strategy 与 Model 配置冲突** → DeepSpeed Z3 + LoRA 容易触发 grad 不同步。

## 5. 何时需要绕开默认对象

|需求|替换对象|
|---|---|
|自定义 reward|Plugin → orms|
|跨节点训练|Strategy（DeepSpeed/FSDP/Megatron）|

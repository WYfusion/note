## 1. 数据准备

```bash
mkdir -p data
python - <<'PY'
from datasets import load_dataset
import json, re

def extract_answer(text):
    m = re.search(r'####\s*(-?[0-9.,]+)', text)
    return m.group(1).replace(',', '').strip() if m else None

ds = load_dataset('openai/gsm8k', 'main', split='train')
rows = []
for r in ds:
    ans = extract_answer(r['answer'])
    if ans is None:
        continue
    rows.append({
        'messages': [{'role': 'user',
                      'content': r['question'] +
                                 '\n请逐步推理，最后将答案放在 <answer></answer> 中。'}],
        'final_answer': ans,
    })
import json
with open('data/gsm8k-grpo.jsonl', 'w') as f:
    for r in rows[:5000]:
        f.write(json.dumps(r, ensure_ascii=False) + '\n')
print(len(rows[:5000]))
PY
```

## 2. 奖励函数

```python
# rewards.py
import re

def accuracy_reward(completion: str, final_answer: str, **kw) -> float:
    m = re.search(r'<answer>(.*?)</answer>', completion, re.S)
    if not m:
        return 0.0
    pred = m.group(1).strip().replace(',', '')
    try:
        return 1.0 if abs(float(pred) - float(final_answer)) < 1e-6 else 0.0
    except ValueError:
        return 1.0 if pred == final_answer else 0.0

def format_reward(completion: str, **kw) -> float:
    has_answer = bool(re.search(r'<answer>.*?</answer>', completion, re.S))
    has_step = '步骤' in completion or 'Step' in completion or '\n' in completion
    return 0.5 * (has_answer + has_step) / 2.0
```

## 3. 必要的 SFT 预热（先做 1 轮）

```bash
# GRPO 需要 base 模型有基本格式遵循能力
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset data/gsm8k-grpo.jsonl \
    --train_type lora --lora_rank 16 \
    --target_modules all-linear \
    --num_train_epochs 1 \
    --max_length 2048 \
    --learning_rate 1e-4 \
    --output_dir out/gsm8k-sft
```

## 4. GRPO 训练

```bash
swift rlhf \
    --rlhf_type grpo \
    --model out/gsm8k-sft \
    --dataset data/gsm8k-grpo.jsonl \
    --reward_funcs rewards.accuracy_reward rewards.format_reward \
    --num_generations 8 \
    --temperature 1.0 \
    --max_completion_length 1024 \
    --learning_rate 5e-7 \
    --beta 0.04 \
    --per_device_train_batch_size 1 \
    --gradient_accumulation_steps 8 \
    --num_train_epochs 1 \
    --max_length 4096 \
    --gradient_checkpointing true \
    --bf16 true \
    --logging_steps 5 \
    --save_steps 200 \
    --output_dir out/gsm8k-grpo \
    --report_to swanlab
```

## 7. 调优要点与排错

> [!important]
> 
> - **num_generations**：4-16 之间常用 8。太小奖励信号方差大。
> 
> - **temperature 1.0** 必要：探索多样性。
> 
> - **lr 5e-7 ~ 1e-6**：GRPO 不能用 SFT 的 1e-4。
> 
> - **beta**：0.01-0.1，越大越保守贴近 SFT。

> [!important]
> 
> - **reward 不涨**：先用 SFT 模型当起点，确认格式遵循基础够。
> 
> - **completion_length 过短**：模型直接给答案不推理 → 加 length 奖励或修改 format reward。
> 
> - **OOM**：num_generations × max_completion_length 很贵，先减 num_generations 到 4。
> 
> - **kl 爆炸**：lr 太大 → 降一档。

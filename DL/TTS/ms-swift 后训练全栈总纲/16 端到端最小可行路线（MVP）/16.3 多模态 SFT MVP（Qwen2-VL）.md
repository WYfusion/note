## 1. 数据准备

```bash
mkdir -p data/images

# 用 LLaVA-Instruct 子集（包含图片 + 多轮对话）
python - <<'PY'
from datasets import load_dataset
import json, os

ds = load_dataset('liuhaotian/LLaVA-Instruct-150K', split='train').shuffle(seed=42).select(range(10000))
with open('data/llava-10k.jsonl', 'w') as f:
    for r in ds:
        msgs = []
        for turn in r['conversations']:
            role = 'user' if turn['from'] == 'human' else 'assistant'
            content = turn['value'].replace('<image>', '<image>')
            msgs.append({'role': role, 'content': content})
        f.write(json.dumps({
            'messages': msgs,
            'images': [f'data/images/{r["image"]}'],
        }, ensure_ascii=False) + '\n')
print('done')
PY
# 图片需另行下载（COCO 等）：略
```

## 2. 数据校验

```python
import json
from pathlib import Path

bad = 0
for ln, line in enumerate(open('data/llava-10k.jsonl'), 1):
    r = json.loads(line)
    n_token = sum((m.get('content') or '').count('<image>') for m in r['messages'])
    n_files = len(r.get('images', []))
    if n_token != n_files:
        print(f'L{ln} 占位 {n_token} 文件 {n_files}')
        bad += 1
        continue
    for p in r['images']:
        if not Path(p).exists():
            print(f'L{ln} 缺图 {p}')
            bad += 1
            break
print(f'bad={bad}')
```

## 3. 冒烟训练

```bash
head -n 100 data/llava-10k.jsonl > data/mm_smoke.jsonl
swift sft \
    --model Qwen/Qwen2-VL-7B-Instruct \
    --dataset data/mm_smoke.jsonl \
    --train_type lora --lora_rank 8 \
    --target_modules all-linear \
    --freeze_vit true \
    --freeze_aligner false \
    --num_train_epochs 1 \
    --max_length 4096 \
    --output_dir out/mm-smoke
```

## 4. 正式训练

```bash
swift sft \
    --model Qwen/Qwen2-VL-7B-Instruct \
    --dataset data/llava-10k.jsonl \
    --train_type lora --lora_rank 16 \
    --target_modules all-linear \
    --freeze_vit true \
    --freeze_aligner false \
    --num_train_epochs 1 \
    --max_length 4096 \
    --learning_rate 1e-4 \
    --warmup_ratio 0.05 \
    --per_device_train_batch_size 1 \
    --gradient_accumulation_steps 16 \
    --gradient_checkpointing true \
    --bf16 true \
    --logging_steps 5 \
    --save_steps 200 \
    --output_dir out/qwen2vl-mvp \
    --report_to swanlab
```

## 5. 评测

```bash
swift eval \
```

## 6. 部署 + 调用

```bash
swift deploy \
    --adapters out/qwen2vl-mvp \
    --infer_backend vllm \
    --port 8000 \
    --served_model_name qwen2vl-mvp &

sleep 30 && python - <<'PY'
import base64
from openai import OpenAI

c = OpenAI(base_url='http://localhost:8000/v1', api_key='x')

with open('data/images/sample.jpg', 'rb') as f:
    b64 = base64.b64encode(f.read()).decode()

r = c.chat.completions.create(
    model='qwen2vl-mvp',
    messages=[{'role':'user','content':[
        {'type':'image_url','image_url':{'url': f'data:image/jpeg;base64,{b64}'}},
        {'type':'text','text':'描述这张图片'},
    ]}],
    max_tokens=512,
)
print(r.choices[0].message.content)
PY
```

## 7. 时间与成本表

|阶段|耗时|资源|环境|5 min|—|
|---|---|---|---|---|---|
|数据 + 图片下载|15 min|—|冒烟|10 min|1×A100|
|正式 SFT (10k)|~90 min|1×A100|评测|10 min|1×A100|
|部署测试|10 min|1×A100|**合计**|**~140 min**|**1×A100 80G**|

> [!important]
> 
> **MVP 完成后下一步**：
> 
> 1. 把 LLaVA 换成业务图文数据（OCR / 表格 / 业务截图）。
> 
> 1. 视觉能力差时考虑 `freeze_vit false` 解冻视觉塔（注意 lr 更小）。
> 
> 1. 加视频帧采样训练（Qwen2-VL 原生支持）。
> 
> 1. 进入多模态 GRPO（视觉推理）。

> [!important]
> 
> - **占位符与图片数对齐**：每张图必须有 `<image>` 占位符。
> 
> - **图片路径**：相对路径要相对于训练运行目录，建议绝对路径。
> 
> - **显存爆**：`per_device_train_batch_size=1` + `max_length=4096` 是 7B-VL 在 80G 上的安全档；上 32k 长度时务必启用动态分辨率裁剪。

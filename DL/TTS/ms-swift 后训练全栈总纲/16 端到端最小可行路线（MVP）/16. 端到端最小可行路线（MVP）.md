## 1. MVP-1：通用聊天增强

![[16. 端到端最小可行路线（MVP） - 1. MVP-1：通用聊天增强 - 图 01.excalidraw|800]]

```bash
MODEL=Qwen/Qwen3-8B
DATA=AI-ModelScope/alpaca-gpt4-data-zh
OUT=output/mvp1

# 1. SFT-LoRA
swift sft --model $MODEL --dataset $DATA \
    --train_type lora --target_modules all-linear \
    --learning_rate 1e-4 --num_train_epochs 3 \
    --max_length 2048 --packing true \
    --output_dir $OUT

# 2. Eval
swift eval --model $OUT/checkpoint-xxx \
    --eval_backend Native --eval_dataset gsm8k mmlu --infer_backend vllm

# 3. Merge
swift export --model $MODEL --adapters $OUT/checkpoint-xxx \
    --merge_lora true --output_dir $OUT-merged

# 4. 部署
swift deploy --model $OUT-merged --infer_backend vllm \
    --served_model_name mvp1 --port 8000
```

## 2. MVP-2：领域专家模型

![[16. 端到端最小可行路线（MVP） - 2. MVP-2：领域专家模型 - 图 02.excalidraw|800]]

```bash
# 1. CPT 领域语料
swift sft --model Qwen/Qwen3-8B-Base \
    --dataset domain_corpus.jsonl \
    --use_chat_template false \
    --train_type lora --learning_rate 5e-5 \
    --num_train_epochs 1 --output_dir output/cpt

# 2. SFT 领域指令（从 CPT base 继续）
swift sft --model output/cpt-merged \
    --dataset domain_sft.jsonl \
    --train_type lora --learning_rate 1e-4 \
    --num_train_epochs 3 --output_dir output/sft

# 3. DPO 风格对齐
swift rlhf --rlhf_type dpo --model output/sft-merged \
    --dataset domain_dpo.jsonl \
    --beta 0.1 --learning_rate 5e-6 --output_dir output/dpo

# 4. AWQ 量化 + 部署
swift export --model output/dpo-merged \
    --quant_method awq --quant_bits 4 \
    --dataset calib.jsonl --output_dir output/awq
swift deploy --model output/awq --infer_backend vllm
```

## 3. MVP-3：推理增强模型（数学/代码）

![[16. 端到端最小可行路线（MVP） - 3. MVP-3：推理增强模型（数学 代码） - 图 03.excalidraw|800]]

```bash
# 1. 高质量 SFT（CoT 数据）
swift sft --model Qwen/Qwen3-8B \
    --dataset AI-MO/NuminaMath-CoT \
    --train_type lora --learning_rate 1e-4 \
    --output_dir output/sft

# 2. GRPO 推理提升（DAPO 优化）
NPROC_PER_NODE=8 swift rlhf \
    --rlhf_type grpo --model output/sft-merged \
    --dataset AI-MO/NuminaMath-CoT \
    --reward_funcs accuracy format \
    --use_vllm true --vllm_mode colocate \
    --num_generations 16 --max_completion_length 4096 \
    --epsilon 0.2 --epsilon_high 0.28 \
    --dynamic_sample true --overlong_filter true \
    --beta 0.04 --learning_rate 1e-6 \
    --output_dir output/grpo

# 3. 评测 GSM8K / MATH 启动 vLLM 部署
swift eval --model output/grpo --eval_dataset gsm8k math
swift deploy --model output/grpo --infer_backend vllm --merge_lora true
```

## 4. MVP-4：RAG 系统

![[16. 端到端最小可行路线（MVP） - 4. MVP-4：RAG 系统 - 图 04.excalidraw|800]]

```bash
# 1. Embedding 微调
swift sft --task_type embedding --model Qwen/Qwen3-Embedding-0.6B \
    --dataset emb.jsonl --hard_negatives 7 --temperature 0.05

# 2. Reranker 微调
swift sft --task_type reranker --model Qwen/Qwen3-Reranker-0.6B \
    --dataset rerank.jsonl

# 3. Generator SFT（使用检索增强后的 prompt）
swift sft --model Qwen/Qwen3-8B --dataset rag_sft.jsonl \
    --train_type lora --max_length 8192
```

## 5. MVP-5：多模态业务模型

```bash
# 多模态 SFT（冻结 ViT）
swift sft --model Qwen/Qwen2-VL-7B-Instruct \
    --dataset my_vl.jsonl \
    --train_type lora --target_modules all-linear \
    --freeze_vit true --aligner_lr 1e-5 \
    --learning_rate 1e-4 --max_length 8192 \
    --output_dir output/vl

# 多模态评测与部署
swift eval --model output/vl-merged --eval_backend VLMEvalKit \
    --eval_dataset MME MMBench

swift deploy --model output/vl-merged --infer_backend vllm \
    --merge_lora true --served_model_name mvp5-vl
```

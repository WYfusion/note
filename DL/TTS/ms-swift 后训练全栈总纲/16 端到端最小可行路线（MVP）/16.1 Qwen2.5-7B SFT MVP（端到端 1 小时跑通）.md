## 0. 定位

> 一句话：1×A100 80G + 1 小时 + 公开数据，跑通 SFT 完整链路：数据 → 训练 → 评测 → 部署 → 调用。

## 1. 环境准备

```bash
# Python 3.10 + CUDA 12.x
conda create -n msswift python=3.10 -y && conda activate msswift
pip install ms-swift evalscope vllm openai
# 验证
swift --help
```

## 2. 数据准备（5 分钟）

```bash
# 用 ms-swift 内置的 alpaca 子集（10k 条）
mkdir -p data
python - <<'PY'
from datasets import load_dataset
import json, random
random.seed(42)
ds = load_dataset('tatsu-lab/alpaca', split='train').shuffle(seed=42).select(range(10000))
with open('data/alpaca-10k.jsonl', 'w') as f:
    for r in ds:
        msg = [
            {'role': 'user',
             'content': r['instruction'] + ('\n' + r['input'] if r['input'] else '')},
            {'role': 'assistant', 'content': r['output']},
        ]
        f.write(json.dumps({'messages': msg}, ensure_ascii=False) + '\n')
print('done')
PY
wc -l data/alpaca-10k.jsonl
```

## 3. 小样本冒烟（5 分钟）

```bash
# 200 条 + 1 epoch，先确认链路
head -n 200 data/alpaca-10k.jsonl > data/smoke.jsonl
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset data/smoke.jsonl \
    --train_type lora --lora_rank 8 \
    --target_modules q_proj v_proj \
    --num_train_epochs 1 \
    --max_length 2048 \
    --learning_rate 1e-4 \
    --output_dir out/smoke
```

冒烟通过 = loss 单调下降，没有 NaN。

## 4. 正式 SFT（约 40 分钟）

```bash
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset data/alpaca-10k.jsonl \
    --train_type lora --lora_rank 16 \
    --target_modules all-linear \
    --num_train_epochs 1 \
    --max_length 2048 \
    --learning_rate 1e-4 \
    --warmup_ratio 0.05 \
    --per_device_train_batch_size 4 \
    --gradient_accumulation_steps 8 \
    --gradient_checkpointing true \
    --bf16 true \
    --logging_steps 5 \
    --save_steps 200 \
    --save_total_limit 3 \
    --output_dir out/qwen-sft-mvp \
    --report_to swanlab
```

## 5. 评测（5 分钟）

```bash
swift eval \
    --model out/qwen-sft-mvp \
    --eval_dataset gsm8k mmlu \
    --eval_limit 200 \
    --infer_backend vllm \
    --output_dir out/qwen-sft-mvp/eval
cat out/qwen-sft-mvp/eval/summary.json
```

基线对比（Qwen2.5-7B-Instruct 原始）：差距应在 ±2%。SFT 主要在风格 / 业务对齐，不在通用 benchmark 上做大跃升。

## 6. 部署 + 调用（5 分钟）

```bash
# 部署
swift deploy \
    --adapters out/qwen-sft-mvp \
    --infer_backend vllm \
    --port 8000 \
    --served_model_name qwen-mvp &

sleep 30 && python - <<'PY'
from openai import OpenAI
c = OpenAI(base_url='http://localhost:8000/v1', api_key='x')
for q in ['你好', '1+1=', '写一首关于春天的诗']:
    r = c.chat.completions.create(
        model='qwen-mvp',
        messages=[{'role':'user','content':q}],
        temperature=0.7, max_tokens=256,
    )
    print('Q:', q); print('A:', r.choices[0].message.content); print('---')
PY
```

## 0. 定位

> 一句话：通过 `dataset_info.json` **声明式注册**自定义数据集，让 `--dataset my-data` 像内置数据集一样使用。

---

## 1. dataset_info.json 字段表

|字段|含义|示例|`dataset_name`|注册名（CLI 用）|`my-finance-sft`|
|---|---|---|---|---|---|
|`hf_dataset_id` / `ms_dataset_id`|HF/MS Hub id|`tatsu-lab/alpaca`|`dataset_path`|本地路径（jsonl/csv/parquet）|`/data/finance.jsonl`|
|`subsets`|子集列表|`['default']`|`split`|切分键|`['train','validation']`|
|`columns`|**字段映射**到 ms-swift 标准|`{"prompt":"query","answer":"response"}`|`tags`|分类（chat/text-generation/...）|`['chat','zh']`|

## 2. 一个最小可用的注册文件

```json
[
  {
    "dataset_name": "my-finance-sft",
    "dataset_path": "/data/finance/sft.jsonl",
    "columns": {
      "question": "query",
      "answer":   "response"
    },
    "tags": ["chat", "finance", "zh"]
  },
  {
    "dataset_name": "my-finance-dpo",
    "dataset_path": "/data/finance/dpo.jsonl",
    "columns": {
      "prompt":     "query",
      "chosen":     "response",
      "rejected":   "rejected_response"
    },
    "tags": ["rlhf", "dpo"]
  }
]
```

```bash
# 通过 --custom_dataset_info 指向上面的文件
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --custom_dataset_info /data/finance/dataset_info.json \
    --dataset my-finance-sft#5000 \
    --train_type lora
```

## 3. 高级：preprocess 函数自定义

当原始字段无法直接映射时，可以注册自定义 preprocess：

```python
# my_pkg/preprocess.py
from swift.llm import register_dataset

@register_dataset('my-mixed-task')
def preprocess_mixed(row):
    """把任意自定义字段转成 ms-swift 标准 messages."""
    role_map = {'u': 'user', 'a': 'assistant'}
    msgs = []
    for turn in row['dialog']:        # 任意原始字段名
        msgs.append({
            'role':    role_map[turn['who']],
            'content': turn['utt'],
        })
    return {
        'messages': msgs,
        'system':   row.get('persona', ''),
    }
```

```bash
# 训练时通过 --external_plugins 指向你的注册文件
swift sft --external_plugins my_pkg/preprocess.py \
          --dataset my-mixed-task \
          --model Qwen/Qwen2.5-7B-Instruct
```

## 4. 数据子采样、混合与权重

|语法|含义|`name#5000`|取前 5000 行（或随机采样）|
|---|---|---|---|
|`name#0.1`|采样 10%|`name1 name2 name3`|合并多个数据集|
|`--dataset_ratio 1 0.3 2`|混合权重|`name:subset`|指定 subset|

```bash
swift sft \
    --dataset 'AI-ModelScope/alpaca-gpt4-data-zh#10000' \
              'my-finance-sft#3000' \
              'swift/self-cognition' \
    --dataset_ratio 1 1 0.3 \
    --model Qwen/Qwen2.5-7B-Instruct
```

> [!important]
> 
> **经验**：自我认知数据（self-cognition）按 1-5% 比例混入即可让模型回答"你是谁"时使用你设定的身份，过多会损伤主任务。

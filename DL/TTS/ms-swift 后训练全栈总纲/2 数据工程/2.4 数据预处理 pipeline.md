## 0. 定位

> 一句话：从原始 jsonl 到训练 batch 的**六步流水线**：load → standardize → encode → filter → pack → collate。

## 2. 每步关键参数

|步骤|关键参数|作用|
|---|---|---|
|① load|`--dataset` / `--custom_dataset_info`|来源|
|② standardize|preprocess 函数|对齐 schema|
|③ encode|`--max_length` / `--template`|token 化、loss mask|
|④ filter|`--truncation_strategy`|delete / left / right|
|⑤ pack|`--packing` / `--packing_length`|提升 GPU 利用率|
|⑥ collate|`--padding_side` / `--pad_to_multiple_of`|对齐到 8/16 提升 TC|
|性能|`--num_proc` / `--streaming` / `--lazy_tokenize`|并行 / 流式 / 延迟|

## 3. 完整示例：从 jsonl 到 DataLoader

```python
from datasets import load_dataset
from swift.llm import (
    get_template, get_model_tokenizer, EncodePreprocessor,
    DataCollatorWithPadding, packing_collator,
)

# ① load + ② standardize（messages 格式直接 load）
raw = load_dataset('json', data_files='data.jsonl', split='train')

# ③ encode + ④ filter（max_length 截断；超长丢弃）
_, tok = get_model_tokenizer('Qwen/Qwen2.5-7B-Instruct', load_model=False)
template = get_template('qwen', tok, max_length=4096,
                       truncation_strategy='delete')
encoded = raw.map(
    EncodePreprocessor(template=template),
    remove_columns=raw.column_names, num_proc=8,
)
encoded = encoded.filter(lambda x: x['input_ids'] is not None, num_proc=4)

# ⑤ pack（可选）
from swift.llm import packing_dataset
packed = packing_dataset(encoded, max_length=4096, num_proc=8)

# ⑥ collate
collator = packing_collator(tok)

# 接入标准 DataLoader
from torch.utils.data import DataLoader
loader = DataLoader(packed, batch_size=4, shuffle=True, collate_fn=collator)
batch = next(iter(loader))
print(batch['input_ids'].shape, batch['labels'].shape)
```

## 4. lazy_tokenize vs streaming

|模式|触发参数|适用|**全量预 encode**|默认|数据 < 100GB，可缓存|
|---|---|---|---|---|---|
|**lazy_tokenize**|`--lazy_tokenize true`|encode 在 DataLoader 内做，不缓存|**streaming**|`--streaming true`|数据 ≥ TB 级，不能落盘|

## 5. truncation_strategy 三种行为

```python
# delete: 超长样本丢弃（推荐 SFT，质量优先）
# left:   左截断，保留尾部（推荐对话最后一轮）
# right:  右截断，保留头部（少用）

# 命令行：
# --truncation_strategy delete  # 或 left / right
```

> [!important]
> 
> - **SFT**：`delete`，避免 loss 监督到被切掉的回答。
> 
> - **CPT**：`right` 或 packing，把所有 token 都用上。
> 
> - **多轮对话**：`left`，确保最新一轮 assistant 完整保留。

## 6. 性能调参速查

|瓶颈|调参|
|---|---|
|磁盘占满|`--lazy_tokenize true` 或 `--streaming true`|
|OOM|降 `--max_length`、用 `--gradient_checkpointing`|

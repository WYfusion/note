## 0. 定位

> 一句话：ms-swift 支持 **10+ 种任务的数据 schema**，每种任务都有最小必填字段集合，理解 schema 即理解 loss 与 forward。

---

## 1. 任务 ↔ 必填字段速查

|任务|关键字段|对应训练命令|
|---|---|---|
|SFT（监督微调）|`messages`（assistant 为 label）|`swift sft`|
|DPO/SimPO/CPO/ORPO|`messages` • `chosen`/`rejected`|`swift rlhf --rlhf_type dpo\|simpo\|...`|
|PPO|`messages`（仅 prompt）|`swift rlhf --rlhf_type ppo`|
|Embedding|`anchor` • `positive` (+ `negative`)|`swift sft --task_type embedding`|
|分类|`messages` • `label`(int)|`swift sft --task_type seq_cls`|
|CPT（继续预训练）|`response`（纯文本）|`swift pt`|
|RM（奖励模型）|`chosen` • `rejected`|`swift rlhf --rlhf_type rm`|
|KTO|`messages` • `label`(bool)|`swift rlhf --rlhf_type kto`|
|GRPO|`messages` • `solution`/可验证字段|`swift rlhf --rlhf_type grpo`|
|Reranker|`query` • `positive` • `negative`|`swift sft --task_type reranker`|
|多模态 SFT|`messages` • `images`/`videos`/`audios`|`swift sft`|

## 2. 三组关键样本

```jsx
# === RM / DPO（pairwise）===
{"messages":[{"role":"user","content":"如何看待开源大模型?"}],
 "chosen":   [{"role":"assistant","content":"开源大模型推动了AI的民主化..."}],
 "rejected": [{"role":"assistant","content":"开源就是免费,没什么好说的."}]}

# === KTO（pointwise）===
{"messages":[
  {"role":"user","content":"今天股市怎么样?"},
  {"role":"assistant","content":"我不能预测股市..."}
], "label": true}   // 期望/不期望（true=desirable）

# === GRPO（可验证）===
{"messages":[{"role":"user","content":"求 1+2+3+...+100"}],
 "solution": "5050"}                // accuracy reward 与之对比
```

## 3. Embedding / Reranker 专用 schema

```jsx
# embedding：InfoNCE 三元组（hard negative 可选多个）
{"anchor":   "如何配置 ms-swift?",
 "positive": "参考 ms-swift 官方快速入门文档...",
 "negative": ["vLLM 是一个推理引擎", "transformers 提供 model loader"]}

# reranker：query/doc 对，二分类
{"query": "LoRA 原理",
 "positive": "LoRA 通过低秩矩阵 BA 注入到权重...",
 "negative": "AWQ 是一种激活感知的量化方法"}
```

## 4. 多模态 SFT schema

```jsx
# 图像：messages 内用 <image> 占位，图像路径放 images 字段
{"messages":[
  {"role":"user","content":"<image>这是什么动物?"},
  {"role":"assistant","content":"图中是一只布偶猫,体型大、毛色为重点色..."}
], "images": ["/data/imgs/cat_001.jpg"]}

# 视频：videos 字段
{"messages":[
  {"role":"user","content":"<video>视频里发生了什么?"},
  {"role":"assistant","content":"..."}
], "videos": ["/data/videos/clip_42.mp4"]}
```

## 5. schema 选择决策

![[2.2 任务类型 schema 全景 - 5. schema 选择决策.excalidraw|800]]

> [!important]
> 
> - **DPO 数据 prompt 必须一致**：`chosen` 与 `rejected` 共享同一 `messages` 上下文。
> 
> - **KTO label 是 bool**：true=desirable，false=undesirable，与 DPO 不可互转。
> 
> - **GRPO 数据需含可验证字段**：accuracy reward 才有意义，开放式生成不要用。

## 0. 定位

> 一句话：ReAct、ToolBench、AgentFlan 是三个常用的 Agent 训练数据集，ms-swift 提供一键加载。

---

## 1. 三个数据集对比

|数据集|规模|工具|风格|质量|
|---|---|---|---|---|
|**AgentFlan**|~24k 轨迹|多场景|OpenAI|高|
|**Glaive-FC**|~50k|合成|OpenAI|中|

## 2. 一键训：ToolBench

```bash
swift sft \
    --model Qwen/Qwen2.5-7B-Instruct \
    --dataset toolbench \                # ms-swift 内置加载器
    --train_type lora \
    --target_modules all-linear \
    --agent_template react \
    --loss_scale react \
    --max_length 8192 \
    --output_dir out/qwen-toolbench
```

## 3. ReAct 轨迹示例

```jsx
user: I want to find the cheapest flight from NYC to LA on 2026-05-01.

assistant:
Thought: I need to search flights

Observation: [{"airline":"Delta","price":150,"departure":"08:00"}, ...]

Thought: I have flights, now find the cheapest
Final Answer: The cheapest flight is Delta at $150 departing 08:00.
```

## 5. 数据质量检查

```python
import json
from collections import Counter

def inspect_agent_data(path):
    stats = Counter()
    with open(path) as f:
        for line in f:
            sample = json.loads(line)
            n_turns = len(sample['messages'])
            stats['samples'] += 1
            stats['avg_turns'] += n_turns
            for msg in sample['messages']:
                if msg['role'] == 'assistant' and msg.get('tool_calls'):
                    stats['tool_calls'] += 1
            stats['has_final'] += any(
                m['role']=='assistant' and m.get('content') for m in sample['messages'])
    n = stats.pop('samples')
    print(f'Total samples: {n}')
    print(f'Avg turns:     {stats["avg_turns"]/n:.1f}')
    print(f'Tool calls/sample: {stats["tool_calls"]/n:.2f}')
    print(f'Has final answer: {stats["has_final"]/n*100:.1f}%')

inspect_agent_data('toolbench.jsonl')
```

## 6. 常见质量问题

|问题|表现|应对|
|---|---|---|
|tool_call 后无 observation|上下文断裂|过滤该样本|
|超长轨迹不收敛|10+ 轮仍未 Final|裁剪或丢弃|

> - **首选 AgentFlan**：质量高，规模适中，与 Qwen 佃谐。

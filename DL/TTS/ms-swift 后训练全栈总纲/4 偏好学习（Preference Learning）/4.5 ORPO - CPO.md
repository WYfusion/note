## 0. 定位

> 一句话：ORPO **跳过了 SFT 阶段**，在一个训练过程中同时做 SFT + odds-ratio 偏好学习；CPO 是 DPO 的一种加强变体，添加了负例损失。

---

## 1. 三者起源对比

|方法|是否需 SFT 阶段|是否需 ref|损失核心|DPO|是|是|logp 差 + ref logp 差|
|---|---|---|---|---|---|---|---|
|**ORPO**|不需|不需|SFT loss + odds-ratio 损失|**CPO**|是|不需|SFT loss(chosen) + DPO-like项|

## 2. ORPO 损失

$$\mathcal{L}_{\text{ORPO}} = \mathcal{L}_{\text{SFT}}(y_w) + \lambda \cdot \mathcal{L}_{\text{OR}}$$

其中 odds-ratio 项：

$$\mathcal{L}_{\text{OR}} = -\log \sigma\Big( \log \dfrac{\mathrm{odds}_\theta(y_w \mid x)}{\mathrm{odds}_\theta(y_l \mid x)} \Big),\quad \mathrm{odds}_\theta(y \mid x) = \dfrac{\pi_\theta(y \mid x)}{1 - \pi_\theta(y \mid x)}$$

直观理解：一边拟合 chosen，一边拉开 chosen 与 rejected 的 odds。

## 3. CPO 损失

$$\mathcal{L}_{\text{CPO}} = \mathcal{L}_{\text{SFT}}(y_w) + \lambda \Big[ -\log \sigma\big( \beta \log \pi_\theta(y_w \mid x) - \beta \log \pi_\theta(y_l \mid x) \big) \Big]$$

CPO = SFT 于 chosen + 不含 ref 的 DPO。

## 6. 手写 ORPO Loss

```python
import torch, torch.nn.functional as F

def orpo_loss(policy_chosen_logps, policy_rejected_logps,
              chosen_labels, rejected_labels,
              policy_chosen_logits,                      # 用于 SFT loss
              orpo_beta=0.1):
    # SFT loss（仅 chosen）
    sft_loss = F.cross_entropy(
        policy_chosen_logits[..., :-1, :].reshape(-1, policy_chosen_logits.size(-1)),
        chosen_labels[..., 1:].reshape(-1),
        ignore_index=-100, reduction='mean')
    
    # odds: p / (1 - p)
    log_odds_chosen   = policy_chosen_logps   - torch.log1p(-policy_chosen_logps.exp().clamp(max=0.999))
    log_odds_rejected = policy_rejected_logps - torch.log1p(-policy_rejected_logps.exp().clamp(max=0.999))
    
    or_loss = -F.logsigmoid(log_odds_chosen - log_odds_rejected).mean()
    
    return sft_loss + orpo_beta * or_loss
```

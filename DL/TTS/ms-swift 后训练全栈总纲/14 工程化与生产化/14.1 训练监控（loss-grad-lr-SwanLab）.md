## 4. 自定义 callback 记录额外指标

```python
import torch
from transformers import TrainerCallback

class GradLogger(TrainerCallback):
    def on_step_end(self, args, state, control, model=None, **kw):
        if state.global_step % args.logging_steps != 0:
            return
        total = 0.0
        for p in model.parameters():
            if p.grad is not None:
                total += p.grad.detach().pow(2).sum().item()
        gn = total ** 0.5
        # 写到 logger
        if state.is_world_process_zero:
            state.log_history.append({
                'step': state.global_step,
                'custom/grad_l2': gn,
            })
```

## 5. 健康检查脚本（异常自动停训）

```python
import math
from transformers import TrainerCallback, TrainerControl

class HealthCheck(TrainerCallback):
    def __init__(self, max_grad_norm=100, loss_window=20):
        self.max_grad_norm = max_grad_norm
        self.loss_history = []
        self.window = loss_window

    def on_log(self, args, state, control: TrainerControl, logs=None, **kw):
        if not logs:
            return
        loss = logs.get('loss')
        if loss is not None:
            if math.isnan(loss) or math.isinf(loss):
                print(f'[HealthCheck] step={state.global_step} loss={loss}, 停训')
                control.should_training_stop = True
                return
            self.loss_history.append(loss)
            self.loss_history = self.loss_history[-self.window:]
        gn = logs.get('grad_norm')
        if gn is not None and gn > self.max_grad_norm:
            print(f'[HealthCheck] grad_norm={gn:.2f} 超阈值，停训')
            control.should_training_stop = True
```

## 7. 报警 SLO

|规则|动作|loss 出现 NaN/Inf|立即停训，保 ckpt|
|---|---|---|---|
|grad_norm > 100 持续 5 步|停训，检查 lr|tokens/s 下降 >30%|检查 IO / 数据集|
|GPU SM 占用 < 50%|检查 batch_size / dataloader_workers|显存线性上涨|泄漏，重启 + 复现|

> [!important]
> 
> - **不要等到训练结束才看曲线**：异常应在 100 步内发现。
> 
> - **多机训练只看 rank 0 不够**：要看每个 rank 的 grad / loss 是否一致，否则可能 NCCL 卡死。
> 
> - **logging_steps 太小**：tensorboard / swanlab 写入压力大，可能拖慢训练。

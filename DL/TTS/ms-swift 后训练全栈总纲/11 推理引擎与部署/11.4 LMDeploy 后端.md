## 0. 定位

> 一句话：LMDeploy 是上海 AI 实验室开发的推理引擎，核心是 **TurboMind** 内核，中文优化与 W4A16 量化是高亮。

---

## 1. 核心特点

|特点|说明|
|---|---|
|**W4A16 量化**|友好内置 AWQ，推理 1.8x 提速|
|**INT8 KV cache**|节约显存|

## 2. 启动服务

```bash
swift deploy \
    --model out/qwen-sft \
    --infer_backend lmdeploy \
    --port 8000 \
    --tensor_parallel_size 1 \
    --quant_policy 4 \           # 4=INT4 W4A16, 8=INT8 KV, 0=不量化
    --cache_max_entry_count 0.8
```

## 3. W4A16 量化

LMDeploy 原生支持 AWQ 质量量化，代表性推理加速如下：

```jsx
Qwen2.5-7B / 1×A100 80G
              fp16        AWQ-W4A16
  TTFT:       80ms        45ms
  TPOT:       30ms        18ms
  吞吐:        2500         4500 tokens/s
  显存:        14GB         5GB
```

## 4. 生成 W4A16 权重

```bash
swift export \
    --model out/qwen-sft \
    --quant_method awq \
    --quant_bits 4 \
    --quant_n_samples 256 \
    --output_dir merged/qwen-7b-awq

swift deploy \
    --model merged/qwen-7b-awq \
    --infer_backend lmdeploy \
    --quant_policy 4
```

## 5. 何时选 LMDeploy

> [!important]
> 
> - **中文为主**：中文 prompt 表现会优于 vLLM。
> 
> - **W4A16 量化极少资源**：资源受限可领取推荐。
> 
> - **InternLM 系列模型**：InternLM/InternVL 原生佃谐。

## 6. 与 vLLM 差异总结

|项|vLLM|LMDeploy|
|---|---|---|
|实现语言|Python + CUDA|C++ TurboMind|
|适用语言|中英混合|中文优先|

## 7. 踩坑提示

> [!important]
> 
> - **模型支持范围 ⚤ vLLM**：部分新模型 LMDeploy 未及时适配。
> 
> - **安装麻烦**：TurboMind 需要与 CUDA 版本佃谐，预编译 wheel 是首选。
> 
> - **W4A16 量化微损**：多数任务有 0.5-1.5% 损失，需以 benchmark 验证。

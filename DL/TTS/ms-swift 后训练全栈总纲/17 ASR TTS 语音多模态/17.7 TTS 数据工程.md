## 0. 定位

> 一句话：TTS 数据 "质量 > 数量"，9 条工程铁律覆盖音色、情感、韵律、对齐、版权。

---

## 1. 九条铁律

|条目|规则|
|---|---|
|音频质量优先于数量|5 小时高质量 > 50 小时杂乱|
|单说话人 TTS|音色稳定、风格一致|
|多说话人 TTS|speaker_id 必须准确|
|情感 TTS|emotion / style 标签可靠|
|韵律数据|停顿、重音、语速、语气标注|
|文本清洗|错别字、非法符号、不可读缩写|
|音频切分|避免过短、过长、尾部静音过多|
|对齐质量|文本与音频严格匹配|
|版权合规|音色、声音肖像、授权明确|

## 2. 录音规范

> [!important]
> 
> **专业棚录推荐参数**：
> 
> - 采样率 24k / 48k
> 
> - 16-bit / 24-bit PCM
> 
> - 静音底噪 < -60 dB
> 
> - 单一说话人，连续 1 周内完成同一批
> 
> - 句间静音 0.3-0.5s，文末 < 0.3s
> 
> - 室内 RT60 < 0.3s

## 3. 音频质量自动筛选

```python
import numpy as np, librosa, soundfile as sf
from pathlib import Path

def quality_check(wav_path):
    y, sr = librosa.load(wav_path, sr=None)
    issues = []
    # 长度
    dur = len(y) / sr
    if dur < 1.0:
        issues.append(f'too_short_{dur:.2f}s')
    if dur > 15.0:
        issues.append(f'too_long_{dur:.2f}s')
    # 静音比例
    silence = (np.abs(y) < 0.005).mean()
    if silence > 0.4:
        issues.append(f'silence_ratio_{silence:.2f}')
    # 削波 / 爆音
    clip = (np.abs(y) > 0.99).sum()
    if clip > 5:
        issues.append(f'clipping_{clip}')
    # 信噪比估计
    rms = np.sqrt((y ** 2).mean())
    if rms < 0.01:
        issues.append('too_quiet')
    # 尾部静音
    tail = y[-int(sr * 0.3):]
    if (np.abs(tail) > 0.05).any():
        issues.append('tail_not_silent')
    return issues

for wav in Path('raw/').rglob('*.wav'):
    iss = quality_check(wav)
    if iss:
        print(wav.name, iss)
```

## 4. 文本-音频对齐校验

```python
# 用 ASR 反向验证 TTS 数据对齐
from funasr import AutoModel
import editdistance

asr = AutoModel(model='paraformer-zh')

def verify_alignment(text: str, wav: str, max_cer: float = 0.10):
    pred = asr.generate(input=wav)[0]['text']
    cer = editdistance.eval(pred, text) / max(len(text), 1)
    return cer, cer < max_cer, pred

# 用法
cer, ok, pred = verify_alignment('欢迎使用 Notion AI', 'tts/0042.wav')
if not ok:
    print(f'对齐异常 CER={cer:.2%} 实际识别: {pred}')
```

## 6. 多说话人 / 情感数据组织

```jsx
data/tts/
  spk001_neutral/
    text_001.txt
    text_001.wav
    text_002.txt
    text_002.wav
  spk001_happy/
    ...
  spk002_neutral/
    ...
  manifest.jsonl   # 每行: {speaker_id, emotion, style, text, audio, language, sr}
```

```python
# 检查每说话人 / 情感时长是否充足
from collections import defaultdict
import json, librosa

durs = defaultdict(float)
for line in open('manifest.jsonl'):
    r = json.loads(line)
    key = (r['speaker_id'], r.get('emotion', 'neutral'))
    durs[key] += librosa.get_duration(filename=r['audio'])

for k, v in sorted(durs.items()):
    flag = '✓' if v >= 600 else '⚠️ 不足'  # 经验：每情感 ≥ 10 分钟
    print(f'{k}: {v/60:.1f} min  {flag}')
```

## 7. 版权与合规

> [!important]
> 
> - **音色克隆必须取得授权**，公众人物 / 知名声音不可未经授权克隆。
> 
> - **数据合同要明确**：训练用途、是否衍生 / 转授权、是否商用。
> 
> - **训练集中含真人录音**，需配套保存录音许可文件。
> 
> - **欧盟 / 国内监管**：声音肖像权与人格权日益受法律保护，发布前进行合规审查。
> 
> - **水印 / 防伪**：商用 TTS 建议在合成音频中嵌入不可闻水印用于追溯。

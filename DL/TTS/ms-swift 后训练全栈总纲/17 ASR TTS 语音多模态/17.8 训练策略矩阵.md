## 2. Speech LLM SFT 模板

```bash
swift sft \
    --model Qwen/Qwen2-Audio-7B-Instruct \
    --dataset speech-instruct.jsonl \
    --train_type lora --lora_rank 16 \
    --target_modules all-linear \
    --freeze_vit true \
    --freeze_aligner false \
    --learning_rate 1e-4 \
    --max_length 4096 \
    --num_train_epochs 3 \
    --gradient_accumulation_steps 16 \
    --gradient_checkpointing true \
    --output_dir out/qwen2-audio-sft
```

关键点：`freeze_vit=true` 冻结 audio encoder，`freeze_aligner=false` 训练投影层。

## 3. Speech DPO

```bash
swift rlhf \
    --rlhf_type dpo \
    --model Qwen/Qwen2-Audio-7B-Instruct \
    --dataset speech-dpo.jsonl \
    --train_type lora --lora_rank 8 \
    --beta 0.1 \
    --learning_rate 5e-6 \
    --max_length 4096 \
    --output_dir out/qwen2-audio-dpo
```

数据：每条 (audio + prompt, chosen, rejected, reason)。常用于：

- 客服礼貌度对齐

- 共情风格对齐

- 安全（不响应违规请求）

## 4. Speech GRPO

```bash
swift rlhf \
    --rlhf_type grpo \
    --model out/qwen2-audio-sft \
    --dataset speech-grpo-math.jsonl \
    --reward_funcs accuracy format \
    --num_generations 8 \
    --temperature 1.0 \
    --max_completion_length 1024 \
    --learning_rate 5e-7 \
    --output_dir out/qwen2-audio-grpo
```

奖励示例（口算 / 音频问答）：

```python
# reward_func: 答案 exact_match
import re

def accuracy_reward(completion: str, ground_truth: str, **kw) -> float:
    m = re.search(r'<answer>(.*?)</answer>', completion)
    if not m:
        return 0.0
    return 1.0 if m.group(1).strip() == ground_truth.strip() else 0.0

def format_reward(completion: str, **kw) -> float:
    return 0.5 if '<answer>' in completion and '</answer>' in completion else 0.0
```

## 6. 数据混合（Speech + Text）

```python
# 经验：Speech LLM SFT 时 Speech : Text = 1 : 1 ~ 3 : 1
# 防止训练后纯文本能力下降。

import random, json

with open('speech-sft.jsonl') as fs, open('text-sft.jsonl') as ft, open('mixed.jsonl', 'w') as fo:
    speech = fs.readlines()
    text = ft.readlines()
    target_ratio = 2  # speech : text
    n_text = len(speech) // target_ratio
    sampled_text = random.sample(text, min(n_text, len(text)))
    mixed = speech + sampled_text
    random.shuffle(mixed)
    fo.writelines(mixed)
print(f'speech={len(speech)} text_used={len(sampled_text)} total={len(mixed)}')
```

## 7. 训练顺序建议

![[17.8 训练策略矩阵 - 7. 训练顺序建议 - 图 02.excalidraw|800]]

> [!important]
> 
> - **不要跳过 SFT 直接 GRPO**：base 模型不知道格式，奖励信号无法传递。
> 
> - **音频长度上限**：超过 max_length 会截断，重要信息丢失。Qwen2-Audio 默认 30 秒。
> 
> - **混合 Speech / Text 数据**很重要，否则纯文本能力会显著退化。

## 0. 定位

> 一句话：ASR 训练数据 9 大工程要点：采样率 / 声道 / 清洗 / 长音频 / 文本规范化 / 热词 / 噪声增强 / 切分 / 评测分层。

---

## 1. 9 大要点速览

|要点|规则|
|---|---|
|统一采样率|常用 16k / 24k / 48k，训练前必须一致或显式重采样|
|统一声道|通常转 mono|
|清洗异常|空音频、截断、爆音、低音量、损坏文件|
|长音频处理|VAD 分段，避免直接喂超长音频|
|文本规范化|数字、日期、英文大小写、标点、口语词、特殊符号|
|热词|人名、地名、产品名、术语单独维词表|
|噪声增强|混响、背景噪、人群、电话、远场麦|
|切分原则|短句优先，保留上下文索引|
|评测集冻结|按场景、口音、噪声、设备分层|

## 2. 采样率与声道统一

```python
import librosa, soundfile as sf
from pathlib import Path

TARGET_SR = 16000   # 大多数 ASR 默认 16k
TARGET_CH = 1

def normalize(src: str, dst: str):
    y, sr = librosa.load(src, sr=None, mono=False)
    # 转 mono
    if y.ndim == 2:
        y = y.mean(axis=0)
    # 重采样
    if sr != TARGET_SR:
        y = librosa.resample(y, orig_sr=sr, target_sr=TARGET_SR)
    # 防爆音
    peak = max(abs(y.max()), abs(y.min()))
    if peak > 0.99:
        y = y * (0.95 / peak)
    sf.write(dst, y, TARGET_SR, subtype='PCM_16')

for wav in Path('raw/').rglob('*.wav'):
    normalize(str(wav), str(Path('norm') / wav.name))
```

## 3. VAD 长音频切分

```python
import torch
from funasr import AutoModel

vad = AutoModel(model='fsmn-vad', model_revision='v2.0.4')
asr = AutoModel(model='paraformer-zh', model_revision='v2.0.4')

def split_and_transcribe(long_audio_path: str):
    segs = vad.generate(input=long_audio_path)[0]['value']  # [(start_ms, end_ms), ...]
    results = []
    for s, e in segs:
        if e - s < 200:  # 过短跳过
            continue
        text = asr.generate(input=long_audio_path, start=s, end=e)[0]['text']
        results.append({'start': s, 'end': e, 'text': text})
    return results
```

```python
from collections import Counter
import re

errors = []  # 上线后人工标注的 (asr_text, gt_text) 列表
hot = Counter()
for a, g in errors:
    g_tokens = re.findall(r'[A-Za-z\u4e00-\u9fa5]+', g)
    a_tokens = set(re.findall(r'[A-Za-z\u4e00-\u9fa5]+', a))
    for t in g_tokens:
        if t not in a_tokens and len(t) >= 2:
            hot[t] += 1
hotword_list = [w for w, c in hot.most_common(200) if c >= 3]
```

## 6. 噪声增强

```python
import numpy as np, librosa, soundfile as sf, random

NOISE_DIR = 'noises/'  # MUSAN / DEMAND
RIR_DIR = 'rirs/'      # 房间冲激响应

def augment(y, sr):
    aug_type = random.choice(['noise', 'rir', 'speed', 'volume', 'codec', 'none'])
    if aug_type == 'noise':
        noise, _ = librosa.load(random.choice(list(Path(NOISE_DIR).glob('*.wav'))), sr=sr)
        snr = random.uniform(5, 20)
        scale = np.linalg.norm(y) / np.linalg.norm(noise) / (10 ** (snr / 20))
        y = y + noise[:len(y)] * scale
    elif aug_type == 'rir':
        rir, _ = librosa.load(random.choice(list(Path(RIR_DIR).glob('*.wav'))), sr=sr)
        y = np.convolve(y, rir)[:len(y)]
    elif aug_type == 'speed':
        y = librosa.effects.time_stretch(y, rate=random.uniform(0.9, 1.1))
    elif aug_type == 'volume':
        y = y * random.uniform(0.5, 1.2)
    elif aug_type == 'codec':
        # G.711 / Opus 等编解码模拟
        pass
    return y
```

## 7. 评测集分层

|分层维度|子集示例|用途|
|---|---|---|
|**口音**|普通话 / 粤语 / 川渝 / 东北|方言鲁棒性|
|**设备**|耳机 / 手机 / 远场麦|设备泛化|
|**领域热词**|含 / 不含术语|热词召回|

> [!important]
> 
> - **不要训练集与评测集来自同一说话人**，会让 WER 虚高。
> 
> - **评测集冻结后不要轻易迭代**，否则前后版本不可比。
> 
> - 长音频测试要走完整 VAD → ASR pipeline，不要直接读切好的短音频。

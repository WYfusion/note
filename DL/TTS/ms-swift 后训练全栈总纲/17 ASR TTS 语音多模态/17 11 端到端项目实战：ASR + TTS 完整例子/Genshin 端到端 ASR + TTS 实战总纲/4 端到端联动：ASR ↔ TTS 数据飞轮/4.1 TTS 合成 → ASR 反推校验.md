## 0. 定位

> 一句话：用 ASR v0 反推 TTS v0 合成的 wav，检查语义一致性，高 CER 样本进入「困难集」。

---

## 1. 闭环脚本

```python
# pipeline/loop.py
import jiwer, json, pandas as pd, pathlib
from typing import Iterable

def revcheck_synth(synth_meta: Iterable[dict], asr_call) -> pd.DataFrame:
    rows = []
    for item in synth_meta:
        hyp = asr_call(item['audio_path'])
        rows.append({
            **item,
            'hyp': hyp,
            'cer': jiwer.cer(item['text'], hyp),
        })
    return pd.DataFrame(rows)

if __name__ == '__main__':
    synth = [json.loads(l) for l in pathlib.Path('syn/meta.jsonl').open()]
    from infer.asr_client import call_asr
    df = revcheck_synth(synth, call_asr)
    df.to_parquet('syn/revcheck.parquet')
    print('hard set:', (df['cer'] > 0.15).sum(), 'samples')
```

## 2. 质量门限

|CER 档|合成质量|处置|
|---|---|---|
|< 5%|优|入验证集|
|5–15%|中|丢弃|
|> 15%|差，有价值|入「困难集」|

## 0. 定位

## 1. 原理概述

> [!important]
> 
> Hard-Synth 是一种**难例合成**数据增强：用 LLM 生成领域内多样性文本，用零样本 TTS 以「难听」风格合成音频，进而训练 ASR。

## 2. 脚本骨架

```python
# augment/hard_synth.py
from openai import OpenAI
from infer.tts_client import synth

client = OpenAI()

REWRITE_PROMPT = (
    '帮我以原神角色口吻改写以下句子，使其含多个连错、浊音、生僻词。\n'
    '原句：{text}\n输出：'
)

def expand(seed_texts):
    out = []
    for s in seed_texts:
        rewrite = client.chat.completions.create(
            model='qwen-max',
            messages=[{'role': 'user', 'content': REWRITE_PROMPT.format(text=s)}],
            temperature=0.8,
        ).choices[0].message.content
        wav_path = synth(role='胡桃', style='excited', speed=1.15, text=rewrite)
        out.append({'text': rewrite, 'audio_path': wav_path, 'source': 'hard-synth'})
    return out
```

## 1. 灰度发布机制

```python
# rollout/canary.py
import hashlib, random

def route(uid: str, canary_pct: float, model_a: str, model_b: str) -> str:
    bucket = int(hashlib.md5(uid.encode()).hexdigest(), 16) % 1000
    return model_b if bucket < canary_pct * 10 else model_a
```

## 3. raw audio 回流与抱怨跨足跡

```python
# rollout/audit.py
import json, time, pathlib

def log_user_call(uid, audio_path, pred, score=None):
    pathlib.Path('audit').mkdir(exist_ok=True)
    with pathlib.Path('audit/raw.jsonl').open('a') as f:
        f.write(json.dumps({
            'ts': int(time.time()),
            'uid': uid,
            'audio_path': audio_path,
            'pred': pred,
            'user_score': score,
        }) + '\n')
```

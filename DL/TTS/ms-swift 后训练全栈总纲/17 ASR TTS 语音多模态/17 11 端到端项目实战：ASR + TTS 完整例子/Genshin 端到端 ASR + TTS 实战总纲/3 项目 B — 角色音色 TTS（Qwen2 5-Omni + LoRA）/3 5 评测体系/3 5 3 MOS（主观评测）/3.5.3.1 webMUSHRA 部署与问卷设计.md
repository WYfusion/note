## 0. 定位

> 一句话：webMUSHRA 是开源的主观评测平台，支持 MUSHRA / MOS / AB / ABX 五类问卷。

---

## 1. 部署

```bash
git clone https://github.com/audiolabs/webMUSHRA.git
cd webMUSHRA && docker build -t webmushra .
docker run -p 8000:80 -v $(pwd)/configs:/var/www/html/configs webmushra
# 访问 http://localhost:8000
```

---

## 2. 问卷配置示例

```yaml
# configs/genshin_tts.yaml
testname: "Genshin TTS 主观评测"
testId: genshin_tts_eval
bufferSize: 2048
stopOnErrors: true
showButtonPreviousPage: true
remoteService: service/write.php

pages:
  - type: generic
    id: intro
    name: 评测说明
    content: "请以手机耳机评测，主观打分为 1\u20135\u5206。"

  - type: mushra
    id: trial1
    name: "临试 1 - 蚹厳丽"
    content: "请评价下面语音与参考之间的相似度。"
    reference: configs/audio/yelan_ref.wav
    createAnchor35: true
    createAnchor70: true
    stimuli:
      C1: configs/audio/yelan_syn1.wav
      C2: configs/audio/yelan_syn2.wav
```

---

## 3. 问卷设计准则

> [!important]
> 
> 1. **anchor35 / anchor70** 是低质量错调项，检查评者不随便打分。 
> 
> 1. **每页不超过 4 个候选**，避免评者耳劳。 
> 
> 1. **总时长 < 30 分钟**，超过后评分随机性增加。

## 4. 结果解析

```python
import pandas as pd, json, glob, pathlib

def parse_results(result_dir: str) -> pd.DataFrame:
    rows = []
    for f in glob.glob(f'{result_dir}/*.csv'):
        df = pd.read_csv(f)
        df['rater_id'] = pathlib.Path(f).stem
        rows.append(df)
    return pd.concat(rows)

df = parse_results('mushra_results/')
print(df.groupby('stimulus')['rating'].agg(['mean', 'std', 'count']))
```

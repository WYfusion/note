## 0. 定位

> 一句话：Mean Opinion Score 是主观金标准，需 N 名评者、随机顺序、主调牙，推荐使用 webMUSHRA。

## 2. 代码骨架

```python
# eval/mos.py
import pandas as pd, json, pathlib, hashlib, random
from dataclasses import dataclass
from scipy import stats

@dataclass
class MosScore:
    mean: float
    ci_lower: float
    ci_upper: float
    n: int


def build_mushra_session(syn_paths, ref_paths, anchor_paths, seed=0):
    random.seed(seed)
    sessions = []
    for s, r, a in zip(syn_paths, ref_paths, anchor_paths):
        items = [
            {'id': hashlib.md5(s.encode()).hexdigest()[:8], 'src': s, 'kind': 'syn'},
            {'id': hashlib.md5(r.encode()).hexdigest()[:8], 'src': r, 'kind': 'ref'},
            {'id': hashlib.md5(a.encode()).hexdigest()[:8], 'src': a, 'kind': 'anchor35'},
        ]
        random.shuffle(items)
        sessions.append({'reference': r, 'items': items})
    return sessions


def aggregate_mos(scores_csv: str) -> MosScore:
    df = pd.read_csv(scores_csv)
    df = df[df['kind'] == 'syn']
    mean = df['score'].mean()
    ci = stats.t.interval(0.95, len(df) - 1,
                           loc=mean, scale=stats.sem(df['score']))
    return MosScore(mean, ci[0], ci[1], len(df))


if __name__ == '__main__':
    res = aggregate_mos('mos_results.csv')
    print(f'MOS = {res.mean:.2f} (95% CI [{res.ci_lower:.2f}, {res.ci_upper:.2f}], n={res.n})')
```

## 3. 评测设计原则

> [!important]
> 
> 1. **评者 >= 10 人**，以达 95% CI 宽度 < 0.5。 
> 
> 1. **随机顺序**，避免顺序偏误。 
> 
> 1. **插入主调牙**（低质量例、高质量例）以椛席评者一致性。 
> 
> 1. **评记与口誓**：告知评者合规、仅用于研究。

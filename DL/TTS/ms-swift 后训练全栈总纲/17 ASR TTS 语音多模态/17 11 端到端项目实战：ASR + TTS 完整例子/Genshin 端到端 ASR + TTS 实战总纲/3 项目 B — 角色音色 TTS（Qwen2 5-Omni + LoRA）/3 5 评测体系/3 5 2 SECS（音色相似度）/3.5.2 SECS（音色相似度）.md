## 0. 定位

> 一句话：以 ECAPA-TDNN 或 WeSpeaker 提取 speaker embedding，合成 wav 与参考 wav 的余弦相似度作为音色保现度。

---

## 1. 公式

设 speaker encoder $f$，参考音频 $x_{ref}$、合成音频 $x_{syn}$：

$$\text{SECS} = \cos\!\big(f(x_{ref}), f(x_{syn})\big) = \frac{f(x_{ref})^\top f(x_{syn})}{\|f(x_{ref})\| \|f(x_{syn})\|}$$

取值范围 [-1, 1]，高活动鲁棒年代的 ECAPA-TDNN 上：同一人不同句，$\text{SECS} \in [0.55, 0.85]$；跨人间 $\in [-0.1, 0.4]$。

---

## 2. 实现

```python
# eval/secs.py
import torch, torchaudio, pathlib, numpy as np, pandas as pd
from typing import List

from speechbrain.inference.speaker import EncoderClassifier

encoder = EncoderClassifier.from_hparams(
    source='speechbrain/spkrec-ecapa-voxceleb',
    run_opts={'device': 'cuda'}, savedir='ckpt/ecapa',
)

@torch.inference_mode()
def embed(wav_path: str) -> torch.Tensor:
    wav, sr = torchaudio.load(wav_path)
    if sr != 16_000:
        wav = torchaudio.functional.resample(wav, sr, 16_000)
    emb = encoder.encode_batch(wav.cuda())  # (1, 1, 192)
    return emb.squeeze().cpu()


def secs(syn_path: str, ref_paths: List[str]) -> float:
    syn_emb = embed(syn_path)
    refs = torch.stack([embed(p) for p in ref_paths])
    sims = torch.nn.functional.cosine_similarity(syn_emb.unsqueeze(0), refs)
    return float(sims.mean())


def batch_secs(syn_dir: str, ref_index: dict) -> pd.DataFrame:
    rows = []
    for syn_p in pathlib.Path(syn_dir).glob('*.wav'):
        speaker = syn_p.stem.split('_')[0]
        ref_paths = ref_index.get(speaker, [])
        if not ref_paths: continue
        rows.append({'syn': str(syn_p), 'speaker': speaker,
                      'secs': secs(str(syn_p), ref_paths)})
    return pd.DataFrame(rows)
```

## 0. 定位

> 一句话：WeSpeaker (ResNet-152) 与 ECAPA-TDNN 是两套主流 speaker encoder。

---

## 1. 对比表

|指标|ECAPA-TDNN|WeSpeaker ResNet-152|
|---|---|---|
|参数量|6.2 M|20 M|
|VoxCeleb1-O EER|0.86%|0.65%|
|RTF (CPU)|0.04|0.12|
|中文表现|中|**优**|
|集成|SpeechBrain 开箱即用|需手动加载|

---

## 2. 选型准则

> [!important]
> 
> **Genshin TTS 场景**：需 95% 以上识别准确率 → **使用 WeSpeaker**。 
> 
> **代码实验**：高 RTF 不是问题 → 使用 ECAPA-TDNN，集成代码更简单。

---

## 3. WeSpeaker 集成代码

```python
# eval/wespeaker_secs.py
import wespeaker, torch, torchaudio

model = wespeaker.load_model('chinese')  # ResNet-152 trained on CN-Celeb
model.set_device('cuda:0')

def embed_wespeaker(wav_path: str) -> torch.Tensor:
    return torch.tensor(model.extract_embedding(wav_path))
```

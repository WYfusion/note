## 2. 评测骨架

```bash
# scripts/eval_tts.sh

# 1. 离线合成
swift infer \
  --model out/genshin-tts-omni/checkpoint-final \
  --infer_backend pt \
  --val_dataset data/genshin_tts_test.jsonl \
  --result_path syn/

# 2. 各项评测
python eval/wer.py    syn/ --ref data/genshin_tts_test.jsonl
python eval/secs.py   syn/ --ref data/host_ref/
python eval/style.py  syn/
python eval/rtf.py    syn/
```

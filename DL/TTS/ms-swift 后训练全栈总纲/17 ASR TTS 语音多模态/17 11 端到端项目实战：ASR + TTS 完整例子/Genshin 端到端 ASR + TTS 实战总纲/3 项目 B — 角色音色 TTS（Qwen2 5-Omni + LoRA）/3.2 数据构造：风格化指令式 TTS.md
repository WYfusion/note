## 0. 定位

> 一句话：构造含 system（角色身份） + user（风格+语速+文本） + assistant（音频 token）三段式指令 TTS 数据。

---

## 1. 三段式 Schema

```json
{
  "messages": [
    {"role": "system", "content": "你是原神角色「胡桃」，按指定风格与语速朗读中文文本。"},
    {"role": "user",   "content": "风格: 兴奋\n语速: 1.1\n文本: 呵呵，这可是往生堂的大生意呢！"},
    {"role": "assistant", "content": "<audio>"}
  ],
  "audios": ["data/clean/tts/hutao/excited_001.wav"]
}
```

---

## 2. 构造脚本

```python
# datagen/build_tts_jsonl.py
import json, pathlib, pandas as pd, random
random.seed(0)

SPEED_MAP = {'slow': 0.85, 'normal': 1.0, 'fast': 1.15}

def build(df: pd.DataFrame, out_path: str, role: str = '胡桃'):
    sys_msg = f'你是原神角色「{role}」，按指定风格与语速朗读中文文本。'
    with pathlib.Path(out_path).open('w') as f:
        for _, row in df.iterrows():
            speed = round(random.uniform(0.85, 1.15), 2)
            user = (
                f'风格: {row["style"]}\n'
                f'语速: {speed}\n'
                f'文本: {row["transcription"]}'
            )
            f.write(json.dumps({
                'messages': [
                    {'role': 'system',    'content': sys_msg},
                    {'role': 'user',      'content': user},
                    {'role': 'assistant', 'content': '<audio>'},
                ],
                'audios': [row['audio_path']],
            }, ensure_ascii=False) + '\n')
```

## 3. 风格标注

```python
# datagen/auto_style.py
import librosa, numpy as np

def heuristic_style(audio_path: str) -> str:
    y, sr = librosa.load(audio_path, sr=24000)
    rms = float(np.sqrt(np.mean(y ** 2)))
    f0 = librosa.yin(y, fmin=80, fmax=500, sr=sr)
    f0_std = float(np.nanstd(f0))
    if rms > 0.06 and f0_std > 50: return 'excited'
    if rms < 0.025: return 'gentle'
    return 'formal'
```

> [!important]
> 
> **误区**：「启发式风格标注可以直接上训练」—— 错。必须补以 100–500 句人工抽检修正，否则模型学到间接信号。

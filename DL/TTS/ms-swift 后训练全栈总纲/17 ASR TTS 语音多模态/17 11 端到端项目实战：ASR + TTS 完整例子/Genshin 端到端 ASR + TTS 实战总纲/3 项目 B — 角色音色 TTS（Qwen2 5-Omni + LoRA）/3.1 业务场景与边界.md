## 2. 边界设计

> [!important]
> 
> 本项目训练产出的 checkpoint **仅用于个人学习与原型验证**，合成音频 metadata 必须包含 `synthetic=True` 水印与 版权声明。

```python
# infer/watermark.py
import hashlib, json, soundfile as sf, numpy as np

def tag_synth(wav: np.ndarray, sr: int, meta: dict, out_path: str):
    meta = {**meta, 'synthetic': True, 'project': 'genshin-research-only'}
    fingerprint = hashlib.sha256(wav.tobytes()).hexdigest()[:16]
    sf.write(out_path, wav, sr,
             format='OGG', subtype='OPUS',
             # 在 Vorbis comment 中寫入合成信息
             # (sf 不直接支持，可补以 mutagen)
            )
    json.dump({**meta, 'fingerprint': fingerprint}, open(out_path + '.json', 'w'))
```

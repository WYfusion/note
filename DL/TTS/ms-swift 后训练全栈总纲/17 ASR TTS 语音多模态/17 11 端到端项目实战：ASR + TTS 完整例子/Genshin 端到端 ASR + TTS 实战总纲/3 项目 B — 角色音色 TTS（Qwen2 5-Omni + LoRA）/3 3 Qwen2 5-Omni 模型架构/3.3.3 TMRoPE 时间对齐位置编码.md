## 0. 定位

> 一句话：TMRoPE 在传统 RoPE 上叠加一个「时间轴」位置以对齐视频帧、音频帧、文本 token。

---

## 1. 从 RoPE 到 TMRoPE

传统 RoPE 在位置 $p$ 上以频率 $\theta_i$ 旋转：

$$\text{RoPE}(x_p)_i = R(p\theta_i) \, x_{p,i}, \quad R(\phi) = \begin{pmatrix} \cos\phi & -\sin\phi \\ \sin\phi & \cos\phi \end{pmatrix}$$

TMRoPE 加入时间 $t$（以帧计）与频率 $\phi_i$：

$$\text{TMRoPE}(x_{p,t})_i = R(p\theta_i + t\phi_i) \, x_{p,t,i}$$

> [!important]
> 
> 直觉：二维位置编码同时代表「在哪个序列位置」与「在哪个时间点」。这让模型可以看出「这个词与那个视频帧同时出现」。

---

## 2. 时间轴分配规则

|输入类型|位置 $p$|时间 $t$|
|---|---|---|
|video frame|token 序号|frame 时间戳（0.5 s/frame）|
|image patch|token 序号|0|

## 3. 参考实现

```python
# model/tmrope.py
"""简化版 TMRoPE 实现。生产代码需参考 transformers Qwen2.5-Omni。"""
import torch

def tmrope(
    x: torch.Tensor,            # (B, L, n_head, head_dim)
    pos: torch.Tensor,          # (B, L)
    time: torch.Tensor,         # (B, L)  单位由调用者决定
    base: float = 10_000.0,
    base_t: float = 100.0,
) -> torch.Tensor:
    B, L, H, D = x.shape
    half = D // 2
    inv_freq = 1.0 / (base ** (torch.arange(0, half, device=x.device).float() / half))
    inv_freq_t = 1.0 / (base_t ** (torch.arange(0, half, device=x.device).float() / half))

    angle = pos[..., None].float() * inv_freq + time[..., None].float() * inv_freq_t
    cos = angle.cos()[:, :, None, :].repeat(1, 1, H, 1)
    sin = angle.sin()[:, :, None, :].repeat(1, 1, H, 1)

    x1, x2 = x[..., :half], x[..., half:]
    rotated = torch.cat([x1 * cos - x2 * sin, x1 * sin + x2 * cos], dim=-1)
    return rotated

if __name__ == '__main__':
    x = torch.randn(2, 8, 4, 64)
    pos = torch.arange(8).expand(2, -1)
    time = torch.zeros(2, 8); time[:, 4:] = 1.0  # 后半在 t=1
    out = tmrope(x, pos, time)
    print(out.shape)
```

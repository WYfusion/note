## 0. 定位

> 一句话：$\alpha / r$ 是 LoRA 增量贡献的变量缩放。

---

## 1. 缩放公式

$$\Delta W = \frac{\alpha}{r} BA$$

体现为：同一个 $r$、$\alpha$ 越大，$\Delta W$ 对原权重的冲击越大。换句话说，$\alpha / r$ 类似于 LoRA 贡献的「学习率系数」。

---

## 2. 经验准则

> [!important]
> 
> 1. $\alpha = 2r$ **是社区默认**（LLaMA-Adapter、Alpaca-LoRA 都采此设）。 
> 
> 1. $\alpha = r$：在领域迁移较小、希望保留原表达能力时使用。 
> 
> 1. $\alpha = 4r$：需强烈变更行为（新语种、新任务）时使用，但容易过拟合。 
> 
> 1. **合并后的权重变化范数** 应 < 原权重的 5%，否则安定性存隐患。

---

## 3. 验证代码

```python
import torch
from peft import PeftModel

def ratio_to_base(model: PeftModel) -> dict:
    out = {}
    for n, m in model.named_modules():
        if hasattr(m, 'base_layer') and hasattr(m, 'lora_A'):
            base = m.base_layer.weight
            delta = (m.lora_B.default.weight @ m.lora_A.default.weight) * (m.scaling.get('default', 1.0))
            out[n] = (delta.norm() / base.norm()).item()
    return out
```

## 0. 定位

> 一句话：LoRA 在原权重 $W$ 上叠加低秩裁决分量 $\Delta W = BA$，仅训练 $A, B$，选型重点是「加哪些层」与「选多大 rank」。

---

## 1. 原理公式

设原始线性变换 $W \in \mathbb{R}^{d \times k}$，LoRA 训练 $A \in \mathbb{R}^{r \times k}, B \in \mathbb{R}^{d \times r}$，前向计算：

$$h = (W + \frac{\alpha}{r} BA) x = Wx + \frac{\alpha}{r} B(Ax)$$

参数量：$|A| + |B| = r(d + k) \ll dk$。以 Qwen2-7B 的 q_proj （4096×4096）为例：$dk \approx 16.7M$，而 LoRA(r=16) 仅 $r(d+k) = 16 \times 8192 \approx 131K$，参数压缩为原的 0.78%。

> [!important]
> 
> init：$A \sim \mathcal{N}(0, \sigma^2), B = 0$，保证初始时 $\Delta W = 0$，不改变原模型输出。

## 3. 参数商用 PEFT 实现

```python
# train/build_lora.py
"""手动构造 LoRA Config。说明各字段选型逻辑。"""
from peft import LoraConfig, TaskType, get_peft_model
from transformers import AutoModelForCausalLM

MODEL_NAME = 'Qwen/Qwen2-Audio-7B-Instruct'

def build_lora_model():
    base = AutoModelForCausalLM.from_pretrained(
        MODEL_NAME, torch_dtype='bfloat16', trust_remote_code=True
    )
    # 冻结 audio encoder
    for name, param in base.named_parameters():
        if 'audio_tower' in name or 'multi_modal_projector' in name:
            param.requires_grad = False

    config = LoraConfig(
        r=16,
        lora_alpha=32,
        target_modules=[
            'q_proj', 'k_proj', 'v_proj', 'o_proj',
            'gate_proj', 'up_proj', 'down_proj',  # all-linear
        ],
        lora_dropout=0.05,
        bias='none',
        task_type=TaskType.CAUSAL_LM,
        modules_to_save=None,           # 不额外保存任何完整模块
        init_lora_weights='gaussian',
    )
    model = get_peft_model(base, config)
    model.print_trainable_parameters()  # 预期 \< 1% 参数可训练
    return model

if __name__ == '__main__':
    m = build_lora_model()
```

## 0. 定位

> 一句话：Qwen2-Audio = Whisper 式音频 encoder + 简轻 projector + Qwen2 LLM，以「输入复合 token」方式接入。

---

## 2. 输入拼接逻辑

```python
# 伪代码，说明 Qwen2-Audio 如何拼 audio token 与 text token
from typing import List
import torch

def build_audio_input(
    audio_features: torch.Tensor,    # (n_audio, frames, 4096)
    audio_token_id: int,             # <|audio_bos|>
    audio_pad_token_id: int,         # <|audio|>
    text_input_ids: torch.Tensor,    # (B, L_text)
) -> torch.Tensor:
    """在 text_input_ids 中找到 <|audio|> 占位位置，替换为该音频的帧数个 token。"""
    out = []
    for b in range(text_input_ids.shape[0]):
        ids = text_input_ids[b].tolist()
        new = []
        audio_idx = 0
        for tok in ids:
            if tok == audio_pad_token_id:
                # 插入该音频的所有 frame token
                new.extend([audio_token_id] * audio_features[audio_idx].shape[0])
                audio_idx += 1
            else:
                new.append(tok)
        out.append(new)
    return torch.tensor(out)
```

## 3. 与 Qwen2.5-Omni 的区别

|维度|Qwen2-Audio|Qwen2.5-Omni|
|---|---|---|
|架构|单头（Encoder-LLM）|双头（Thinker + Talker）|
|适用场景|**ASR / 音频 QA**|TTS / 语音对话|

## 0. 定位

> 一句话：多角色不均衡，要用 macro-F1 而非 accuracy，同时输出混淆矩阵诊断哪些角色被混淆。

---

## 1. 指标定义

对类别 $c$：

$$\text{Precision}_c = \frac{TP_c}{TP_c + FP_c}, \quad \text{Recall}_c = \frac{TP_c}{TP_c + FN_c}$$

$$F1_c = \frac{2 P_c R_c}{P_c + R_c}, \quad \text{macro-F1} = \frac{1}{|C|} \sum_{c \in C} F1_c$$

> [!important]
> 
> macro-F1 平等对待每个角色，小角色误判会在总分上如实反映；micro-F1 仅反映大角色表现；weighted-F1 按频加权。评测角色 ASR 必选 macro-F1。

## 2. 完整实现

```python
# eval/speaker_metrics.py
"""角色识别评测：macro-F1 + 混淆矩阵 + 逆向梳理。"""
from __future__ import annotations

import json
import pathlib
from collections import Counter
from typing import Dict, List, Tuple

import numpy as np
import pandas as pd
from sklearn.metrics import (
    classification_report,
    confusion_matrix,
    f1_score,
)


def compute(
    refs: List[str], hyps: List[str], top_k_confused: int = 5,
) -> Dict[str, object]:
    labels = sorted(set(refs) | set(hyps))
    f1_macro = float(f1_score(refs, hyps, labels=labels, average='macro', zero_division=0))
    f1_weight = float(f1_score(refs, hyps, labels=labels, average='weighted', zero_division=0))

    cm = confusion_matrix(refs, hyps, labels=labels)

    # 提取Top-K 混淆对
    confusions: List[Tuple[str, str, int]] = []
    for i, label in enumerate(labels):
        row = cm[i].copy()
        row[i] = 0  # 去掉对角
        for j, count in enumerate(row):
            if count > 0:
                confusions.append((label, labels[j], int(count)))
    confusions.sort(key=lambda x: x[2], reverse=True)

    return {
        'f1_macro': f1_macro,
        'f1_weighted': f1_weight,
        'labels': labels,
        'confusion_matrix': cm.tolist(),
        'top_confusions': confusions[:top_k_confused],
        'classification_report': classification_report(
            refs, hyps, labels=labels, zero_division=0, output_dict=True,
        ),
    }


def plot_confusion(cm: np.ndarray, labels: List[str], out_path: str):
    import matplotlib.pyplot as plt
    fig, ax = plt.subplots(figsize=(max(8, len(labels) * 0.4),) * 2)
    im = ax.imshow(cm, cmap='Blues')
    ax.set_xticks(range(len(labels))); ax.set_yticks(range(len(labels)))
    ax.set_xticklabels(labels, rotation=90); ax.set_yticklabels(labels)
    ax.set_xlabel('Predicted'); ax.set_ylabel('True')
    fig.colorbar(im, ax=ax, fraction=0.046, pad=0.04)
    plt.tight_layout(); plt.savefig(out_path, dpi=150); plt.close()


if __name__ == '__main__':
    df = pd.read_parquet('preds_with_speaker.parquet')
    metrics = compute(df['speaker'].tolist(), df['pred_speaker'].tolist())
    print('macro-F1:', metrics['f1_macro'])
    print('top-5 混淆:')
    for true, pred, cnt in metrics['top_confusions']:
        print(f'  {true} → {pred}: {cnt}')
    plot_confusion(np.array(metrics['confusion_matrix']),
                   metrics['labels'], 'cm.png')
```

## 3. 阅读混淆矩阵

> [!important]
> 
> 常见混淆三个原因： 
> 
> 1. **音色相似**（同为女声柔和）→ 补课增强学习。 
> 
> 1. **训练量不足**（那个角色只有 5 分钟）→ 补数据。 
> 
> 1. **prompt 不能反映角色**（没要求输出 speaker）→ 修改 prompt 模板。

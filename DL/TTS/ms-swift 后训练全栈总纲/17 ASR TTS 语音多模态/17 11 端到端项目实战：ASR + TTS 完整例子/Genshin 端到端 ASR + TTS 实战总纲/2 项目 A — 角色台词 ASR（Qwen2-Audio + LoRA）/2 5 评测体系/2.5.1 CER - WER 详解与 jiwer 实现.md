## 0. 定位

> 一句话：实现中、日、英三语种的字错率（CER）与词错率（WER）计算，以及中文需要的文本预处理。

---

## 1. 公式与直觉

设参考序列 $r = (r_1, \ldots, r_N)$、假设序列 $h = (h_1, \ldots, h_M)$，最小编辑距离为：

$$ED(r, h) = S + D + I$$

其中 $S$ 是替换数、$D$ 是删除数、$I$ 是插入数。错误率：

$$\text{ER} = \frac{S + D + I}{N}$$

中文以「字」为粒度计算 → CER，英文以「词」为粒度 → WER。

> [!important]
> 
> ED 使用 Wagner-Fischer 动态规划，复杂度 $O(NM)$，空间 $O(\min(N,M))$。在 jiwer 中以 RapidFuzz C++ 实现，1万序列对几毫秒内完成。

---

## 2. 完整实现

```python
# eval/cer_wer.py
"""CER / WER 计算器，含全调文本归一。  

设计要点：
1. 全角·半角换、去标点、去多余空格。
2. 中文按字粒度、英文按词粒度。
3. 详细输出 S/D/I/N 细节，供误误调试。
"""
from __future__ import annotations

import re
import unicodedata
from dataclasses import dataclass
from typing import List, Tuple

import jiwer

_PUNCT_RE = re.compile(
    r'[\s\u3000\u00b7\.,!\?;:\-_\*\(\)\[\]\{\}、。！？；：“”‘’《》（）【】]'
)


def normalize_text(text: str, lang: str = 'zh') -> str:
    text = unicodedata.normalize('NFKC', text or '').strip()
    text = _PUNCT_RE.sub('', text)
    if lang == 'en':
        text = text.lower()
    return text


def _to_chars(text: str) -> List[str]:
    return [ch for ch in text if not ch.isspace()]


@dataclass
class ErrorBreakdown:
    n_ref: int
    n_sub: int
    n_del: int
    n_ins: int

    @property
    def cer(self) -> float:
        return (self.n_sub + self.n_del + self.n_ins) / max(self.n_ref, 1)


def per_sample_cer(ref: str, hyp: str, lang: str = 'zh') -> ErrorBreakdown:
    ref_n, hyp_n = normalize_text(ref, lang), normalize_text(hyp, lang)
    if lang == 'en':
        ref_seq = ref_n.split()
        hyp_seq = hyp_n.split()
    else:
        ref_seq = _to_chars(ref_n)
        hyp_seq = _to_chars(hyp_n)

    measures = jiwer.process_words(' '.join(ref_seq), ' '.join(hyp_seq)) \
        if lang == 'en' else \
        jiwer.process_characters(''.join(ref_seq), ''.join(hyp_seq))
    return ErrorBreakdown(
        n_ref=len(ref_seq),
        n_sub=measures.substitutions,
        n_del=measures.deletions,
        n_ins=measures.insertions,
    )


def corpus_cer(refs: List[str], hyps: List[str], lang: str = 'zh') -> float:
    """语料粒度 micro-average CER。"""
    n_ref, n_err = 0, 0
    for r, h in zip(refs, hyps):
        eb = per_sample_cer(r, h, lang=lang)
        n_ref += eb.n_ref
        n_err += eb.n_sub + eb.n_del + eb.n_ins
    return n_err / max(n_ref, 1)


if __name__ == '__main__':
    refs = ['呵呵呵，今天去哪里呢？', '吃饭了吗？']
    hyps = ['哦呵呵今天去哪里',     '吃饭了']
    for r, h in zip(refs, hyps):
        eb = per_sample_cer(r, h, 'zh')
        print(f'cer={eb.cer:.3f} S={eb.n_sub} D={eb.n_del} I={eb.n_ins} N={eb.n_ref}')
    print('corpus_cer:', corpus_cer(refs, hyps, 'zh'))
```

## 3. 误区

> [!important]
> 
> 1. **不去标点** → 「你好，世界」 vs 「你好世界」被计为 2 位错误。 
> 
> 1. **中文用 WER** → 中文无姹件分词，不同分词器会产生不同结果。 
> 
> 1. **不全角·半角换** → 「1234」 vs 「１２３４」被计为错误。

## 0. 定位

> 一句话：ms-swift 多模态训练使用「`messages` + 外部资源」的 jsonl 格式。本页面详解字段语义、反序列化、loss mask 位置判断。

---

## 1. 最小合法样本

```json
{
  "messages": [
    {"role": "user",      "content": "<audio>请转写"},
    {"role": "assistant", "content": "今天天气不错。"}
  ],
  "audios": ["/data/clean/asr/0001.wav"]
}
```

> [!important]
> 
> 三个关键点： 
> 
> 1. `<audio>` 是占位符，训练时会被替换为 `audios[i]` 加载后的 token 序列。 
> 
> 1. `audios` 与 `<audio>` 出现顺序一一对应，多个 `<audio>` 可同时出现。 
> 
> 1. `messages` 中只有 assistant 贡献 loss（默认 mask user）。

## 2. 与 Hugging Face Datasets 互转

```python
# datagen/hf_to_swift.py
import json, pathlib
from datasets import load_dataset

ds = load_dataset(
    'parquet', data_files='data/raw/simon3000/*.parquet',
    split='train', streaming=True,
)

out = pathlib.Path('data/genshin_asr_train.jsonl').open('w')
for i, ex in enumerate(ds):
    if not ex['transcription']:
        continue
    audio_path = ex['audio']['path']
    sample = {
        'messages': [
            {'role': 'user',
             'content': '<audio>请转写这段语音，并告诉我说话人。'},
            {'role': 'assistant',
             'content': f'转写: {ex["transcription"]}\n说话人: {ex["speaker"] or "未知"}'},
        ],
        'audios': [audio_path],
    }
    out.write(json.dumps(sample, ensure_ascii=False) + '\n')
    if i % 5000 == 0:
        print(f'wrote {i}')
out.close()
```

## 3. 多轮多音频 / system message

```python
# datagen/multi_turn.py
sample = {
    'messages': [
        {'role': 'system',
         'content': '你是一位语音助手，能同时转写多段音频。'},
        {'role': 'user',
         'content': '<audio>这个是什么口音？<audio>那这个呢？'},
        {'role': 'assistant',
         'content': '第一个是柔和口音；第二个是兴奋口音。'},
    ],
    'audios': ['a1.wav', 'a2.wav'],
}
```

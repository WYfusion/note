## 0. 定位

> 一句话：把 1.5 产出的 `asr_verified.parquet` 转成 ms-swift `messages + audios` jsonl，并设计多样性指令 prompt。

---

## 1. 多样性 prompt 模板

```python
# datagen/prompts.py
INSTRUCTIONS = [
    '请转写这段语音。',
    '转写并告诉我说话人。',
    '给出转写、角色与情绪，JSON 输出。',
    '给出 JSON，包含 transcription / speaker / emotion / language。',
]

RESPONSE_TPL = (
    '转写: {transcription}\n'
    'JSON: {{"transcription": "{transcription}", '
    '"speaker": "{speaker}", '
    '"emotion": "{emotion}", '
    '"language": "{language}"}}'
)
```

---

## 2. jsonl 生成脚本

```python
# datagen/build_asr_jsonl.py
import json, random, pandas as pd, pathlib
from prompts import INSTRUCTIONS, RESPONSE_TPL

random.seed(0)
df = pd.read_parquet('data/clean/asr_verified.parquet')
out = pathlib.Path('data/genshin_asr_train.jsonl')

with out.open('w', encoding='utf-8') as f:
    for _, row in df.iterrows():
        instr = random.choice(INSTRUCTIONS)
        sample = {
            'messages': [
                {'role': 'user', 'content': f'<audio>{instr}'},
                {'role': 'assistant', 'content': RESPONSE_TPL.format(
                    transcription=row['transcription'],
                    speaker=row['speaker'] or '未知',
                    emotion=row['emotion'] or '平静',
                    language=row['language'],
                )},
            ],
            'audios': [row['audio_path']],
        }
        f.write(json.dumps(sample, ensure_ascii=False) + '\n')
print('written', out)
```

## 3. Loss Mask 考察

> [!important]
> 
> 在 ms-swift 中默认 user 部分不计损失，assistant 部分计损失。若要强调 JSON 字段准确性，可使用 `--loss_scale all` 或自定义 mask。

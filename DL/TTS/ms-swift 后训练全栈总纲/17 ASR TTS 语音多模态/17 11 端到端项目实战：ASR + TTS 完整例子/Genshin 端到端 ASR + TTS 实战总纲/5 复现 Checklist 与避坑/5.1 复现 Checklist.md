> 一句话：从环境到 checkpoint 的逻辑顺序 checklist，错一项训练就崩。

## 1. 环境 checklist

```bash
# scripts/check_env.sh
set -e
python -c 'import torch; assert torch.__version__ >= "2.4", torch.__version__'
python -c 'import torch; assert torch.cuda.is_available()'
python -c 'import flash_attn; print(flash_attn.__version__)'
swift --version | grep -E '3\.(10|11|12)'
nvidia-smi --query-gpu=memory.total --format=csv,noheader
```

---

## 2. 顺序 checklist

|阶段|必验|检验方式|
|---|---|---|
|数据|parquet 读取成功、采样率一致|`scripts/inspect_schema.py`|
|预处理|VAD 后片段时长在 0.6–25 s|`scripts/check_clean.py`|
|反推清洗|asr_verified.parquet 不为空|`du -h data/clean/asr_verified.parquet`|
|烟囱|20 step loss 下降>0.5|`tail -n 5 train.log`|
|全量|train_loss 是否平稳|wandb 面板|
|评测|CER 低于阈值|`eval/eval_asr.py`|
|部署|OpenAI 接口返回 200|`curl http://localhost:8000/v1/chat/completions`|

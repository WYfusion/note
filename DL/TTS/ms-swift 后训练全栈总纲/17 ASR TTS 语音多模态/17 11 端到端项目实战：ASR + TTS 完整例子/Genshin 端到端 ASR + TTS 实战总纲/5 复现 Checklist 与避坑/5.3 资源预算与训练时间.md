## 1. 训练阶段

设训练集总序列 token 为 $T$，单步 token 为 $b$，梯度累积为 $g$，卡数为 $n$，有效 step 数：

$$N_{step} = \frac{T \cdot E}{b \cdot g \cdot n}$$

例：ASR 实验 $T \approx 1.5\times10^7$ token，$b=1$，$g=8$，$n=4$，$E=2$：

$$N_{step} = \frac{1.5 \times 10^7 \times 2}{1 \times 8 \times 4} \approx 9.4 \times 10^5$$

    → 以 A100-80G 预估 1.2 step/s，约 13 小时。

|任务|设备|预估时长|VRAM 峰值|磁盘|
|---|---|---|---|---|
|ASR LoRA|4× A100-80G|~13 h|62 GB|~80 GB|
|TTS LoRA|4× A100-80G|~28 h|74 GB|~140 GB|
|评测|1× A100|~3 h × 2|40 GB|~10 GB|

## 3. 资源估算脚本

```python
# scripts/budget.py
from dataclasses import dataclass

@dataclass
class Budget:
    tokens: int           # train tokens
    micro_bs: int = 1
    grad_accum: int = 8
    n_gpu: int = 4
    epochs: int = 2
    step_per_sec: float = 1.2

    @property
    def steps(self) -> int:
        return int(self.tokens * self.epochs / (self.micro_bs * self.grad_accum * self.n_gpu))

    @property
    def hours(self) -> float:
        return self.steps / self.step_per_sec / 3600

asr = Budget(tokens=1_500_0000)
print(f'ASR: {asr.steps} steps, {asr.hours:.1f} h')
tts = Budget(tokens=4_000_0000, step_per_sec=0.7, epochs=3)
print(f'TTS: {tts.steps} steps, {tts.hours:.1f} h')
```

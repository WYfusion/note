## 0. 定位

> 一句话：解决「谁进训练集、谁进测试集、怎么防角色泄漏」三个问题。

## 1. 筛选三原则

> [!important]
> 
> **原则 1：角色阁下量**：单个角色 < 10 分钟语音不进 TTS（不足以学会音色），但可进 ASR（帮助多说话人鲁棒性）。
> 
> **原则 2：角色隔离**：TTS test 集使用训练集完全未见过的句子（同角色不同句）。ASR test 集可以跨角色。
> 
> **原则 3：语种平衡**：训练集中各语种不低于 10% 及不高于 60%。

## 2. SQL 风格筛选

```python
# scripts/filter.py
import duckdb, pathlib

con = duckdb.connect()
con.execute("""
  CREATE VIEW raw AS SELECT * FROM read_parquet('data/raw/simon3000/*.parquet');
""")

asr_train = con.execute("""
  SELECT * FROM raw
  WHERE transcription IS NOT NULL AND length(transcription) BETWEEN 4 AND 200
    AND speaker IS NOT NULL AND speaker_type = 'playable'
    AND language IN ('zh', 'ja')
    AND duration_s BETWEEN 0.6 AND 25.0
""").fetch_df()
print('ASR 训练侯选:', len(asr_train))

tts_train = con.execute("""
  SELECT * FROM raw
  WHERE speaker = '胡桃' AND language = 'zh'
    AND duration_s BETWEEN 1.0 AND 12.0
    AND transcription IS NOT NULL
""").fetch_df()
print('TTS 胡桃训练侯选:', len(tts_train))
```

## 3. 防角色泄漏的划分函数

```python
from sklearn.model_selection import GroupShuffleSplit
import pandas as pd

def split_by_speaker(df: pd.DataFrame, test_size=0.05, seed=42):
    gss = GroupShuffleSplit(n_splits=1, test_size=test_size, random_state=seed)
    train_idx, test_idx = next(gss.split(df, groups=df['speaker']))
    return df.iloc[train_idx], df.iloc[test_idx]
```

> [!important]
> 
> **误区**：`train_test_split(shuffle=True)` 丢到 ASR 上—— 会让同一角色同时出现在训/测中。一定要用 `GroupShuffleSplit` 按 speaker 分组。

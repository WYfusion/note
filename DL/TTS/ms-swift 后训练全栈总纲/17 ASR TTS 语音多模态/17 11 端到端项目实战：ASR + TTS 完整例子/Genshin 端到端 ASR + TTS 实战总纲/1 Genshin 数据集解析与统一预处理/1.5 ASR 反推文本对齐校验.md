## 0. 定位

> 一句话：用独立 ASR 模型（Paraformer 中文 / Whisper 多语）反推清洗后的 wav，跟原始 transcription 算 CER，高者剔除。

---

## 1. 为什么需要

> [!important]
> 
> Genshin parquet 中人工转写存在少量错字、太多拟声词（「哈哈」「哼」）、以及背景音干扰。如果不过滤，模型会学到错误映射，在评测集上表现倒退。

---

## 2. 反推脚本

```python
# scripts/asr_revcheck.py
import torch, jiwer, pandas as pd
from funasr import AutoModel
from tqdm.auto import tqdm

asr_zh = AutoModel(model='paraformer-zh', vad_model='fsmn-vad', device='cuda:0')

def revcheck(df: pd.DataFrame, lang: str = 'zh') -> pd.DataFrame:
    cers = []
    for _, row in tqdm(df.iterrows(), total=len(df)):
        try:
            hyp = asr_zh.generate(input=row['audio_path'])[0]['text']
        except Exception as e:
            cers.append(1.0); continue
        cers.append(jiwer.cer(row['transcription'], hyp))
    df = df.copy(); df['rev_cer'] = cers
    return df

if __name__ == '__main__':
    df = pd.read_parquet('data/clean/asr_candidates.parquet')
    df = revcheck(df, 'zh')
    keep = df[df['rev_cer'] < 0.05]
    print(f'保留 {len(keep)} / 原 {len(df)}、剔除率 {1-len(keep)/len(df):.2%}')
    keep.to_parquet('data/clean/asr_verified.parquet')
```

## 3. CER 公式

字错率 CER（Character Error Rate）定义为编辑距离除以参考长度：

$$\text{CER} = \frac{S + D + I}{N}$$

- $S$：替换数，$D$：删除数，$I$：插入数，$N$：参考字数。

- 中文按字粒度计算，不使用分词。文本预处理需去除标点与全角/半角差异。

---

## 4. 阈值选择依据

|rev_cer 阈值|保留率|会保留的错误|适用场景|
|---|---|---|---|
|0.02|~ 60%|幾乎无|高质量 TTS|
|0.05|~ 80%|拟声词 / 少量错字|**本项目默认**|
|0.10|~ 92%|能容忍口语化严重词|ASR 鲁棒性增强集|

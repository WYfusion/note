## 0. 定位

> 一句话：解析 simon3000 parquet 的 8 个字段语义，给出同一套表达到双链路训练集的映射。

---

## 1. 原始 Schema

|字段|类型|语义|在 ASR 中用途|在 TTS 中用途|
|---|---|---|---|---|
|`audio`|{path, sampling_rate, array}|原始 wav|输入|label（gold audio token）|
|`transcription`|str|人工转写|label|输入|
|`language`|enum {zh, ja, en, ko, multi}|语种|分语种评测|选取语种子集|
|`speaker`|str (可空)|角色名|分类 label|选取目标角色|
|`speaker_type`|enum (6 类)|npc / playable / unknown / …|过滤 unknown|同|
|`type`|enum (9 类)|combat / story / idle / …|助改善 prompt 多样性|推导风格|
|`inGameFilename`|str|原始资源名|去重键|去重键|

## 2. 读取与初探

```python
# scripts/inspect_schema.py
from datasets import load_dataset
import pandas as pd

ds = load_dataset(
    'parquet',
    data_files='data/raw/simon3000/*.parquet',
    split='train',
    streaming=True,
)

rows = []
for i, ex in enumerate(ds):
    rows.append({
        'language': ex['language'],
        'speaker': ex['speaker'] or '<empty>',
        'speaker_type': ex['speaker_type'],
        'type': ex['type'],
        'has_transcription': bool(ex['transcription']),
        'duration_s': len(ex['audio']['array']) / ex['audio']['sampling_rate'],
    })
    if i >= 50_000:
        break

df = pd.DataFrame(rows)
print('\n语种分布'); print(df['language'].value_counts())
print('\nspeaker 缺失率:', (df['speaker'] == '<empty>').mean())
print('\n平均时长:', df['duration_s'].mean(), 's')
```

## 3. 统一训练 Schema

```python
# schema/sample.py
from dataclasses import dataclass
from typing import Literal, Optional

Language = Literal['zh', 'ja', 'en', 'ko']
Style    = Literal['formal', 'gentle', 'excited']

@dataclass
class UnifiedSample:
    audio_path: str          # 24k wav (TTS) / 16k wav (ASR)
    transcription: str       # 转写 (必给)
    language: Language
    speaker: str             # 角色名 (必给)
    duration_s: float
    style: Optional[Style] = None
    emotion: Optional[str] = None  # e.g. happy/angry/sad
```

## 4. 误区

> [!important]
> 
> **误区**：「直接用 inGameFilename 去重」—— 错。不同语种可能共享同一 inGameFilename，需 `(inGameFilename, language)` 联合去重。

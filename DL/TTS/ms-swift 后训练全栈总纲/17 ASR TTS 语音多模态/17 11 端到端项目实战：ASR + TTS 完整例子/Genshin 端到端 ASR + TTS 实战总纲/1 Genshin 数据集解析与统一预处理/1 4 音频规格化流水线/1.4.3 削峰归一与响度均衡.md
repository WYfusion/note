## 0. 定位

> 一句话：使不同角色录音响度一致（-23 LUFS）、峰值不超过 -3 dBFS，避免训练裂颜。

---

## 1. 指标定义

**峰值 (Peak)**：$\text{peak}(x) = \max_n |x[n]|$，以 dBFS 为单位：

$$\text{peak}_{dB} = 20 \log_{10}\!\big(\text{peak}(x)\big)$$

**RMS 响度**：$\text{rms}(x) = \sqrt{\frac{1}{N}\sum_n x[n]^2}$。

**LUFS**（ITU BS.1770-4）：带频率加权与门阈门控制的响度需要量，与人耳感知更贴近。公式表达为：

$$L_{KG} = -0.691 + 10 \log_{10}\!\Big(\sum_i G_i \cdot \text{mean}_T(z_i^2)\Big)$$

其中 $z_i$ 是经 K-加权滤波后的信号，$G_i$ 是声道加权。

## 2. 完整实现

```python
# audio_ops/loudness.py
"""响度均衡与峰值限制，遵循 EBU R128 / ITU-R BS.1770-4。"""
from __future__ import annotations

import math
from typing import Optional

import numpy as np
import pyloudnorm as pyln
import torch


def _to_numpy(wav: torch.Tensor) -> np.ndarray:
    if wav.dim() == 1:
        return wav.cpu().numpy()
    return wav.mean(0).cpu().numpy() if wav.shape[0] > 1 else wav.squeeze(0).cpu().numpy()


def peak_db(wav: torch.Tensor) -> float:
    arr = _to_numpy(wav)
    return 20.0 * math.log10(max(float(np.max(np.abs(arr))), 1e-9))


def peak_normalize(wav: torch.Tensor, peak_db_target: float = -3.0) -> torch.Tensor:
    """线性增益使峰值达到 peak_db_target。不作压限 / 柔化。"""
    arr = _to_numpy(wav)
    cur = max(float(np.max(np.abs(arr))), 1e-9)
    target_lin = 10 ** (peak_db_target / 20.0)
    gain = target_lin / cur
    return wav * gain


def lufs(wav: torch.Tensor, sr: int) -> float:
    """计算 BS.1770 集成响度（LUFS）。"""
    arr = _to_numpy(wav).astype(np.float32)
    meter = pyln.Meter(sr)  # block-gated meter
    return float(meter.integrated_loudness(arr))


def loudness_normalize(
    wav: torch.Tensor,
    sr: int,
    target_lufs: float = -23.0,
    peak_ceiling_db: float = -1.0,
) -> torch.Tensor:
    """先 LUFS 均衡，再同价限制峰值不超过天花板。"""
    arr = _to_numpy(wav).astype(np.float32)
    meter = pyln.Meter(sr)
    cur_lufs = float(meter.integrated_loudness(arr))
    if not math.isfinite(cur_lufs):
        return wav  # 静音 / 全零
    gain_db = target_lufs - cur_lufs
    out = wav * (10 ** (gain_db / 20.0))

    # 需压限：避免裁剪
    pk_db = peak_db(out)
    if pk_db > peak_ceiling_db:
        out = out * (10 ** ((peak_ceiling_db - pk_db) / 20.0))
    return out


if __name__ == '__main__':
    import torchaudio, sys
    wav, sr = torchaudio.load(sys.argv[1])
    print('peak_in:', peak_db(wav), 'lufs_in:', lufs(wav, sr))
    out = loudness_normalize(wav, sr, target_lufs=-23.0, peak_ceiling_db=-1.0)
    print('peak_out:', peak_db(out), 'lufs_out:', lufs(out, sr))
    torchaudio.save('/tmp/normalized.wav', out, sr)
```

## 3. 选型判断

> [!important]
> 
> **TTS 训练**：使用 LUFS 均衡到 -23 LUFS 、峰值不超过 -1 dBFS。 
> 
> **ASR 训练**：只需峰值归一到 -3 dBFS，保留信息与鲁棒性，不要强行 LUFS。

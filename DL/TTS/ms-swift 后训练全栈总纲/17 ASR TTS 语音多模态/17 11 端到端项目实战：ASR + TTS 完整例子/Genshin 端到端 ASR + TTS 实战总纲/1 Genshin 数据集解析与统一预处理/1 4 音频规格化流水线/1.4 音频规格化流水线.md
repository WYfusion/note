## 2. 采样率选择原则

设原始采样率为 $f_{src}$，目标采样率 $f_{dst}$，以 **Nyquist 采样定理**为依据：

$$f_{dst} \geq 2 f_{\max}\!\left(\text{人耳感知上限}\right) \approx 2 \times 8\,\text{kHz} = 16\,\text{kHz}$$

- ASR 人声信息集中在 <8 kHz，**16 kHz 足够**。

- TTS 需保留高频谐波以获得自然度，**24 kHz 为现代主流**。

|链路|$f_{dst}$|原因|
|---|---|---|
|TTS|24 kHz|Qwen2.5-Omni qwen-tts-tokenizer 采样率|

## 4. 主控脚本

```python
# scripts/normalize_audio.py
"""主控脚本，串联 1.4.1 → 1.4.2 → 1.4.3。"""
from pathlib import Path
import torch, torchaudio, multiprocessing as mp
from typing import Tuple

from audio_ops import resample, vad_split, peak_normalize

TARGET_SR_ASR, TARGET_SR_TTS = 16000, 24000

def process_one(args: Tuple[Path, Path]):
    src, dst_root = args
    wav, sr = torchaudio.load(str(src))
    if wav.shape[0] > 1:
        wav = wav.mean(0, keepdim=True)
    for tag, target_sr in [('asr', TARGET_SR_ASR), ('tts', TARGET_SR_TTS)]:
        out = resample(wav, sr, target_sr)
        out = peak_normalize(out, peak_db=-3.0)
        for i, seg in enumerate(vad_split(out, target_sr)):
            (dst_root / tag / src.stem).mkdir(parents=True, exist_ok=True)
            torchaudio.save(
                str(dst_root / tag / src.stem / f'{i:04d}.wav'),
                seg, target_sr, encoding='PCM_S', bits_per_sample=16,
            )

if __name__ == '__main__':
    src_root = Path('data/raw'); dst_root = Path('data/clean')
    files = list(src_root.rglob('*.wav'))
    with mp.Pool(8) as pool:
        pool.map(process_one, [(p, dst_root) for p in files])
```

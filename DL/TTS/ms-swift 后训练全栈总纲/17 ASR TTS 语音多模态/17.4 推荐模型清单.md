## 1. 模型矩阵

|目标|推荐模型|规模 / 备注|
|---|---|---|
|**端到端全模态**|Qwen3-Omni / All-to-All Speech LLM|支持 speech-in/out|
|**多语种 ASR**|Whisper / SenseVoice / FunASR-Nano 类|Whisper-Large-v3 是默认基线|
|**TTS**|CosyVoice / VITS / FastSpeech / XTTS / ChatTTS 类|CosyVoice 中文音色稳定|
|**说话人识别**|CAM++ / ECAPA-TDNN 类|嵌入向量比对|

## 7. 选型决策建议

> [!important]
> 
> - **中文 + 工业稳定**：FunASR/Paraformer + ms-swift LLM + CosyVoice2。
> 
> - **多语言 + 通用**：Whisper-v3 + ms-swift LLM + XTTS-v2。
> 
> - **共情对话 / 端到端**：Qwen2.5-Omni + ms-swift SFT。
> 
> - **资源受限**：SenseVoice-Small（同时 ASR + 情绪）+ ms-swift LoRA 7B。

> [!important]
> 
> - TTS 商业化必看 **音色版权与肖像授权**，开源模型不等于可商用。
> 
> - 流式场景必看 **流式版本**，CosyVoice 与 Whisper 默认是非流式。

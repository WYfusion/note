## 0. 定位

> 一句话：语音评测分 8 大方向，每个方向有不同指标。**业务完成率** 比单点指标更重要。

---

## 1. 评测维度总表

|方向|指标|
|---|---|
|**VAD**|Precision、Recall、F1、漏切率、误切率|
|**说话人分离**|DER、JER|
|**TTS**|MOS、SMOS、PESQ、STOI、MCD、Speaker Similarity|
|**生产部署**|P50/P95/P99、QPS、RTF、显存、失败率|

## 2. ASR 指标实现

```python
import editdistance, jieba

def cer(ref: str, hyp: str) -> float:
    return editdistance.eval(list(ref), list(hyp)) / max(len(ref), 1)

def wer(ref: str, hyp: str) -> float:
    r, h = ref.split(), hyp.split()
    return editdistance.eval(r, h) / max(len(r), 1)

def cer_zh(ref: str, hyp: str) -> float:
    # 中文按字符
    return cer(ref.replace(' ', ''), hyp.replace(' ', ''))

def hotword_recall(ref: str, hyp: str, hotwords: list) -> float:
    hit = sum(1 for w in hotwords if w in ref and w in hyp)
    in_ref = sum(1 for w in hotwords if w in ref)
    return hit / max(in_ref, 1)

rtf = lambda decode_time, audio_dur: decode_time / audio_dur  # < 1 即实时
```

## 3. TTS 指标

```python
# MOS / SMOS：主观评分（1-5），需众包或专业听众
# PESQ / STOI：客观语音质量（与参考对比）
# MCD：Mel Cepstral Distortion（合成 vs 参考）
# Speaker Similarity：说话人嵌入余弦相似度

import torch
from speechbrain.pretrained import EncoderClassifier

spk_model = EncoderClassifier.from_hparams(source='speechbrain/spkrec-ecapa-voxceleb')

def speaker_similarity(ref_wav: str, syn_wav: str) -> float:
    e1 = spk_model.encode_batch(load(ref_wav))
    e2 = spk_model.encode_batch(load(syn_wav))
    return torch.nn.functional.cosine_similarity(e1, e2).item()

# MCD
import pymcd
def mcd(ref_wav, syn_wav):
    return pymcd.Calculate_MCD(MCD_mode='plain').calculate_mcd(ref_wav, syn_wav)
```

## 4. 语音理解 / 对话评测

```python
# 语音理解：ASR 文本对照 → LLM 输出对照 → 答案 EM / BLEU / 人工
# 语音对话：完成率 / 轮次 / 延迟 / 打断恢复率

from dataclasses import dataclass

@dataclass
class DialogRun:
    completed: bool        # 是否完成核心任务（订票、查询、解答）
    n_turns: int
    e2e_latency_ms: list   # 每轮端到端延迟
    interrupt_recovered: bool
    human_pref: int        # 1-5

def summarize(runs: list) -> dict:
    n = len(runs)
    return {
        'completion_rate': sum(r.completed for r in runs) / n,
        'avg_turns': sum(r.n_turns for r in runs) / n,
        'avg_latency_ms': sum(sum(r.e2e_latency_ms) / len(r.e2e_latency_ms) for r in runs) / n,
        'p95_latency_ms': sorted([l for r in runs for l in r.e2e_latency_ms])[int(0.95 * sum(len(r.e2e_latency_ms) for r in runs))],
        'interrupt_recovery': sum(r.interrupt_recovered for r in runs) / n,
        'human_pref_avg': sum(r.human_pref for r in runs) / n,
    }
```

## 5. 端到端业务完成率（最重要）

> [!important]
> 
> 单点指标会骗人：
> 
> - ASR CER 5% 听上去很好，但若错的恰是关键术语，业务完成率可能 < 50%。
> 
> - LLM 文本胜率 60%，但 TTS 合成异常，用户体验仍然是负反馈。
> 
> - 一定要以 **"用户最终是否完成任务"** 为核心，分模块归因诊断。

## 6. 分层评测脚本

```python
import json, pandas as pd

rows = []
for line in open('eval_runs.jsonl'):
    r = json.loads(line)
    rows.append({
        'scene': r['scene'],          # 客服 / 教育 / 医疗
        'accent': r['accent'],        # 普通话 / 粤语
        'noise': r['noise_level'],    # clean / café / phone
        'cer': r['asr_cer'],
        'biz_done': r['business_completed'],
        'latency_ms': r['e2e_latency_ms'],
    })
df = pd.DataFrame(rows)

print('--- 按场景 ---')
print(df.groupby('scene')[['cer', 'biz_done', 'latency_ms']].mean())
print('--- 按口音 ---')
print(df.groupby('accent')[['cer', 'biz_done']].mean())
print('--- 按噪声 ---')
print(df.groupby('noise')[['cer', 'biz_done']].mean())
```

## 7. 上线 SLO 模板

|层级|指标|SLO|
|---|---|---|
|ASR|CER (café 噪声)|≤ 12%|
|LLM|意图准确率|≥ 92%|
|TTS|MOS|≥ 4.0|
|端到端|业务完成率|≥ 88%|
|端到端|失败率|≤ 0.5%|

> [!important]
> 
> - **离线评测 ≠ 上线表现**：离线集分布偏离生产 → 必须留 5-10% 灰度流量做线上 A/B。
> 
> - **MOS 至少 30 个评测员**，否则方差大不可信。
> 
> - **业务完成率需要业务验收，不要 LLM 自评**（避免奉承式高分）。

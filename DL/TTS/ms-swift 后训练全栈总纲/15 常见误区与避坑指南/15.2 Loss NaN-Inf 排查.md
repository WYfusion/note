## 0. 定位

> 一句话：Loss NaN/Inf = 数值不稳。原因 7 类——lr / dtype / 梯度爆炸 / 数据 / 损失函数 / 模板 / 硬件。

---

## 1. 7 类原因

|原因|典型表现|[1.lr](http://1.lr) 过大|开训 100 步内 NaN|
|---|---|---|---|
|2.fp16 上下溢|fp16 全参偶发 NaN|3.梯度爆炸|grad_norm 飙到 1e6|
|4.数据脏|样本里有 NaN / 全 0 / 极长|5.损失函数错|分母 0 / log(0)|
|6.模板 mask 错|label 全 -100 → loss=NaN|7.硬件故障|偶发，重启不复现|

## 3. 常见修复

```bash
# 1. 降 lr + 加 warmup
swift sft ... \
    --learning_rate 5e-5 \           # 之前 1e-4
    --warmup_ratio 0.05 \            # 之前 0.01
    --lr_scheduler_type cosine

# 2. fp16 → bf16
swift sft ... \
    --bf16 true --fp16 false

# 3. 加大 grad_clip
swift sft ... \
    --max_grad_norm 1.0              # 之前 5.0

# 4. 数据筛过短 / 过长 / 异常
swift sft ... \
    --max_length 4096 \
    --min_length 8
```

## 4. 数据脏样本检测

```python
import json, math, statistics

def scan(path):
    lens, errs = [], []
    for ln, line in enumerate(open(path), 1):
        try:
            r = json.loads(line)
        except json.JSONDecodeError:
            errs.append((ln, 'json'))
            continue
        msgs = r.get('messages', [])
        if not msgs:
            errs.append((ln, 'empty messages'))
            continue
        total = sum(len(m.get('content', '') or '') for m in msgs)
        if total == 0:
            errs.append((ln, 'all empty content'))
            continue
        if total > 50000:
            errs.append((ln, f'too long {total}'))
            continue
        lens.append(total)
    print(f'total={len(lens)+len(errs)}  err={len(errs)}')
    if lens:
        print(f'len p50={int(statistics.median(lens))} p99={int(sorted(lens)[int(0.99*len(lens))])}')
    for ln, why in errs[:20]:
        print(f'  L{ln} {why}')

scan('train.jsonl')
```

from swift.llm import get_model_tokenizer, get_template

model, tok = get_model_tokenizer('Qwen/Qwen2.5-7B-Instruct', load_model=False)

## 6. 偶发 NaN 兜底

```python
# 自定义 callback：发现 NaN 自动跳过当前 batch（不停训）
import math
from transformers import TrainerCallback

class SkipNaN(TrainerCallback):
    def __init__(self, n_max=20):
        self.n_max = n_max
        self.skipped = 0

    def on_log(self, args, state, control, logs=None, **kw):
        if not logs:
            return
        loss = logs.get('loss')
        if loss is None or not math.isfinite(loss):
            self.skipped += 1
            print(f'[SkipNaN] step {state.global_step} 跳过, 累计 {self.skipped}')
            if self.skipped > self.n_max:
                print('[SkipNaN] 累计过多，停训')
                control.should_training_stop = True
```

> [!important]
> 
> - **bf16 是首选**：bf16 数值范围比 fp16 大得多，工程上几乎不再用 fp16 训 LLM。
> 
> - **不要忽视梯度裁剪**：max_grad_norm=1.0 / 5.0 是必加的。
> 
> - **GRPO / DPO 偶发 NaN**：通常是 logprob 数值溢出，提高 beta 或降 lr 即可。

## 定位

> VAE 的概率框架、ELBO 推导、重参数化技巧

## 1. 生成模型的目标

我们希望学习数据 $x$ 的分布 $p(x)$。引入潜变量 $z$ 后：

$$p(x) = \int p(x|z) p(z) dz$$

但这个积分是难解的（intractable），因为 $p(z|x)$ 后验分布未知。

---

## 2. ELBO 推导

引入近似后验 $q_\phi(z|x)$，我们可以推导出证据下界（ELBO）：

$$\log p(x) = \underbrace{\mathbb{E}_{q(z|x)}[\log p(x|z)]}_{\text{重建项}} - \underbrace{D_{KL}(q(z|x) \| p(z))}_{\text{KL 正则化}} + \underbrace{D_{KL}(q(z|x) \| p(z|x))}_{\geq 0, \text{近似误差}}$$

$$\therefore \log p(x) \geq \underbrace{\mathbb{E}_{q(z|x)}[\log p(x|z)] - D_{KL}(q(z|x) \| p(z))}_{\text{ELBO}}$$

```python
def elbo_loss(x, mu_q, log_sigma_q, mu_p, log_sigma_p, x_recon):
    """计算 ELBO 损失
    
    最大化 ELBO = 最小化 (-ELBO)
    -ELBO = -E[log p(x|z)] + KL(q||p)
    """
    # 1. 重建损失（负对数似然的近似）
    recon_loss = F.mse_loss(x_recon, x, reduction='sum')
    
    # 2. KL 散度（两个高斯分布之间）
    # KL(N(mu_q, sigma_q) || N(mu_p, sigma_p))
    kl = (log_sigma_p - log_sigma_q 
          + (torch.exp(2*log_sigma_q) + (mu_q - mu_p)**2) 
          / (2 * torch.exp(2*log_sigma_p)) 
          - 0.5).sum()
    
    return recon_loss + kl
```

## 3. 重参数化技巧

$$z = \mu + \sigma \odot \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

将随机采样转化为确定性函数 + 外部噪声，使梯度可以通过 $\mu$ 和 $\sigma$ 反向传播。

> [!important]
> 
> **思辨：为什么 VITS 用 GAN 对抗损失替代 MSE 重建损失？** 标准 VAE 用 MSE/L1 作为重建损失，但这导致输出**模糊**（均值回归）。语音的细节（相位、谐波结构）需要感知级的质量信号，而 GAN 判别器正好提供这种信号。这是 VITS 的关键创新：**用 GAN 的感知损失替代 VAE 的质量损失**。

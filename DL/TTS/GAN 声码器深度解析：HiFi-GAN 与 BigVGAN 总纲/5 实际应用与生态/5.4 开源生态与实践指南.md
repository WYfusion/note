## 2. 快速推理示例

### 2.1 HiFi-GAN

```python
import torch, json
from hifigan.models import Generator
from hifigan.env import AttrDict

with open('config_v1.json') as f:
    h = AttrDict(json.load(f))
vocoder = Generator(h)
vocoder.load_state_dict(torch.load('generator_v1')['generator'])
vocoder.eval().remove_weight_norm()

with torch.no_grad():
    audio = vocoder(mel_spectrogram)  # [B, 1, T]
```

### 2.2 BigVGAN v2

```python
from bigvgan import BigVGAN
import librosa, torch

model = BigVGAN.from_pretrained(
    'nvidia/bigvgan_v2_24khz_100band_256x',
    use_cuda_kernel=True
)
model.eval().cuda()

wav, sr = librosa.load('input.wav', sr=24000)
mel = model.get_mel_spectrogram(torch.from_numpy(wav).cuda())
with torch.inference_mode():
    audio = model(mel)
```

## 3. 微调建议

> [!important]
> 
> **微调策略**：
> 
> 1. **学习率**：使用原训练学习率的 1/10（如 2e-5）
> 
> 1. **步数**：通常 100k-500k 步即可收敛
> 
> 1. **数据量**：建议 ≥ 1h 目标域数据
> 
> 1. **冻结判别器**：前 10k 步冻结判别器，让生成器先适应新数据分布

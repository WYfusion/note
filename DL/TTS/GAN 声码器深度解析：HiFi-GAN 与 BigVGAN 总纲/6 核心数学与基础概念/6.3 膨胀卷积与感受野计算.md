## 1. 膨胀卷积定义

$$(x *_d w)[t] = \sum_{k=0}^{K-1} w[k] \cdot x[t - d \cdot k]$$

有效感受野 = $K + (K-1)(d-1) = K \cdot d - d + 1$

---

## 2. HiFi-GAN ResBlock 中的膨胀率

HiFi-GAN V1 的每个 ResBlock 使用膨胀率 $D_r = [[1,1],[3,1],[5,1]]$：

|**层**|**kernel**|**dilation**|**有效感受野**|1|3|1|3|
|---|---|---|---|---|---|---|---|
|2|3|3|7|3|3|5|11|

堆叠 3 层后总感受野 = $3 + 2 \times (1+3+5) = 21$ 个采样点。

```python
import torch.nn as nn

# HiFi-GAN ResBlock 中的膨胀卷积
for d in [1, 3, 5]:
    conv = nn.Conv1d(
        channels, channels, kernel_size=3,
        dilation=d,
        padding=(3 * d - d) // 2  # 保持序列长度不变
    )
```

> [!important]
> 
> **设计动机**：膨胀卷积在不增加参数的情况下扩大感受野，让生成器能感知更长范围的上下文。配合 MRF 的多 kernel size 并行结构，实现了多尺度的音频模式感知。

## 1. 问题：为什么需要周期性激活？

语音波形本质上是多个正弦分量的叠加（傅里叶级数）。标准激活函数（ReLU / Leaky ReLU）是**分段线性**的，不含任何周期性信息。模型必须完全通过学习卷积核权重来隐式编码周期性，这在 OOD 场景下外推能力很差。

---

## 2. Snake 函数详解

$$f_\alpha(x) = x + \frac{1}{\alpha} \sin^2(\alpha x)$$

### 2.1 三角恒等式展开

利用 $\sin^2(\theta) = \frac{1 - \cos(2\theta)}{2}$：

$$f_\alpha(x) = x + \frac{1}{2\alpha}(1 - \cos(2\alpha x))$$

这说明 Snake 实际上是**恒等映射 + 余弦波动**，周期为 $\frac{\pi}{\alpha}$。

### 2.2 导数分析

$$f'_\alpha(x) = 1 + \sin(2\alpha x)$$

由于 $\sin(2\alpha x) \in [-1, 1]$，有 $f'_\alpha(x) \in [0, 2]$。导数始终非负，保证了**单调性**。

```python
import torch
import torch.nn as nn
import matplotlib.pyplot as plt

class Snake1d(nn.Module):
    """Snake 周期激活函数"""
    def __init__(self, channels):
        super().__init__()
        self.alpha = nn.Parameter(torch.ones(1, channels, 1))
    
    def forward(self, x):
        return x + (1.0 / self.alpha) * torch.sin(self.alpha * x) ** 2

# 可视化
x = torch.linspace(-5, 5, 1000)
for alpha_val in [1, 5, 20]:
    snake = Snake1d(1)
    snake.alpha.data.fill_(alpha_val)
    y = snake(x.unsqueeze(0).unsqueeze(0)).squeeze()
    # alpha 越大，振荡频率越高
```

### 2.3 α 的物理含义

|**α 值**|**行为**|**对应音频**|α → 0|趋近恒等映射（无周期性）|直流分量|
|---|---|---|---|---|---|
|α ~ 1-10|低频振荡|基频 / 低频谐波|α ~ 50-200|高频振荡|高频谐波 / 辅音|

> [!important]
> 
> **关键洞察**：每个卷积通道有独立的 α 参数。训练后，不同通道的 α 会自然分化，分别编码不同频率的周期成分——低频通道 α 小、高频通道 α 大。这与傅里叶分析的思想天然一致。

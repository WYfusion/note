## 1. 工作原理：1D → 2D Reshape

对于周期 $p$，将长度为 $T$ 的 1D 波形重塑为 2D：

$$x \in \mathbb{R}^{T} \xrightarrow{\text{reshape}} X \in \mathbb{R}^{\lfloor T/p \rfloor \times p}$$

```mermaid
flowchart LR
    A["1D 波形
[1, T]"] -->|"reshape(T//p, p)"| B["2D 矩阵
[T//p, p]"]
    B --> C["2D Conv
kernel (5,1)"]
    C --> D["判别分数"]
```

直觉：**每隔** $p$ **个采样点取一次样，形成的子序列应该是平滑的**（因为同一基频成分在固定间隔处的采样近似恒定）。如果生成波形的周期结构不正确，这些子序列就会出现异常。

### 1.1 为什么用 2D Conv 的 kernel 宽度 = 1？

2D 卷积的 kernel 形状为 $(5, 1)$——在时间维度（列方向）上卷积，但在周期维度（行方向）上宽度为 1。这意味着**每个周期位置被独立处理**，判别器专注于检查时间方向上的一致性。

## 2. 素数周期的数学依据

使用素数 $p = [2, 3, 5, 7, 11]$ 而非合数的关键优势在于**最小化子判别器之间的重叠**。

对于任意两个素数 $p_i, p_j$，它们的最小公倍数为 $\text{lcm}(p_i, p_j) = p_i \cdot p_j$，这意味着两个子判别器共同采样到的位置间隔为 $p_i \cdot p_j$——远大于各自的周期。

|**周期组合**|**重叠间隔**|**重叠率**|$p=2, p=3$（素数）|每 6 个采样点|16.7%|
|---|---|---|---|---|---|
|$p=2, p=4$（2的幂）|每 4 个采样点|**50%**|$p=4, p=8$（2的幂）|每 8 个采样点|**50%**|

> [!important]
> 
> **消融实验证实**：使用 $p=[2,4,8,16,32]$（2 的幂次）导致 MOS 从 4.10 下降到 3.90（−0.20）。高重叠率意味着不同子判别器提供的信息高度冗余，梯度信号不够多样化。

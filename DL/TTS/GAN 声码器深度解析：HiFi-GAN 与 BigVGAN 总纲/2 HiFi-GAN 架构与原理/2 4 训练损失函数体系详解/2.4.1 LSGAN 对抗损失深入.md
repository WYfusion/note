## 1. LSGAN 损失函数

**判别器损失**：

$$\mathcal{L}_D = \frac{1}{2} \mathbb{E}_x [(D(x) - 1)^2] + \frac{1}{2} \mathbb{E}_s [(D(G(s)))^2]$$

**生成器损失**：

$$\mathcal{L}_G = \frac{1}{2} \mathbb{E}_s [(D(G(s)) - 1)^2]$$

---

## 2. 梯度分析

|**GAN 变体**|**生成器梯度**|$D(G(s)) \approx 0$ **时**|$D(G(s)) \approx 1$ **时**|
|---|---|---|---|
|**LSGAN**|$D(G(s)) - 1$|$-1$（正常）|$0$（平滑收敛）|

> [!important]
> 
> LSGAN 的梯度是**线性**的，不会在任何点爆炸或消失。这是 HiFi-GAN 和 BigVGAN 能稳定训练的关键原因之一。

## 3. HiFi-GAN 中的实现

```python
def discriminator_loss(disc_real_outputs, disc_gen_outputs):
    """判别器 LSGAN 损失"""
    loss = 0
    for dr, dg in zip(disc_real_outputs, disc_gen_outputs):
        r_loss = torch.mean((dr - 1.0) ** 2)  # 真实样本→目标 1
        g_loss = torch.mean(dg ** 2)            # 生成样本→目标 0
        loss += r_loss + g_loss
    return loss

def generator_loss(disc_gen_outputs):
    """生成器 LSGAN 损失"""
    loss = 0
    for dg in disc_gen_outputs:
        loss += torch.mean((dg - 1.0) ** 2)  # 生成样本→目标 1
    return loss
```

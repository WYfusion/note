## 0. 定位

> Mel 重建损失的作用原理、$\lambda_{mel}=45$ 的实验依据、实现细节

---

## 1. 数学定义

$$\mathcal{L}_{Mel}(G) = \| \phi(x) - \phi(G(s)) \|_1$$

其中 $\phi(\cdot)$ 是将波形转换为 log-Mel 频谱图的函数。

---

## 2. 为什么 $\lambda_{mel} = 45$ 这么大

|**配置**|**MOS**|$\Delta$|
|---|---|---|
|w/o Mel Loss|3.25|**−0.85**|

Mel 损失的角色是**全局频谱约束**——确保生成波形的频谱包络与输入 Mel 一致。没有这个约束，生成器在早期训练中可能产生与输入完全不匹配的波形。

---

## 3. 实现

```python
import torchaudio

class MelLoss(nn.Module):
    """Mel 频谱重建损失"""
    def __init__(self, sr=22050, n_fft=1024, hop=256, n_mels=80):
        super().__init__()
        self.mel_fn = torchaudio.transforms.MelSpectrogram(
            sample_rate=sr, n_fft=n_fft,
            hop_length=hop, n_mels=n_mels
        )
    
    def forward(self, y_real, y_gen):
        mel_real = torch.log(self.mel_fn(y_real) + 1e-9)
        mel_gen = torch.log(self.mel_fn(y_gen) + 1e-9)
        return nn.functional.l1_loss(mel_real, mel_gen) * 45
```

> [!important]
> 
> **思辨：为什么用 L1 而非 L2？**
> 
> L1 损失对异常值更鲁棒（不会被少数极大误差主导），并且产生更尖锐的频谱重建。在音频领域，L1 Mel 损失已被广泛证明优于 L2。

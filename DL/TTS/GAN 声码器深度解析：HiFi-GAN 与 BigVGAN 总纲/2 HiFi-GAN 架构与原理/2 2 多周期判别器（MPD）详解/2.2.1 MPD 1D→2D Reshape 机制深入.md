## 1. Reshape 操作的数学本质

将 1D 波形 $\mathbf{x} \in \mathbb{R}^T$ 重塑为 2D 矩阵 $mathbf{X} in mathbb{R}^{lfloor T/p rfloor times p}$：

$$X[i, j] = x[i \cdot p + j], \quad i = 0, \dots, \lfloor T/p \rfloor - 1, \quad j = 0, \dots, p-1$$

即矩阵的第 $j$ 列包含所有符合 $t \equiv j \pmod{p}$ 的采样点——这实际上是以周期 $p$ 对波形进行的**多相分解（Polyphase Decomposition）**。

---

## 2. 素数周期的数学不重叠性

对于两个素数 $p_i, p_j$，它们的最小公倍数 $text{lcm}(p_i, p_j) = p_i cdot p_j$。

这意味着两个子判别器**同时采样到的第一个共同位置**是 $p_i cdot p_j$：

- $p=2, p=3$：共同位置间隔 = 6，重叠率 = 16.7%

- $p=2, p=4$：共同位置间隔 = 4，重叠率 = **50%**

> [!important]
> 
> **中国剩余定理视角**：5 个素数 $p=[2,3,5,7,11]$ 的乘积为 2310。根据中国剩余定理，对于任意 $t in [0, 2310)$，$(t bmod 2, t bmod 3, t bmod 5, t bmod 7, t bmod 11)$ 的组合是唯一的。即在 2310 个采样点内，5 个子判别器的采样模式完全不重叠。

## 3. PyTorch 实现细节

```python
def reshape_for_period(x, period):
    """1D→2D reshape 的完整实现（含 padding）
    
    Args:
        x: [B, 1, T] 输入波形
        period: 周期 p
    Returns:
        [B, 1, T//p, p] 的 2D 表示
    """
    B, C, T = x.shape
    # 填充到 period 的整数倍
    if T % period != 0:
        pad_len = period - (T % period)
        x = torch.nn.functional.pad(x, (0, pad_len), 'reflect')
        T = T + pad_len
    # reshape: [B, 1, T] -> [B, 1, T//p, p]
    return x.view(B, C, T // period, period)

# 示例
wav = torch.randn(1, 1, 22050)  # 1秒 22kHz
for p in [2, 3, 5, 7, 11]:
    reshaped = reshape_for_period(wav, p)
    print(f"p={p}: {wav.shape} -> {reshaped.shape}")
    # p=2:  [1,1,22050] -> [1,1,11025,2]
    # p=11: [1,1,22050] -> [1,1,2005,11]
```

> [!important]
> 
> **Reflect Padding**：当 $T$ 不是 $p$ 的整数倍时需要填充。HiFi-GAN 使用 `reflect` 填充而非 `zero` 填充，避免在边界引入不自然的断裂。

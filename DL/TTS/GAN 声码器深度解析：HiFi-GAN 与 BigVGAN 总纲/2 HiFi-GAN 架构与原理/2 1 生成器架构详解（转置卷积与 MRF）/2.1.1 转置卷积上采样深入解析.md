## 0. 定位

> 转置卷积在 HiFi-GAN/BigVGAN 中的完整数学推导、参数设计的数学依据、棋盘格伪影的严格证明

---

## 1. 转置卷积的数学定义

对于 stride $s$、kernel size $k$、padding $p$ 的 1D 转置卷积：

$$L_{\text{out}} = (L_{\text{in}} - 1) \times s - 2p + k$$

要实现恰好 $s$ 倍上采样（$L_{\text{out}} = s \cdot L_{\text{in}}$），需要：

$$s \cdot L_{\text{in}} = (L_{\text{in}} - 1) \times s - 2p + k$$

$$\Rightarrow k - 2p = s \quad \Rightarrow \quad p = \frac{k - s}{2}$$

当 $k = 2s$ 时，$p = s/2$，这正是 HiFi-GAN 的选择。

---

## 2. HiFi-GAN V1 的四层转置卷积

|**层**|**stride** $s$|**kernel** $k$|**padding** $p$|**输入 ch**|**输出 ch**|**累计上采样**|
|---|---|---|---|---|---|---|
|2|16|32|8|256|128|×256|
|4|4|8|2|64|32|×4096|

总上采样倍率 = $16 \times 16 \times 4 \times 4 = 4096$，但实际对应 hop_size=256 的上采样。

```python
import torch.nn as nn
from torch.nn.utils import weight_norm

def build_upsample_layers(h_u=512, k_u=(16,16,4,4)):
    """HiFi-GAN V1 的四层转置卷积"""
    layers = nn.ModuleList()
    ch = h_u
    for s in k_u:
        layers.append(weight_norm(
            nn.ConvTranspose1d(
                ch, ch // 2,
                kernel_size=s * 2,   # k = 2s
                stride=s,
                padding=s // 2       # p = s/2
            )
        ))
        ch //= 2
    return layers
```

---

## 3. 棋盘格伪影的数学解释

转置卷积可以视为「先插入零再卷积」。当 $k$ 不是 $s$ 的整数倍时，卷积核在不同输出位置上的有效覆盖不同，产生周期性的幅度起伏。

当 $k = 2s$ 时，每个输出位置恰好被卷积核覆盖 $k/s = 2$ 次，保证幅度均匀。

> [!important]
> 
> **实践经验**：在音频声码器中，棋盘格伪影会表现为可听的周期性噪声（buzzing）。$k=2s$ 是最小的无伪影配置，也是 HiFi-GAN 和 BigVGAN 的默认选择。

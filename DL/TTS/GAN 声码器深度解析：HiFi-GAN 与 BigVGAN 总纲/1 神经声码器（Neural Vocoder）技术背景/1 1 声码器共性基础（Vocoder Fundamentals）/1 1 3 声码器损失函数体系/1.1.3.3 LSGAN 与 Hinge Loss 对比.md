## LSGAN

$$\mathcal{L}^G = \mathbb{E}[(D(G(z))-1)^2], \quad \mathcal{L}^D = \mathbb{E}[(D(x)-1)^2 + D(G(z))^2]$$

- L2 平方增长 → 远离决策边界的样本惩罚更重

- 训练后期可能过度惩罚已优样本

## Hinge Loss

$$\mathcal{L}^G = -\mathbb{E}[D(G(z))], \quad \mathcal{L}^D = \mathbb{E}[\max(0,1-D(x)) + \max(0,1+D(G(z)))]$$

- 判别器有 margin（"够好就停"）

- 训练更稳定

> [!important]
> 
> **BigVGAN 从 LSGAN 切换到 Hinge 后更稳定。** Hinge Loss 的 margin 机制防止判别器过度自信，是大规模训练的更好选择。

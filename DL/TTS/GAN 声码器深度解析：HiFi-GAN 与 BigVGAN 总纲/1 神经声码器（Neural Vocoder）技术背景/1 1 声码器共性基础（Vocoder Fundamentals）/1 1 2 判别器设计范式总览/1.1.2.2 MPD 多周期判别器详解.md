## 核心创新

将 1D 波形按 period $p$ reshape 为 2D：$[B, 1, T] \to [B, 1, T/p, p]$

使用互质 period = {2, 3, 5, 7, 11}，确保覆盖不同基频的谐波。

## 2D 卷积的含义

- 沿 **period 维度**滑动 → 比较**相邻周期**的一致性 = 谐波质量

- 沿 **时间维度**滑动 → 比较**同周期**内的细节

```python
# Reshape: [B, 1, T] → [B, 1, T//p, p]
x = x.view(B, 1, T // period, period)
# 2D Conv: kernel=(5,1) 沿时间维度滑动
```

> [!important]
> 
> **为什么用互质数？** 互质保证不同 period 捕获的周期模式**无重叠**，最大化覆盖率。period=2 捕获偶数谐波，period=3 捕获三分频谐波等。

## 相位的重要性

$$\phi(m,k) = \text{atan2}(\text{Im}(X), \text{Re}(X))$$

相位携带时间细节信息。仅有幅度谱的重建会产生金属质感伪影。

## Griffin-Lim 算法

迭代估计相位（通常 30~60 次）：

```python
def griffin_lim(magnitude, n_iter=32, hop=256, n_fft=1024):
    phase = np.random.uniform(-np.pi, np.pi, magnitude.shape)
    for _ in range(n_iter):
        complex_spec = magnitude * np.exp(1j * phase)
        waveform = librosa.istft(complex_spec, hop_length=hop)
        recon = librosa.stft(waveform, n_fft=n_fft, hop_length=hop)
        phase = np.angle(recon)  # 用重建的相位更新
    return waveform
```

> [!important]
> 
> **Griffin-Lim 的根本局限**：它假设幅度谱完全正确，只优化相位一致性。实际上 Mel 逆映射引入的幅度误差会被放大。神经声码器（HiFi-GAN/Vocos）通过端到端学习同时处理幅度和相位，完全超越了 Griffin-Lim。

## 数值调整

### 取整倍数_make_divisible(ch)

转化为离**ch**最近的8的整数倍，对硬件友好

这里的$ch + divisor / 2$功能的解释:
	取$ch=divisor\times k+i$，$k$为自然数，$i$为$[0,7]$

- $i$若为$[0,divisor/2)$时，结果为$divisor\times k$，
- $i$若为$[divisor/2,divisor)$时，结果为$divisor\times k+divisor$

```python
def _make_divisible(ch, divisor=8, min_ch=None):
    """
    divisor: 8代表着通道数需要被8整除，即新得到通道数new_ch为8的倍数
    https://github.com/tensorflow/models/blob/master/research/slim/nets/mobilenet/mobilenet.py
    """
    if min_ch is None:
        min_ch = divisor  # 最小的通道数为8
    new_ch = max(min_ch, int(ch + divisor / 2) // divisor * divisor)
    if new_ch < 0.9 * ch:
        new_ch += divisor	# 确保新通道不比原始通道数的90%小，不然则加一个divisor
    return new_ch
```

### 统一赋值partial(func,*args,**kwargs)

需要频繁使用一个相同参数的函数时，可以使用functools文件中的partial()类，可以实现对于func函数、方法的统一赋值。

第一个参数是被赋值的函数，其他后面的参数实际是第一个参数函数的参数

```python
from functools import partial
import torch.nn as nn
norm_layer = partial(nn.BatchNorm2d, eps=1e-3, momentum=0.1)
adjust_channels = partial(InvertedResidualConfig.adjust_channels,
                                  width_coefficient=width_coefficient)
```

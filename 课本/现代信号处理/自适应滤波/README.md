## 算法概览

### 1. Kalman滤波器
- **适用场景**：线性动态系统的状态估计
- **核心思想**：贝叶斯框架下的预测-更新递归
- **特点**：最优状态估计（MMSE），需要系统模型

### 2. LMS算法
- **适用场景**：系统辨识、噪声消除、回声消除
- **核心思想**：随机梯度下降
- **特点**：简单高效，$O(M)$复杂度，收敛较慢

### 3. RLS算法
- **适用场景**：需要快速收敛的自适应滤波
- **核心思想**：递归最小二乘
- **特点**：收敛快，$O(M^2)$复杂度，可跟踪时变系统

## 算法比较

| 特性 | Kalman | LMS | RLS |
|------|--------|-----|-----|
| 计算复杂度 | $O(n^3)$ | $O(M)$ | $O(M^2)$ |
| 收敛速度 | 快 | 慢 | 快 |
| 需要模型 | 是 | 否 | 否 |
| 适应时变 | 是 | 有限 | 是 |
| 实现难度 | 中等 | 简单 | 中等 |

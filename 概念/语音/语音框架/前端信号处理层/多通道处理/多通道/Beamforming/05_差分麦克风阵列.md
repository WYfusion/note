# 差分麦克风阵列

## 1. 概述

**差分麦克风阵列 (Differential Microphone Array, DMA)** 是一类特殊的波束形成器，通过对相邻麦克风信号进行差分运算，实现高指向性。其核心思想是模拟声学中的**梯度传感器**。

### 1.1 基本原理

差分阵列利用相邻麦克风间的**压力差**或**压力梯度**来增强指向性：

```
一阶差分：  🎤 ←d→ 🎤
            p₁    p₂
            ↓      ↓
         Δp = p₂ - p₁

二阶差分：  🎤 ←d→ 🎤 ←d→ 🎤
            p₁    p₂    p₃
```

### 1.2 优势

1. **小尺寸高指向性**：阵列尺寸远小于波长时仍有高指向性
2. **低频性能好**：适合语音频段（100-8000 Hz）
3. **结构简单**：只需2-3个麦克风

### 1.3 挑战

1. **白噪声增益低**：对传感器噪声敏感
2. **需要精确匹配**：麦克风间相位/幅度误差影响大
3. **动态范围受限**：差分放大噪声

## 2. 一阶差分阵列

### 2.1 数学模型

**时域模型**：
考虑两个间距为 $d$ 的麦克风，声源位于方向 $\theta$：

$$x_1(t) = s(t)$$
$$x_2(t) = s(t - \tau(\theta))$$

其中时延：
$$\tau(\theta) = \frac{d\cos\theta}{c}$$

**一阶差分输出**：
$$y(t) = x_2(t) - x_1(t) = s(t - \tau(\theta)) - s(t)$$

### 2.2 频域分析

对时域信号进行傅里叶变换：

$$Y(\omega) = X_2(\omega) - X_1(\omega)$$
$$= S(\omega)e^{-j\omega\tau(\theta)} - S(\omega)$$
$$= S(\omega)(e^{-j\omega\tau(\theta)} - 1)$$

**阵列响应**：
$$H(\theta, \omega) = e^{-j\omega\tau(\theta)} - 1$$

利用欧拉公式：
$$e^{-j\omega\tau} = \cos(\omega\tau) - j\sin(\omega\tau)$$

因此：
$$H(\theta, \omega) = \cos(\omega\tau) - 1 - j\sin(\omega\tau)$$

**幅度响应**：
$$|H(\theta, \omega)| = \sqrt{(\cos(\omega\tau) - 1)^2 + \sin^2(\omega\tau)}$$

利用三角恒等式：
$$(\cos x - 1)^2 + \sin^2 x = 2 - 2\cos x = 2(1 - \cos x) = 4\sin^2(x/2)$$

得到：
$$|H(\theta, \omega)| = 2\left|\sin\left(\frac{\omega\tau(\theta)}{2}\right)\right|$$

代入 $\tau(\theta) = \frac{d\cos\theta}{c}$：

$$\boxed{|H(\theta, \omega)| = 2\left|\sin\left(\frac{\omega d\cos\theta}{2c}\right)\right|}$$

### 2.3 指向性分析

**正前方** ($\theta = 0°$)：
$$|H(0°, \omega)| = 2\left|\sin\left(\frac{\omega d}{2c}\right)\right|$$

**侧向** ($\theta = 90°$)：
$$|H(90°, \omega)| = 2\left|\sin(0)\right| = 0$$

**后方** ($\theta = 180°$)：
$$|H(180°, \omega)| = 2\left|\sin\left(-\frac{\omega d}{2c}\right)\right| = 2\left|\sin\left(\frac{\omega d}{2c}\right)\right|$$

**结论**：一阶差分阵列呈现**8字形指向性**（双向性）。

### 2.4 归一化波束图

归一化到正前方响应为1：

$$B(\theta) = \frac{|H(\theta, \omega)|}{|H(0°, \omega)|} = \frac{\sin\left(\frac{\omega d\cos\theta}{2c}\right)}{\sin\left(\frac{\omega d}{2c}\right)}$$

**低频近似** ($\omega d \ll c$)：

利用 $\sin x \approx x$ 当 $x \ll 1$：

$$B(\theta) \approx \frac{\frac{\omega d\cos\theta}{2c}}{\frac{\omega d}{2c}} = \cos\theta$$

这正是**偶极子**（dipole）的指向性！

### 2.5 白噪声增益

对于空间白噪声，白噪声增益为：

$$WNG = \frac{|H(\theta_0, \omega)|^2}{\int_0^{2\pi} |H(\theta, \omega)|^2 d\theta}$$

对于一阶差分（$\theta_0 = 0°$）：

$$WNG_{1st} = \frac{4\sin^2\left(\frac{\omega d}{2c}\right)}{\int_0^{2\pi} 4\sin^2\left(\frac{\omega d\cos\theta}{2c}\right) d\theta}$$

**低频近似**：
$$WNG_{1st} \approx \frac{(\omega d/c)^2}{\int_0^{2\pi} (\omega d\cos\theta/2c)^2 d\theta} = \frac{(\omega d/c)^2}{\pi(\omega d/2c)^2} = \frac{4}{\pi}$$

$$\boxed{WNG_{1st, dB} \approx 10\log_{10}\left(\frac{4}{\pi}\right) \approx 1.05 \text{ dB}}$$

**结论**：一阶差分的白噪声增益很低，对传感器噪声敏感。

## 3. 二阶差分阵列

### 3.1 数学模型

三个等间距麦克风，间距为 $d$：

$$x_1(t) = s(t + \tau)$$
$$x_2(t) = s(t)$$
$$x_3(t) = s(t - \tau)$$

其中 $\tau = \frac{d\cos\theta}{c}$。

**二阶差分**：
$$y(t) = x_3(t) - 2x_2(t) + x_1(t)$$

这是**二阶导数**的离散近似。

### 3.2 频域分析

$$Y(\omega) = X_3(\omega) - 2X_2(\omega) + X_1(\omega)$$
$$= S(\omega)(e^{-j\omega\tau} - 2 + e^{j\omega\tau})$$
$$= S(\omega) \cdot 2(\cos(\omega\tau) - 1)$$

利用三角恒等式 $\cos x - 1 = -2\sin^2(x/2)$：

$$Y(\omega) = -4S(\omega)\sin^2\left(\frac{\omega\tau}{2}\right)$$

**幅度响应**：
$$|H(\theta, \omega)| = 4\sin^2\left(\frac{\omega d\cos\theta}{2c}\right)$$

**归一化**：
$$\boxed{B(\theta) = \frac{\sin^2\left(\frac{\omega d\cos\theta}{2c}\right)}{\sin^2\left(\frac{\omega d}{2c}\right)}}$$

### 3.3 低频近似

当 $\omega d \ll c$ 时：

$$B(\theta) \approx \frac{(\omega d\cos\theta/2c)^2}{(\omega d/2c)^2} = \cos^2\theta$$

这是**四极子**（quadrupole）的指向性！

### 3.4 指向性对比

| 阵列类型 | 低频指向性 | 零点 | 主瓣宽度 |
|----------|------------|------|----------|
| 全向 | 1 | 无 | 360° |
| 一阶差分 | $\cos\theta$ | 90°, 270° | ~180° |
| 二阶差分 | $\cos^2\theta$ | 90°, 270° | ~90° |

### 3.5 白噪声增益

二阶差分的白噪声增益更低：

$$\boxed{WNG_{2nd, dB} \approx -4.8 \text{ dB}}$$

比一阶差分低约6 dB。

## 4. 端射式差分阵列

### 4.1 配置

麦克风沿声源方向排列（endfire configuration）：

```
声源 → 🎤 ←d→ 🎤 ←d→ 🎤
       1      2      3
```

### 4.2 一阶端射

**权重设计**：
$$\mathbf{w} = [1, -e^{j\omega d/c}]^T$$

**阵列响应**：
$$H(\theta, \omega) = 1 - e^{j\omega d(\cos\theta - 1)/c}$$

**心形指向性**（cardioid）：

低频近似下：
$$B(\theta) \approx \frac{1 + \cos\theta}{2}$$

**特点**：
- 前向增益最大
- 后向零点（$\theta = 180°$）
- 单向性好

### 4.3 超心形指向性

通过调整权重，可以获得不同的指向性：

$$\mathbf{w} = [1, -\alpha e^{j\omega d/c}]^T$$

- $\alpha = 1$：心形（cardioid）
- $\alpha = 0.75$：超心形（supercardioid）
- $\alpha = 0.5$：超指向性（hypercardioid）

**超心形**：
$$B(\theta) = 0.25 + 0.75\cos\theta$$

**超指向性**：
$$B(\theta) = 0.5 + 0.5\cos\theta$$

## 5. 鲁棒差分阵列设计

### 5.1 白噪声增益约束

差分阵列的主要问题是白噪声增益低。解决方法：

**约束优化**：
$$\min_{\mathbf{w}} \mathbf{w}^H\mathbf{\Gamma}\mathbf{w} \quad \text{s.t.} \quad \mathbf{w}^H\mathbf{w} \geq \gamma$$

其中 $\gamma$ 是最小白噪声增益。

### 5.2 正则化差分

在差分运算中加入正则化项：

$$y(t) = x_2(t) - \alpha x_1(t)$$

其中 $\alpha < 1$ 降低对噪声的敏感性。

**频域**：
$$H(\omega) = 1 - \alpha e^{-j\omega\tau}$$

选择 $\alpha$ 使得白噪声增益满足要求。

### 5.3 频率依赖权重

不同频率使用不同的差分系数：

$$\alpha(\omega) = \begin{cases}
1, & \omega > \omega_c \\
\omega/\omega_c, & \omega \leq \omega_c
\end{cases}$$

低频减小差分强度，提高鲁棒性。

## 6. 实际设计考虑

### 6.1 麦克风间距选择

**准则**：
$$d < \frac{\lambda_{min}}{2} = \frac{c}{2f_{max}}$$

对于语音（$f_{max} = 8$ kHz）：
$$d < \frac{343}{2 \times 8000} \approx 2.1 \text{ cm}$$

**实际选择**：$d = 1-2$ cm

### 6.2 麦克风匹配

差分阵列对麦克风不匹配极其敏感。

**相位误差影响**：
设相位误差为 $\Delta\phi$，则输出信噪比下降：

$$\Delta SNR = -20\log_{10}\left|\frac{\Delta\phi}{\omega\tau}\right|$$

**要求**：相位匹配 < 1°，幅度匹配 < 0.5 dB

### 6.3 动态范围

差分运算会放大低频噪声。需要：

1. **高通滤波**：去除DC和极低频
2. **动态范围压缩**：限制输出幅度
3. **自动增益控制**：适应不同声压级

## 7. Python实现

```python
import numpy as np

class DifferentialArray:
    def __init__(self, d, c=343, order=1):
        """
        差分麦克风阵列

        参数:
            d: 麦克风间距 (m)
            c: 声速 (m/s)
            order: 差分阶数 (1 or 2)
        """
        self.d = d
        self.c = c
        self.order = order

    def steering_vector(self, theta, f):
        """计算导向矢量"""
        tau = self.d * np.cos(theta) / self.c
        omega = 2 * np.pi * f

        if self.order == 1:
            # 两个麦克风
            d = np.array([1, np.exp(-1j * omega * tau)])
        elif self.order == 2:
            # 三个麦克风
            d = np.array([
                np.exp(1j * omega * tau),
                1,
                np.exp(-1j * omega * tau)
            ])
        return d

    def weights(self, f, alpha=1.0):
        """计算差分权重"""
        if self.order == 1:
            w = np.array([1, -alpha])
        elif self.order == 2:
            w = np.array([1, -2, 1])
        return w

    def beam_pattern(self, f, theta_range=np.linspace(0, 2*np.pi, 360)):
        """计算波束图"""
        w = self.weights(f)
        B = []

        for theta in theta_range:
            d = self.steering_vector(theta, f)
            response = np.abs(np.dot(w.conj(), d))
            B.append(response)

        B = np.array(B)
        # 归一化
        B = B / np.max(B)
        return theta_range, B

    def white_noise_gain(self, f):
        """计算白噪声增益"""
        w = self.weights(f)
        wng = np.abs(np.dot(w.conj(), w))
        wng_db = 10 * np.log10(wng)
        return wng_db

# 使用示例
dma = DifferentialArray(d=0.02, order=1)
theta, B = dma.beam_pattern(f=1000)
wng = dma.white_noise_gain(f=1000)

print(f"白噪声增益: {wng:.2f} dB")
```

## 8. 应用场景

### 8.1 助听器

**优势**：
- 小尺寸（1-2 cm间距）
- 低频指向性好
- 适合语音频段

**配置**：
- 2个麦克风
- 一阶差分或心形
- 自适应混合全向/指向性

### 8.2 手机/平板

**优势**：
- 利用现有麦克风
- 无需额外空间
- 成本低

**配置**：
- 2-3个麦克风
- 端射式配置
- 结合单通道降噪

### 8.3 会议麦克风

**优势**：
- 桌面放置
- 抑制键盘噪声
- 定向拾音

**配置**：
- 2-4个麦克风
- 心形或超心形
- 自适应零点控制

---

## 9. 优缺点总结

### 9.1 优点

1. **小尺寸高指向性**：$d \ll \lambda$ 时仍有指向性
2. **低频性能好**：适合语音应用
3. **结构简单**：2-3个麦克风即可
4. **成本低**：硬件要求低

### 9.2 缺点

1. **白噪声增益低**：对传感器噪声敏感
2. **匹配要求高**：需要精确校准
3. **动态范围受限**：差分放大噪声
4. **频率依赖**：高频性能下降

---

## 10. 与其他方法对比

| 特性 | 差分阵列 | 延迟求和 | MVDR |
|------|----------|----------|------|
| 阵列尺寸 | 很小 | 中等 | 中等 |
| 低频性能 | 好 | 差 | 中 |
| 白噪声增益 | 低 | 高 | 中 |
| 匹配要求 | 高 | 低 | 中 |
| 计算复杂度 | 低 | 低 | 高 |
| 自适应性 | 无 | 无 | 有 |

**选择建议**：
- 小尺寸设备 → 差分阵列
- 远场拾音 → 延迟求和或MVDR
- 强噪声环境 → MVDR或GSC

# 依赖注入

## 概述

依赖注入是FastAPI的核心特性，用于管理共享逻辑和资源。

## 基本用法

```python
from fastapi import Depends

async def get_query_param(q: str = "default"):
    return q

@app.get("/items/")
def read_items(q: str = Depends(get_query_param)):
    return {"q": q}
```

## 类作为依赖

```python
class Database:
    def __init__(self):
        self.data = []

    def add(self, item):
        self.data.append(item)

def get_db():
    db = Database()
    yield db
    # 清理资源

@app.post("/db/items/")
def add_item(item: dict, db: Database = Depends(get_db)):
    db.add(item)
    return db.data
```

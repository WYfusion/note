# 学习场景预设

为了方便后续命令的学习与理解，我们将假设当前处于一个名为 `project_alpha` 的项目文件夹中。

### 假设的文件结构树

```text
/home/user/project_alpha/      <-- 当前所在目录
├── docs/                      # 文档目录
│   ├── manual.txt             # 操作手册
│   └── references/            # 参考资料目录
├── src/                       # 源代码目录
│   ├── main.py                # 主程序
│   ├── config.py              # 配置文件
│   └── scripts/               # 脚本目录
│       └── run.sh             # 启动脚本
├── data/                      # 数据目录
│   ├── raw.csv                # 原始数据
│   └── processed.json         # 处理后的数据
└── logs/                      # 日志目录
    ├── app.log                # 应用运行日志
    └── error.log              # 错误日志
```

在后续的教程中，我们默认当前的工作目录（Working Directory）是 `/home/user/project_alpha/`。

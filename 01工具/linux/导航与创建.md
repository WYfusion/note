# 基础导航与创建

本节介绍如何在 Linux 系统中查看当前位置、切换目录以及创建文件和文件夹。

> **前置场景**：假设我们当前位于 `/home/user/project_alpha/` 目录下。

## 1. 查看与切换 (`pwd`, `ls`, `cd`)

### `pwd` - 我在哪里？
**P**rint **W**orking **D**irectory，显示当前工作目录的绝对路径。

```bash
pwd
# 输出: /home/user/project_alpha
```

### `ls` - 这里有什么？
**L**i**s**t，列出目录内容。

- **基本列出**：
  ```bash
  ls
  # 输出: data  docs  logs  src
  ```

- **显示详细信息 (`-l`)**：
  包括权限、所有者、大小、修改时间。
  ```bash
  ls -l
  # 输出示例:
  # drwxr-xr-x 2 user user 4096 Oct 10 10:00 data
  # -rw-r--r-- 1 user user  120 Oct 10 09:00 README.md
  ```

- **显示隐藏文件 (`-a`)**：
  Linux 中以 `.` 开头的文件（如 `.gitignore`）默认隐藏。
  ```bash
  ls -la
  ```

- **以人类可读格式显示大小 (`-h`)**：
  将字节转换为 KB, MB, GB。
  ```bash
  ls -lh
  ```

### `cd` - 去哪里？
**C**hange **D**irectory，切换工作目录。

- **进入子目录**：
  ```bash
  cd src
  # 现在位于 /home/user/project_alpha/src
  ```

- **返回上一级目录 (`..`)**：
  ```bash
  cd ..
  # 回到 /home/user/project_alpha
  ```

- **返回上一次所在的目录 (`-`)**：
  ```bash
  cd -
  ```

- **回到用户主目录 (`~`)**：
  ```bash
  cd ~
  # 或者直接输入 cd
  ```

---

## 2. 创建 (`mkdir`, `touch`)

### `mkdir` - 创建目录
**M**a**k**e **Dir**ectory。

- **创建一个目录**：
  ```bash
  mkdir models
  ```

- **创建多级目录 (`-p`)**：
  如果父目录不存在，`-p` (parents) 会自动创建它们。
  ```bash
  # 假设 build 目录不存在
  mkdir -p build/output/logs
  ```

### `touch` - 创建空文件
原本用于更新文件的时间戳，如果文件不存在，则创建一个空文件。

- **创建一个新文件**：
  ```bash
  touch notes.md
  ```

- **同时创建多个文件**：
  ```bash
  touch file1.txt file2.txt
  ```

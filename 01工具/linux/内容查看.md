# 内容查看

本节介绍如何在终端中快速查看文件内容，无需打开图形化编辑器。

> **前置场景**：
> - `logs/app.log` (一个较大的日志文件，1000行)
> - `docs/manual.txt` (一个小文本文件)

## 1. 查看全部内容 (`cat`)
**C**oncaten**at**e，常用于查看小型文件。

- **基本使用**：
  ```bash
  cat docs/manual.txt
  ```

- **显示行号 (`-n`)**：
  ```bash
  cat -n src/main.py
  ```

## 2. 分页查看 (`less`)
用于查看大文件。它允许你向前/向后滚动，而不会一次性加载整个文件占用屏幕。

- **基本使用**：
  ```bash
  less logs/app.log
  ```
  **操作快捷键**：
  - `Space` / `PageDown`：向下翻一页
  - `b` / `PageUp`：向上翻一页
  - `Arrow Up/Down`：滚动一行
  - `/keyword`：搜索 "keyword"（按 `n` 跳转下一个，`N` 跳转上一个）
  - `q`：退出

## 3. 查看开头与结尾 (`head`, `tail`)
最常用于检查日志文件。

### `head` - 查看开头
默认显示前 10 行。

- **查看前 10 行**：
  ```bash
  head src/main.py
  ```

- **查看前 20 行 (`-n`)**：
  ```bash
  head -n 20 src/main.py
  ```

### `tail` - 查看结尾
默认显示最后 10 行。

- **查看最后 5 行**：
  ```bash
  tail -n 5 logs/app.log
  ```

- **⭐️ 实时监控 (`-f`)**：
  **非常重要**。用于实时监视正在写入的文件（如运行中的程序日志）。
  ```bash
  tail -f logs/app.log
  ```
  *（按 `Ctrl + C` 停止监控）*

---

## 4. 简单统计 (`wc`)
**W**ord **C**ount，统计行数、字数、字节数。

- **统计行数 (`-l`)**：
  ```bash
  wc -l data/raw.csv
  # 输出: 500 data/raw.csv (表示有500行)
  ```

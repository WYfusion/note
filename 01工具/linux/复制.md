在 Ubuntu 中复制文件夹（目录）到指定文件夹，需要使用 `cp` 命令并加上 `-r`（递归）选项（因为文件夹包含子文件 / 子目录，必须递归复制）。

### 基本命令格式

```bash
    cp -r 源文件夹路径 目标文件夹路径
```

- `-r`：表示递归复制，用于复制目录（必须加此选项，否则无法复制文件夹）。

### 具体示例

假设要将 `~/source_folder` 文件夹复制到 `~/destination_folder` 中，分以下几种情况：

#### 1. 若目标文件夹不存在

执行命令后，会**创建目标文件夹**并将源文件夹的所有内容复制进去：

```bash
    cp -r ~/source_folder ~/destination_folder
```

结果：生成 `~/destination_folder` 文件夹，其内容与 `~/source_folder` 完全相同。

#### 2. 若目标文件夹已存在

执行命令后，会将源文件夹**整个复制到目标文件夹内部**（作为子目录）：


```bash
    cp -r ~/source_folder ~/existing_destination
```

结果：源文件夹会被复制为 `~/existing_destination/source_folder`。

#### 3. 复制文件夹内的所有内容（不含文件夹本身）

如果只想复制源文件夹内的文件和子目录，而不复制源文件夹本身，可以用通配符 `*`：


```bash
    cp -r ~/source_folder/* ~/destination_folder/
```

结果：`source_folder` 内的所有内容会被复制到 `destination_folder` 中。

### 常用补充选项

- `-v`：显示复制过程（ verbose，详细输出），方便查看进度：
``` bash
    cp -rv ~/source_folder ~/destination_folder
```
- `-f`：强制覆盖目标中已存在的同名文件 / 目录（无需确认）：
```bash
    cp -rf ~/source_folder ~/destination_folder
```
- `-p`：保留文件的权限、所有者、时间戳等属性：
```bash
    cp -rp ~/source_folder ~/destination_folder
```

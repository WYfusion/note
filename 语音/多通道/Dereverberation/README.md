## 🔑 核心概念

### 混响三要素
1. **直达声**：直接传播的声音
2. **早期反射**：前50ms的反射，增强响度
3. **后期混响**：50ms后的反射，降低清晰度

### 关键参数
- **RT60**：混响时间（衰减60dB所需时间）
- **DRR**：直达声混响比
- **C50**：清晰度指数

### 主要方法
- **逆滤波**：抵消房间响应
- **线性预测**：预测并去除混响
- **谱减法**：频域减去混响成分
- **深度学习**：端到端学习映射

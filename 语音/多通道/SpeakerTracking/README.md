## 🔑 核心概念

### 追踪三要素

1. **状态表示**：位置、速度、加速度
2. **运动模型**：预测下一时刻状态
3. **观测模型**：连接状态与测量

### 关键算法

- **卡尔曼滤波**：线性高斯系统最优
- **扩展卡尔曼滤波**：非线性系统近似
- **粒子滤波**：非线性非高斯通用方法
- **深度学习**：端到端学习复杂模式

### 性能指标

- **位置误差**：RMSE
- **轨迹平滑度**：速度变化
- **计算复杂度**：实时性
- **鲁棒性**：对噪声和遮挡的抵抗力

## 📊 方法对比

| 方法 | 系统类型 | 计算复杂度 | 精度 | 实时性 |
|------|----------|------------|------|--------|
| KF | 线性高斯 | 低 | 高 | ✓ |
| EKF | 非线性高斯 | 中 | 中 | ✓ |
| UKF | 非线性高斯 | 中 | 高 | ✓ |
| PF | 任意 | 高 | 高 | △ |
| MHT | 多目标 | 很高 | 高 | ✗ |
| RNN | 任意 | 中 | 高 | ✓ |
| 端到端 | 任意 | 高 | 最高 | △ |

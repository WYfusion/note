# 自然语言理解 (Natural Language Understanding, NLU)

## 作用
理解用户输入文本的语义，提取结构化信息。

## 核心任务

### 意图识别 (Intent Detection)
判断用户想做什么（如：播放音乐、查天气）

### 槽位填充 (Slot Filling)
提取关键信息（如：城市=北京，时间=明天）

### 命名实体识别 (NER)
识别人名、地名、机构名等实体

### 指代消解 (Coreference Resolution)
解析代词指代（"它"指什么）

### 情感分析
判断文本情感倾向

## 端到端口语理解 (SLU)
直接从语音到语义，跳过ASR
